## Content Summary

FCT is the final verification step to ensure assembled boards operate correctly under real-world conditions.

**Key Capabilities**:
- Custom test fixtures (tailored design)
- Real-world testing under operating conditions
- Complete coverage from power to outputs
- Full documentation and test records
- NPI turnaround: 24-48 hours

**Testing Scope**:
- Power-on verification
- Signal integrity checks
- Communication interface testing
- Load testing
- Firmware loading and verification
- Analog/digital circuit performance
- Sensor/actuator verification

**Integration**:
- Part of inspection chain: AOI + X-Ray + ICT/FCT
- Process control with SPI feedback loop
- Defect tracking with PPM monitoring
- Traceability: Barcode + lot
- IPC-A-610 Class 2/3

**Test Coverage**:
- Functional verification beyond electrical connectivity
- Operating behavior validation
- Performance specification compliance
- Environmental stress screening (optional)
