## Content Summary

### AOI (Automated Optical Inspection)

Advanced 2D/3D AOI inspection for bare PCBs and SMT assemblies.

**Key Capabilities**:
- 2D & 3D AOI technology
- Defect escape rate: <100 ppm
- 100% inline coverage
- Integration with SPI + X-Ray + ICT
- NPI turnaround: 24-48 hours

**Defect Detection**:
- Component presence/absence
- Component orientation
- Solder joint quality
- Tombstoning
- Bridging/shorts
- Insufficient/ excess solder
- Coplanarity issues
- Wrong components
- Polarity errors

### SPI (Solder Paste Inspection)

Integrated 2D/3D SPI for solder paste volume, height, alignment verification before component placement.

**Key Capabilities**:
- 2D/3D SPI full coverage
- Paste volume measurement
- Height verification
- Alignment checking
- First-pass yield improvement
- NPI turnaround: 24-48 hours

**Defects Prevented**:
- Insufficient paste (opens)
- Excess paste (shorts)
- Paste misalignment
- Stencil aperture issues
- Print bridging

**Process Control**:
- SPI feedback loop to stencil printer
- Process control integration
- Defect tracking with PPM monitoring
- Traceability: Barcode + lot
- IPC-A-610 Class 2/3
