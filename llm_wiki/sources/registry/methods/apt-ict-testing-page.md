## Content Summary

APTPCB In-Circuit Test (ICT) services with custom bed-of-nails fixtures for comprehensive electrical verification of PCBAs.

### Key Capabilities

- **Test Coverage**: ≥80% nets typical
- **Fixture Type**: Custom bed-of-nails design
- **Integration**: Boundary-scan for BGA support
- **Test Speed**: Seconds per board
- **NPI Turnaround**: 24-48 hours

### Defect Detection Capabilities

- Short circuits (solder bridges)
- Open circuits (lifted pads, open traces, broken leads)
- Missing components
- Incorrect component values/types
- Incorrectly oriented polarized components
- Electrically damaged components
- Solder joint quality issues (electrical verification)
- Manufacturing defects (resistor/capacitor/inductor values, transistor functionality)

### Test Process

1. Fixture design & fabrication based on PCBA design
2. Test program development using CAD data and BOM
3. Test setup & calibration
4. Automated testing with sequential signal application
5. Defect identification with precise fault location
6. Reporting & analysis with detailed test reports

### Quality Integration

- Part of comprehensive Quality Management System
- Complements AOI (physical defects) with electrical verification
- Pre-filter before functional testing
- Feed-forward data for process improvement
- Traceable test results

### Standards

- IPC-A-610 Class 2/3
