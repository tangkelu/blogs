# Source Summary

## What It Covers

- Electrical, thermal, mechanical, and physical properties for `N7000-2HT` laminate with `N7000-3` prepreg
- Toughened polyimide material positioning for severe-condition multilayers
- Public IPC-4101, UL94, old GIJ / GIL, and UL file framing

## Why It Matters

- Adds a source-backed AGC N7000-2HT laminate row distinct from existing `N7000-3` and `N7000-3F` coverage, including the laminate/prepreg pairing boundary.

## Extraction Notes

- Keep `2.5 GHz` and `10 GHz` dielectric rows separate.
- Treat severe-condition language as material-positioning context, not finished-board qualification evidence.
- Treat all values as typical data, not specification limits.
