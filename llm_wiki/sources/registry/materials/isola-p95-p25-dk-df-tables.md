# Source Summary

## What It Covers

- Construction-specific `Dk / Df` data for `P95/P25`
- Core and prepreg dielectric table context across listed frequencies and constructions
- Standard versus alternate availability labels inside the table

## Why It Matters

- The main `P95/P25` datasheet gives baseline dielectric values, while this table prevents flattening the product into one universal `Dk / Df` row.
- It supports condition-aware material comparison in high-temperature layer-count and hi-rel drafts.

## Extraction Notes

- Use only with construction, resin-content, thickness, frequency, and standard/alternate context preserved.
- Do not turn the table into a universal product average.
- Do not use standard/alternate availability labels as distributor stock, lead-time, or HIL procurement claims.
