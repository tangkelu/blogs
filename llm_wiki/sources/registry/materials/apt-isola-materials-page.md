## Content Summary

Isola Group high-speed and RF laminate portfolio for server, storage, and microwave applications.

### Isola Product Portfolio

| Grade | Dk (10GHz) | Df (10GHz) | Applications |
|-------|------------|------------|--------------|
| **370HR** | ~4.0 | ~0.020 | Standard FR-4, cost-effective |
| **FR408HR** | 3.65 | 0.0090 | PCIe Gen4/5, 25G optical |
| **I-Speed** | ~3.4 | ~0.006 | High-speed digital |
| **I-Tera MT40** | ~3.4 | 0.0045 | 56G PAM4, high-speed |
| **Astra MT77** | 3.0 | 0.0017 | RF/microwave, radar |

### Key Capabilities
- Full-spectrum coverage from FR-4 to microwave
- Spread-glass fabric options (skew control)
- Hybrid stack-up engineering (RF + digital)
- Glass style documentation (1080, 2116, 3313)
- Copper roughness data for SI correlation
- Polar Si9000e impedance simulations
- TDR/VNA validation

### Applications
- Server and storage boards
- NVMe switch fabrics
- PCIe Gen4/Gen5 host adapters
- 25G/56G optical transceivers
- RF control and DSP layers
- Radar systems

### Stocked Materials
- FR408HR (most popular low-loss FR-4)
- I-Speed, I-Tera MT40
- Astra MT77
- Common core and prepreg thicknesses

Tier-2 internal source for Isola material capabilities. FR408HR is the default choice for cost-sensitive low-loss applications; Astra MT77 for RF/microwave.
