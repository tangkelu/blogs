# Source Summary

## What It Covers

- TSMC positions `CoWoS` inside its `3DFabric` advanced-packaging family
- `CoWoS-S` uses a silicon interposer
- `CoWoS-R` uses an RDL interposer
- `CoWoS-L` combines RDL-based interposer structure with embedded local silicon interconnect
- The platform is framed for high-performance computing, AI, and high-bandwidth memory integration
