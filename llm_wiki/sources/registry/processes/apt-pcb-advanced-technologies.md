## Content Summary

Specialized PCB technologies for demanding applications.

### HDI (High Density Interconnect)
| Feature | Capability |
|---------|------------|
| **Microvia** | 0.075mm (3mil) |
| **Line/Space** | 3/3 mil |
| **Configurations** | 1+N+1, 2+N+2, Any Layer |
| **Stacking** | Staggered, stacked microvias |
| **Laser Drilling** | UV laser ablation |
| **ELIC** | Every Layer Interconnect |

### Flex & Rigid-Flex
| Feature | Capability |
|---------|------------|
| **Flex Layers** | 1-8+ flex layers |
| **Bend Cycles** | Static or dynamic (20,000+ cycles) |
| **Materials** | Polyimide, LCP |
| **Stiffeners** | FR4, aluminum, polyimide |
| **Coverlay** | PI film + adhesive |

### Heavy Copper
| Feature | Capability |
|---------|------------|
| **Copper Weight** | Up to 20oz (700µm) |
| **Layer Count** | 2-16 layers |
| **Techniques** | External plating, embedded |
| **Applications** | Power electronics, high current |

### High-Frequency / RF
| Feature | Capability |
|---------|------------|
| **Materials** | Rogers, Taconic, Arlon, PTFE |
| **DK Range** | 2.2-10.2 |
| **Processing** | Plasma desmear for PTFE |
| **Tolerance** | ±0.01mm for RF features |

### Metal Core / Thermal
| Feature | Capability |
|---------|------------|
| **Base Materials** | Aluminum, copper |
| **Thermal Conductivity** | 1-8 W/mK |
| **Dielectric** | Epoxy, thermally conductive |
| **Applications** | LED, power conversion |

### Ceramic PCB
| Feature | Capability |
|---------|------------|
| **Materials** | Al₂O₃, AlN, BeO |
| **Thermal Conductivity** | 24-285 W/mK |
| **Processing** | DPC, DBC, HTCC, LTCC |
| **Applications** | High-power RF, LEDs |
