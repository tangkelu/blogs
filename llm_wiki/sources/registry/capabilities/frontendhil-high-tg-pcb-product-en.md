# Source Summary

HIL High-Tg PCB product specifications for harsh environments.

## Key Capabilities

### Glass Transition Temperature (Tg)
- **Standard**: 170–180°C
- **Advanced**: ≥200°C (up to 280°C)

### Z-Axis CTE
- **Below Tg**: 50–70 ppm/°C
- **Above Tg**: ~220–300 ppm/°C

### Layer Count
- **Standard**: 2–28 layers
- **Advanced**: 40+ layers

### Base Materials
- **Standard**: High-Tg FR-4 (S1000-2M, IT-180A; Tg ≥170°C)
- **Advanced**: Polyimide, Megtron 6, RO4350B

### Thermal Performance
- **Lead-free reflow**: 3 cycles at 260°C
- **Delamination time**: T260 >10 min, T288 >5 min
- **Thermal cycling**: −40°C ↔ +125°C
- **Warp control**: ≤0.5–0.75%

### Lamination
- **Temperature**: 185–195°C
- **Pressure**: 250–450 psi
- **Pre-bake**: 120–150°C, 2–6 hours

### Registration
- Accuracy: ±75 μm

### Reliability Features
- CAF mitigation via resin chemistry
- Moisture control (MSL-grade storage)
- Cross-section verification per IPC-A-600
- Thermal shock validation

### Standards
- IPC-6012 Class 2/3
- IPC-4101
- IATF 16949
- ISO 13485
