# Source Summary

APT Metal Core PCB (MCPCB) manufacturing capabilities.

## Key Capabilities

### Metal Base
- Aluminum (Al)
- Copper (Cu)
- Iron (Fe)

### Board Size
- Max: 43.3″ × 19″

### Layer Count
- 1–4 layers

### Copper Weight
- 0.5–4 oz

### Trace/Space
- 4/4 mil (with 0.5 oz copper)

### Surface Finish
- HASL
- ENIG
- OSP
- Immersion Silver (Ag)

### Quality Standard
- IPC Class 2/3

### Impedance Control
- ±5%

### Lead Times
- DFM review: <24 h
- Prototype: 24–48 h

### Applications
- LED lighting
- Power electronics
- Automotive
- Superior thermal conductivity and heat dissipation
