## Content Summary

Specialized PCB technologies for demanding thermal, power, and environmental requirements.

### High-Tg PCB
| Feature | Specification |
|---------|-------------|
| **Tg Range** | 170–180°C |
| **Td** | >300°C |
| **Applications** | Automotive, industrial, multiple reflow |
| **Benefit** | Reduces Z-axis expansion, improves reliability |

### Heavy Copper PCB
| Feature | Specification |
|---------|-------------|
| **Copper Weight** | 3oz to 20oz (up to 700µm) |
| **Techniques** | External plating, embedded copper |
| **Applications** | Power electronics, motor drives, automotive |
| **Thermal** | Improved heat dissipation |

### Ceramic PCB
| Feature | Specification |
|---------|-------------|
| **Materials** | Al₂O₃, AlN, BeO |
| **Thermal Conductivity** | 24–285 W/mK |
| **Processing** | DPC, DBC, HTCC, LTCC |
| **Applications** | High-power LEDs, RF power, automotive |

### IC Substrate
| Feature | Specification |
|---------|-------------|
| **Line/Space** | Fine pitch for chip packaging |
| **Materials** | BT resin, ABF film |
| **Applications** | BGA, CSP, SiP packaging |

### Halogen-Free
| Feature | Specification |
|---------|-------------|
| **Standard** | IEC 61249-2-21 |
| **Cl/Br** | <900ppm each, <1500ppm total |
| **Applications** | Environmentally sensitive products |

### Teflon/PTFE PCB
| Feature | Specification |
|---------|-------------|
| **Materials** | PTFE-based high-frequency laminates |
| **Df** | Very low (0.0009–0.002) |
| **Processing** | Plasma desmear required |
| **Applications** | Radar, antenna, microwave |
