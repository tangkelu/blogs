# Canonical Summary

> APTPCB maintains comprehensive certifications covering quality management, environmental compliance, and industry-specific requirements for PCB manufacturing and assembly.

## Core Certifications

### Quality Management
| Certification | Scope | Application |
|---------------|-------|-------------|
| **ISO 9001:2015** | Quality Management System | All PCB/PCBA operations |
| **IATF 16949** | Automotive Quality Management | Automotive sector production |
| **AS9100** | Aerospace Quality Management | Aerospace & defense applications |
| **ISO 13485** | Medical Device Quality Management | Medical/healthcare products |

### Environmental & Safety
| Certification | Scope | Status |
|---------------|-------|--------|
| **RoHS** | Restriction of Hazardous Substances | Compliant |
| **REACH** | Chemical Registration/Evaluation | Compliant |
| **UL Listed** | Safety certification | Certified (file number on request) |

### Industry Standards
| Standard | Application | Class Support |
|----------|-------------|---------------|
| **IPC-A-600** | Acceptability of PCBs | Class 2, Class 3 |
| **IPC-A-610** | Acceptability of Assemblies | Class 2, Class 3 |
| **IPC-6012** | Qualification & Performance | Class 2, Class 3, 3A |
| **IPC-6013** | Qualification & Performance (Flex) | Available |
| **MIL-PRF-31032** | Military Printed Circuit Boards | Process-capable |
| **MIL-PRF-55110** | Legacy military spec | Reference only |

## Industry-Specific Requirements

### Automotive (IATF 16949)
- Production part approval process (PPAP)
- Advanced product quality planning (APQP)
- Failure mode effects analysis (FMEA)
- Measurement system analysis (MSA)
- Statistical process control (SPC)

### Aerospace (AS9100 + AS9102)
- First article inspection (FAI) per AS9102
- Configuration management
- Traceability requirements
- Counterfeit parts prevention
- Risk management per AS9131

### Medical (ISO 13485)
- Design history file (DHF)
- Device master record (DMR)
- Device history record (DHR)
- Corrective/preventive action (CAPA)
- Process validation

## Compliance Notes

> **Certification Scope**: Certifications apply to specific manufacturing locations and processes. Customer-specific requirements may invoke additional standards or clauses.

> **Refresh Status**: Core ISO certifications require periodic surveillance audits. Customer-specific compliance (e.g., UL file numbers, military QPL listing) should be verified per project.
