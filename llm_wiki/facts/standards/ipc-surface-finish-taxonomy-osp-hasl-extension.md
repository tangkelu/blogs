# Canonical Summary

> Public IPC metadata supports a corrected finish taxonomy: `IPC-4552` for ENIG, `IPC-4553` for immersion silver, `IPC-4554` for immersion tin, `IPC-4555` for OSP, and `IPC-4556` for ENEPIG. HASL / solder coating should not be mislabeled as `IPC-4554`; public IPC-6012F TOC metadata is a safer public anchor for rigid-board HASL / solder-coating vocabulary.

## Stable Facts

- IPC public revision metadata and TOC records support `IPC-4552` as ENIG standard identity.
- IPC public revision metadata and TOC records support `IPC-4553` as immersion-silver standard identity.
- IPC public revision metadata and TOC records support `IPC-4554` as immersion-tin standard identity.
- IPC public TOC records support `IPC-4555` as OSP standard identity.
- IPC public revision metadata and TOC records support `IPC-4556` as ENEPIG standard identity.
- Public `IPC-6012F` TOC metadata includes rigid-board surface-finish / solder-coating vocabulary, making it a safer public anchor for HASL / Pb-free solder-coating discussion than misusing `IPC-4554`.

## Conditions And Methods

- Use this card when drafts compare ENIG, ENEPIG, immersion silver, immersion tin, OSP, HASL, or lead-free HASL.
- Use public TOCs for identity and topic discovery only.
- Require licensed standards, customer specifications, or supplier process records before quoting thickness, shelf life, solderability criteria, corrosion acceptance, porosity, black-pad controls, or rework rules.

## Limits And Non-Claims

- This card does not reproduce IPC requirements.
- It does not support finish-thickness tables, shelf-life numbers, black-pad guarantees, wire-bond guarantees, solderability pass criteria, or RF-loss rankings.
- It does not prove a supplier's finish capability, XRF cadence, process control, cost, yield, or delivery performance.
