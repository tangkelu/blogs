# Blocked Claims Needing Official Sources Or Dated Capability Records

- `antenna`: gain, efficiency, range, TRP/TIS, keep-out numerics, OTA readiness, wireless protocol performance.
- `automotive`: `automotive-grade`, `AEC-Q PCB`, `ISO 26262-compliant PCB`, EV / ADAS readiness, IATF-certified supplier claims.
- `backplane`: exact residual-stub targets, connector qualification scope, drill tolerance, insertion-force and plating values, current turnaround promises.
- `castellated`: default hole / edge geometry, plating-window, solder-joint reliability claims, current shop specialization proof.
- `cavity`: exact cavity depth, wall, clearance, plating stack, shield-can production readiness.
- `ceramic`: conductivity, dielectric, CTE, metallization, process-window, hermeticity, and exact HTCC/LTCC/thin-film process claims unless tied to exact official records.
- `consumer`, `industrial`, `motherboard`, `micro PCB`, `custom`: supplier capability, density, miniaturization limits, assembly yield, QA rate, price, MOQ, lead time, and project-proof claims.
- `copper-core`, `LED`, `power`, `thick copper`, `extreme copper`: current-carrying tables, thermal resistance, exact copper weights, heat-dissipation outcomes, reflow profile settings, lead time, cost, and quality-rate claims.
- `high-frequency`, `low-loss`, `impedance controlled`: default impedance values, universal tolerance bands, insertion-loss / return-loss outcomes, RF frequency ceilings, material-substitution claims.
- `high-reliability`, `medical`: compliance, certification, traceability ownership, FDA applicability, implantable or life-critical readiness, medical-device release claims.
- `prototype`, `quickturn`: exact turn windows, same-day / 24h / 48h promises, no-MOQ claims, current logistics / delivery commitments.
- `ultra-thin`: exact finished thickness windows, handling yield, bend / warpage behavior, assembly survival claims.
