## Non-Negotiable Drafting Rules

For all `go_now_conservative` blogs, the following claim classes remain blocked:

1. shared `B` capability numerics such as `trace/space`, drill or laser-via minima, aspect ratio, annular ring, impedance tolerance, registration tolerance, and similar capability tables
2. standards-threshold or acceptance-threshold recovery, including `IPC Class 2 / 3 / 3A`, `IPC-6012 / 6013`, `IST`, `PLT`, coupon plans, plating minima, bow/twist limits, and other threshold-style criteria
3. supplier-proof claims, including supplier approval, qualification, compliance, release authority, accepted-lot status, and transferable HIL-specific factory-capability claims
4. process-window or recipe claims, including lamination-count allowances, stack recipes, bake/press cycles, generic build-up defaults, and transferable manufacturability windows
5. high-speed or RF budget numerics, including channel-loss budgets, insertion-loss tables, Nyquist/budget claims, RF geometry rules, and similar article-grade performance tables
6. dynamic commercial numerics, including lead time, quick-turn windows, yield, cost uplifts, MOQ, and region/customer commercial claims

Interpretation rule:

- if a sentence needs one of those blocked classes to sound persuasive, the sentence should be rewritten or dropped
- conservative generation now means structure, workflow, material-family, and validation-language reuse only where the current corpus already supports it
