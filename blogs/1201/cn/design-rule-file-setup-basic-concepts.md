对于新手而言，这个术语可能听起来有些 intimidating（吓人）。但请相信我，它并非高深莫测的魔法，而是你设计之旅中最可靠的“导航系统”和“安全护栏”。它定义了你的 PCB 在物理世界中应该是什么样子，确保你的创意能够被工厂精确、可靠地制造出来。

本篇旗舰教程将围绕 `design rule file setup` 这一核心，带你走完从零理解 PCB 概念、材料、设计到最终交付的完整链路。我们将用最直白的方式，把所有基础知识一次讲透。

## “design rule file setup”需要先理解哪些部件与术语？

**提问：** 在设置任何规则之前，我必须先了解电路板本身。一块 PCB 到底是由什么构成的？那些专业术语（比如“过孔”、“焊盘”）又是什么意思？

**解释：**
想象一下，一块 PCB 就像一个多层的“三明治”。每一层都有其特定的功能，它们协同工作，构成了电子信号的高速公路。在你开始设置设计规则前，必须先认识这些“食材”和“烹饪术语”。

一块最基础的 PCB 主要由以下几个部分构成：
1.  **基板 (Substrate):** 这是 PCB 的骨架，通常由玻璃纤维和环氧树脂混合而成的绝缘材料制成，最常见的就是大名鼎鼎的 **FR-4**。它为电路板提供了机械强度和刚性。
2.  **铜箔 (Copper Foil):** 附着在基板上的薄铜层，通过蚀刻工艺形成我们所说的“导线”或“走线”(Trace)，用于连接各个电子元器件。
3.  **阻焊层 (Soldermask):** 覆盖在铜箔上的一层薄薄的绝缘保护漆，通常是绿色，但也可以是红、蓝、黑等多种颜色。它的核心作用（**soldermask purpose**）是保护不需要焊接的铜箔不被氧化，并防止在焊接过程中出现意外的“焊锡桥连”（短路）。
4.  **丝印层 (Silkscreen):** 在阻焊层之上，印有元器件标号（如 R1, C1）、公司 Logo 或其他标识性文字和符号的图层。它不导电，主要用于辅助安装和调试。

为了让你更清晰地掌握这些概念，我们整理了一张速查表：

| 术语 (Term) | 定义 (Definition) | 常见误区 |
| :--- | :--- | :--- |
| **焊盘 (Pad)** | 用于焊接元器件引脚的裸露铜面区域。它是元器件与电路板连接的“停靠点”。 | 焊盘不只是一个点，它有形状（圆形、方形、椭圆形）和尺寸，需要精确匹配元器件。 |
| **走线 (Trace)** | 连接不同焊盘的铜箔路径，相当于电路中的导线。 | 走线的宽度不是随意的，它直接决定了能承载的电流量。 |
| **过孔 (Via)** | 连接不同导电层的金属化小孔。它就像楼层间的“电梯”，让信号可以上下穿梭。 | 过孔不是越大越好，它会占用布线空间，并可能影响信号完整性。 |
| **网络 (Net)** | 一组在电气上相互连接的焊盘和走线的集合。例如，所有连接到“GND”的元素共同组成一个“GND 网络”。 | Netlist 是逻辑连接，而走线是物理实现。设计的目标就是将 Netlist 100% 物理化。 |
| **封装 (Footprint)** | 元器件在 PCB 上的物理尺寸和焊盘布局。它定义了元器件在板上的“占地面积”。 | 封装错误是新手最常犯的错误之一，会导致元器件无法安装。 |
| **环形圈 (Annular Ring)** | 过孔或焊盘孔周围的铜环。**Annular ring definition** 指的是孔壁到铜环外缘的宽度，它确保了钻孔后仍有足够的铜与孔壁连接，保证电气导通。 | 环形圈太小可能导致钻孔偏差时断裂，造成开路。 |

## 单/双层 PCB 是如何堆叠起来的？

**提问：** 我知道了基本构成，那么这些层是如何组合在一起的？单层板和双层板的结构有什么不同？

**解释：**
“层叠”(Stackup) 是描述 PCB 各层如何堆叠的结构图。对于初学者来说，最常接触的是[单/双层 PCB](https://hilpcb.com/en/products/single-double-layer-pcb)。它们的结构相对简单，是理解更复杂多层板的基础。

*   **单层 PCB (Single-Layer PCB):** 这是最简单的结构。它只有一层导电铜箔，所有元器件和走线都在这一面。结构通常是：`丝印层 + 阻焊层 + 铜箔层 + 基板`。
*   **双层 PCB (Double-Layer PCB):** 它在基板的两面都有一层铜箔，允许在顶部和底部同时布线。这极大地增加了布线密度和灵活性。结构通常是：`顶层丝印 + 顶层阻焊 + 顶层铜箔 + 基板 + 底层铜箔 + 底层阻焊 + 底层丝印`。

**实操：**
一个标准的 1.6mm 厚度双层板的层叠结构示例如下。这个结构信息是你设置设计规则时需要输入到 EDA 软件中的重要参数。

| # | 层类型 (Layer Type) | 材料 (Material) | 厚度 (Thickness) | 描述 |
| :-: | :--- | :--- | :--- | :--- |
| 1 | Silkscreen (Top) | Ink | ~0.01mm | 顶层元器件标识 |
| 2 | Soldermask (Top) | LPI | ~0.01mm | 顶层阻焊油 |
| 3 | Copper (Top) | Copper | 0.035mm (1oz) | 顶层信号/电源层 |
| 4 | Substrate | FR-4 | 1.51mm | 核心绝缘基板 |
| 5 | Copper (Bottom) | Copper | 0.035mm (1oz) | 底层信号/电源层 |
| 6 | Soldermask (Bottom) | LPI | ~0.01mm | 底层阻焊油 |
| 7 | Silkscreen (Bottom) | Ink | ~0.01mm | 底层元器件标识 |
| | **Total Thickness** | | **~1.6mm** | **标准板厚** |

对于更复杂的[多层板](https://hilpcb.com/en/products/multilayer-pcb)，层叠设计会涉及芯板 (Core) 和半固化片 (Prepreg)，并需要精确计算阻抗。在这种情况下，我们强烈建议在设计初期就与像 **HilPCBPCB Factory (HILPCB)** 这样的制造商沟通，获取他们推荐的层叠方案，以确保设计的可制造性和电气性能。

## 材料、铜厚与焊油：它们影响什么？

**提问：** FR-4、1oz 铜厚、绿色焊油……这些选项我该怎么选？它们对我的设计有什么实际影响？

**解释：**
这些参数不仅是“选项”，它们直接决定了你 PCB 的成本、性能和可靠性。在 `design rule file setup` 阶段，正确理解并设定这些参数至关重要。

*   **基板材料 (Material):**
    *   **FR-4:** 行业标准，性价比极高，适用于绝大多数应用。如果你不确定用什么，选它准没错。HILPCB 提供高品质的[FR-4 PCB](https://hilpcb.com/en/products/fr4-pcb)制造服务。
    *   **高频材料 (Rogers, Teflon):** 用于射频、微波等高速信号电路，介电常数更稳定，但价格昂贵。
    *   **铝基板 (Aluminum Core):** 用于大功率 LED 照明等需要高效散热的场景。

*   **铜厚 (Copper Weight):**
    *   以“盎司/平方英尺”(oz/ft²) 为单位。1oz 约等于 35µm (微米)。
    *   **1oz (35µm):** 最常见的选择，适用于大多数信号线和中低电流的电源线。
    *   **2oz (70µm) 或更厚:** 用于需要承载大电流的电源线或散热平面，可以有效降低温升和电压降。但铜越厚，蚀刻精细线路的难度也越大，最小线宽/间距的要求会更苛刻。

*   **阻焊油 (Soldermask):**
    *   **目的 (Purpose):** 它的首要任务是绝缘和保护。它精确地暴露出需要焊接的焊盘，覆盖住其他所有铜面。
    *   **颜色:** 绿色是最常见的，因为它对人眼友好，便于人工检查 (AOI)。其他颜色如红、蓝、黑、白在功能上没有本质区别，主要出于美学或品牌识别的考虑。但请注意，某些特殊颜色（如哑光黑）可能会增加检查难度和制造成本。

<div class="div-style-3">
    <h4>操作步骤：选择基础参数</h4>
    <ol>
        <li><strong>确定应用场景：</strong>是普通数字电路还是高频/大功率应用？这决定了基板材料。</li>
        <li><strong>评估电流需求：</strong>计算你电路中最大电流的走线，使用在线工具估算所需的线宽和铜厚。</li>
        <li><strong>选择阻焊颜色：</strong>如果没有特殊要求，选择“绿色”是最稳妥、经济的选择。</li>
        <li><strong>记录参数：</strong>将这些选择（如 FR-4, 1.6mm, 1oz 铜厚, 绿色阻焊）记录下来，它们将是你设置设计规则的基础。</li>
    </ol>
</div>

## 从原理图到 PCB：Netlist、封装与 DRC

**提问：** 我已经画好了电路原理图，如何将它变成一块真正的 PCB 设计？`design rule file setup` 在这个过程中扮演什么角色？

**解释：**
从逻辑上的原理图 (Schematic) 到物理上的 PCB 布局 (Layout) 是整个设计流程的核心。这个转换过程依赖于几个关键元素，而设计规则检查 (DRC) 则是保证转换质量的“守门员”。

1.  **原理图 (Schematic):** 这是电路的逻辑蓝图，定义了元器件以及它们之间的连接关系。
2.  **封装 (Footprints):** 你需要为原理图中的每一个元器件指定一个物理封装。例如，一个电阻在原理图中只是一个符号，但在 PCB 上它需要一个带有两个焊盘的实际封装。
3.  **网表 (Netlist):** 当你将封装分配给所有元器件后，EDA 软件会生成一个网表文件。**Netlist and footprints** 是连接原理图和 PCB 的桥梁。Netlist 告诉 PCB 编辑器：“这个芯片的第 1 脚需要连接到那个电阻的第 2 脚”。
4.  **设计规则检查 (DRC - Design Rule Check):** 这就是 `design rule file setup` 发挥作用的地方。你将前面确定的所有物理约束（如最小线宽、最小间距、过孔尺寸、安全距离等）输入到 DRC 规则文件中。然后，EDA 软件会像一个不知疲倦的审查员，实时或定期检查你的设计是否违反了这些规则。

**实操：**
DRC 规则文件通常包含以下核心设置：
*   **Clearance (间距):** 定义不同电气元素之间的最小距离，如线-线、线-焊盘、焊盘-焊盘。这是防止短路的关键。
*   **Track Width (线宽):** 定义不同网络的最小、推荐和最大走线宽度。电源线通常比信号线宽。
*   **Via Size (过孔尺寸):** 定义过孔的钻孔直径和焊盘直径。这关系到 **annular ring definition** 是否满足制造要求。
*   **Hole Size (孔尺寸):** 定义所有非电镀孔（如安装孔）的尺寸。
*   **Silkscreen Rules (丝印规则):** **Silkscreen design rules** 确保丝印文字不会印在焊盘上（这会影响焊接），并保证文字清晰可读。例如，设置丝印到焊盘的最小间距、最小字符高度等。

在你的 EDA 软件（如 Altium Designer, KiCad, Eagle）中找到“Design Rules”或“Board Setup”菜单，将制造商推荐的参数或你根据项目需求计算出的值填入其中。一个好的习惯是：**先设置规则，再开始布线。**

## 布局布线的三步法（模块→走线→检查）

**提问：** 规则设置好了，元器件也导入了 PCB 编辑器。接下来该如何高效地完成布局和布线？

**解释：**
一个优秀的 PCB 布局布线，不仅要实现电气连接，还要考虑信号完整性、散热、可制造性和美观。我们可以遵循一个经典的三步法来系统地完成这项工作。

1.  **模块化布局 (Component Placement):**
    *   **先大后小：** 首先放置连接器、安装孔、大型 IC 等结构性或核心元器件。
    *   **按功能分区：** 将相关的电路模块（如电源部分、模拟信号部分、数字控制部分）集中放置。这有助于缩短关键路径，减少干扰。
    *   **就近原则：** 将旁路电容尽可能靠近 IC 的电源引脚放置。将晶振和其匹配电容紧凑地放在微控制器旁边。

2.  **关键优先布线 (Routing):**
    *   **先电源后信号：** 首先处理电源和地的布线。通常使用较宽的走线或铜平面 (Copper Pour) 来确保低阻抗路径。
    *   **先关键后普通：** 优先布线高速信号线（如 USB, HDMI）、时钟线、模拟信号线等对路径敏感的线路。保持它们简短、直接，并远离干扰源。
    *   **常规布线：** 最后完成剩余的普通信号线连接。

3.  **持续检查 (Checking):**
    *   **运行 DRC：** 在布线过程中要频繁运行 DRC，而不是等到全部完成后才检查。这可以让你及时发现并修正错误，避免返工。
    *   **目视检查：** 软件不是万能的。你需要自己检查走线是否流畅、布局是否整洁、丝印是否清晰。例如，检查是否有 90 度直角走线（应尽量避免，使用 45 度角代替），检查丝印是否被元器件遮挡。

<div class="div-style-1">
    <h4>知识要点：布局布线最佳实践</h4>
    <ul>
        <li><strong>电源完整性：</strong>确保电源和地网络有足够宽的走线或完整的平面层。</li>
        <li><strong>信号完整性：</strong>高速信号线避免使用过孔，并考虑阻抗匹配。</li>
        <li><strong>散热考虑：</strong>大功率元器件下方可以放置散热过孔，或连接到大面积的铜箔上。</li>
        <li><strong>可制造性：</strong>元器件之间留出足够的空间，便于焊接和维修。</li>
    </ul>
</div>

## 输出 Gerber、钻孔、BOM 的标准流程

**提问：** 我的设计已经完成并通过了所有检查。我应该给制造商发送哪些文件？

**解释：**
你不能直接把你的 EDA 工程文件（如 `.PcbDoc` 或 `.kicad_pcb`）发给工厂。你需要生成一套标准化的、被行业广泛接受的制造文件。这套文件就像是 PCB 的“施工图纸”。

**核心交付文件清单：**
1.  **Gerber Files:** 这是最重要的文件集。每一层（铜层、阻焊层、丝印层、板框等）都会生成一个独立的 Gerber 文件。它是 PCB 各层图像的精确描述。
2.  **NC Drill File (钻孔文件):** 包含了所有钻孔（过孔、元器件孔、安装孔）的精确坐标和尺寸信息。
3.  **BOM (Bill of Materials - 物料清单):** 一个详细的清单，列出了板上需要安装的所有元器件的型号、封装、数量和位号。**BOM preparation tips** 包括：确保每个元器件都有唯一的制造商型号 (MPN)，提供替代料选项，并仔细核对数量和位号。
4.  **Pick and Place File (贴片坐标文件):** 也叫 Centroid File。它告诉自动贴片机每个元器件的精确坐标和旋转角度。
5.  **装配图 (Assembly Drawing):** 一张可视化的图纸，清晰地标明了每个元器件在板上的位置和方向，供人工焊接或检查时参考。

<div class="div-style-3">
    <h4>操作步骤：生成制造文件</h4>
    <ol>
        <li>在 EDA 软件中，找到“Export”、“Fabrication Outputs”或“Plot”菜单。</li>
        <li>选择 Gerber RS-274X 或 X2 格式，确保单位、精度等设置正确。勾选所有需要的图层进行输出。</li>
        <li>单独生成 NC Drill 文件，注意单位和格式与 Gerber 文件保持一致。</li>
        <li>导出 BOM 和 Pick and Place 文件，通常为 `.csv` 或 `.xls` 格式。</li>
        <li>将所有生成的文件打包成一个 `.zip` 压缩包。在提交订单前，强烈建议使用免费的在线 Gerber Viewer 或 BOM Viewer 工具，最后一次检查文件是否完整、正确。</li>
    </ol>
</div>

## 如何让设计与 HILPCB 的 DFM 评审无缝衔接

**提问：** 我如何确保我的设计能被 HILPCB 顺利、低成本地制造出来，避免因为设计问题而被退回或延误？

**解释：**
这就是 DFM (Design for Manufacturability - 可制造性设计) 的价值所在。一个好的 `design rule file setup` 是 DFM 的第一步，但与制造商的紧密协作能让这个过程更上一层楼。我们的目标是让你的设计与 **HILPCB** 的生产能力完美匹配。

**实操与协同：**
1.  **获取制造能力参数：** 在开始设计前，访问 HILPCB 网站或联系我们的技术支持，获取我们最新的工艺能力表。这份文件会详细列出最小线宽/间距、最小过孔尺寸、板厚选项等关键参数。将这些“官方”参数直接用于你的 `design rule file setup`，可以从源头上避免大量问题。
2.  **复杂层叠预审核：** 如果你正在设计一个多层板或需要进行阻抗控制的板，不要自己猜测层叠结构。将你的需求（如层数、板厚、阻抗要求）发送给 HILPCB 的工程师团队，我们会为你提供一个经过验证的、可生产的层叠方案。
