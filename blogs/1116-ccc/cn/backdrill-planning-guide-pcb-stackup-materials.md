## 叠层设计的起点：明确输入与期望输出

任何成功的叠层设计，都始于一份清晰的需求文档，而非直接打开 EDA 工具。错误的输入必然导致错误的输出。

### 设计输入：四大核心要素

在规划叠层之前，请务必与团队确认以下四个维度的关键参数：

1.  **信号完整性 (SI) 要求**：
    *   **最高速率/频率**：是 10 Gbps 的 PCIe 3.0，还是 56 Gbps 的 PAM4？这直接决定了你是否需要低损耗材料和背钻。
    *   **阻抗目标**：单端 50Ω、差分 90/100Ω 是常规操作。但 USB、HDMI (`hdmi pcb stackup guide`) 等特定协议有更严格的公差要求（例如 ±7%）。
    *   **关键网络**：哪些是时钟线、数据线？它们需要走内层（带状线）以获得更好的屏蔽，还是可以走外层（微带线）？

2.  **电源完整性 (PI) 要求**：
    *   **最大电流**：核心处理器、FPGA 的 Vcore 电流有多大？这决定了电源层的铜厚，1 oz 还是 2 oz，甚至是重铜。
    *   **PDN 目标阻抗**：低阻抗的电源网络需要紧密耦合的电源/地平面，这会影响介质层的厚度选择。

3.  **热管理与可靠性**：
    *   **功耗与环境**：主要发热器件的功率是多少？产品将在何种温度下工作？这决定了是否需要 High-Tg (高玻璃化转变温度) 材料来保证 `thermal reliability stackup`。
    *   **安规要求**：CTI (相比漏电起痕指数) 是否需要达到 600V？这在工业和医疗产品中至关重要。

4.  **结构与制造限制**：
    *   **总厚度**：是否有 1.6mm 或 2.0mm 的标准限制？
    *   **BGA 间距**：0.4mm pitch 的 BGA 可能需要 HDI (高密度互连) 工艺。
    *   **成本预算**：这是最现实的约束，决定了我们是在 FR-4 范围内优化，还是升级到 Rogers 等高性能材料。

### 设计输出：一份可交付的“施工图”

一份专业的叠层设计输出，应包含以下内容，清晰到工厂可以直接采纳：

*   **层压结构图**：包含每层的类型（信号/地/电源）、材料型号（如 S1141、IT-180A）、芯板(Core)和半固化片(PP)的厚度、铜厚、以及最终压合后的总厚度。
*   **阻抗设计报告**：明确指出各阻抗控制层的线宽、线距、参考层以及预期的阻抗值与公差。
*   **制造说明**：特殊工艺要求，如背钻深度、控深钻、树脂填充等。

## 核心叠层范式：从 4 层到 10 层的演进

没有“万能”的叠层，但存在经过验证的“范式”。以下是 HILPCB 基于数万次量产数据总结出的经典叠层结构，你可以将其作为自己团队标准库的起点。

| 层数 | 结构 (S:信号, G:地, P:电源) | 核心优势与应用场景 |
| :--- | :--- | :--- |
| **4 层** | S1 / G2 / P3 / S4 | **成本最低**。适用于对 EMI 要求不高的 IoT、MCU 控制板。注意：GND 和 PWR 平面为信号提供了参考，但 S1 和 S4 上的高速信号质量较差。 |
| **6 层** | S1 / G2 / S3 / S4 / P5 / S6 | **经典信号/地交错**。S3/S4 走高速信号（带状线），S1/S6 走低速信号。G2/P5 提供了良好的屏蔽和电源分配。是 `hdmi pcb stackup guide` 的常见推荐。 |
| **8 层** | S1 / G2 / S3 / P4 / G5 / S6 / P7 / S8 | **双带状线层**。S3 和 S6 是理想的高速信号层，被 G2/P4 和 G5/P7 完美屏蔽。P4/G5 的紧密耦合提供了优秀的平面电容，改善了电源完整性。 |
| **10 层** | S1 / G2 / S3 / P4 / G5 / S6 / G7 / P8 / S9 / S10 | **高速与电源隔离**。增加了 G7 层，为 S6 提供了更纯净的参考，同时将高速数字部分（如 S3/S6）与敏感的模拟或电源部分（如 S9）有效隔离。 |

## 叠层微调：信号、电源、地与铜厚的搭配原则

确定了基础范式后，真正的细节设计才刚刚开始。

### 信号与参考层的“绑定”

*   **就近参考原则**：每一层高速信号的下方或上方，必须有一个连续的、完整的地平面或电源平面作为其回流路径。路径越短，EMI 越小。
*   **避免跨分割**：高速信号线绝对不能跨越参考平面上的分割区域。如果必须跨越，需要在跨越处放置一个缝合电容。
*   **带状线 vs. 微带线**：内层信号（带状线）被上下两个参考平面包裹，EMI 抑制和阻抗控制都优于外层信号（微带线）。因此，最关键的信号（如时钟、25G+ SerDes）应优先走内层。

### 铜厚的选择：1oz 还是 2oz？

*   **信号层**：通常使用 0.5oz (18µm) 或 1oz (35µm) 的基铜。0.5oz 有利于实现更精细的线宽线距，适用于高密度设计。
*   **电源/地平面**：1oz 是标准配置。如果单板总电流超过 50A，或有局部大电流路径，应考虑将电源层或局部区域升级到 2oz (70µm) 甚至更厚。
*   **注意**：加厚的铜箔会影响蚀刻精度。2oz 铜厚很难做出 4mil 以下的线宽线距。在进行 `surface finish comparison` 时，厚铜也会影响表面涂覆的均匀性。

### 背钻规划：高速设计的“最后一公里”

当信号速率超过 10-25 Gbps，过孔中未使用的部分（Stub，残桩）就像一根天线，会产生严重的信号反射和衰减。此时，背钻（Backdrilling 或 Controlled Depth Drilling）就成了必需品。

    <h4>背钻规划三步法 (Backdrill Planning Guide)</h4>
    <ol>
        <li><strong>识别与计算</strong>：
            <ul>
                <li><strong>识别对象</strong>：定位所有速率 >10 Gbps 的信号网络。</li>
                <li><strong>计算最大残桩长度</strong>：一个经验法则是，Stub 长度（英寸）应小于 `0.15 * Trise / Tpd`，其中 Trise 是信号上升时间，Tpd 是介质的传播延迟。对于 28Gbps NRZ 信号，残桩长度通常需要控制在 10mil (254µm) 以内。</li>
            </ul>
        </li>
        <li><strong>定义背钻层对</strong>：
            <ul>
                <li>根据叠层和布线，确定需要背钻的过孔。例如，一个信号从 L1 传输到 L3，在一个 12 层的板上，就需要从 L12 反向钻到 L3，移除 L4-L12 的残桩。</li>
                <li>在钻孔表中明确标注：`Backdrill: L12 to L3, Target Remaining Stub < 8mil`。</li>
            </ul>
        </li>
        <li><strong>与制造商沟通</strong>：
            <ul>
                <li>提供清晰的背钻图纸。</li>
                <li>确认制造商的背钻深度控制能力。HILPCB 的标准能力是 ±50µm，这对于绝大多数高速设计已足够。</li>
                <li>背钻孔径通常比原始过孔大 8-10mil，以确保能完全清除铜壁。</li>
            </ul>
        </li>
    </ol>

## 混压与特殊材料：应对极端挑战

标准 FR-4 无法解决所有问题。当遇到射频、高功耗或柔性需求时，我们就需要引入特殊材料和混压工艺。

### 混压：Rogers + FR-4

这是最常见的成本优化方案。将昂贵的 Rogers 材料（如 RO4350B）仅用于高频信号层，而其他数字和电源层则使用成本效益高的 FR-4 材料（如 IT-180A）。

**挑战与 HILPCB 的解决方案**：
*   **CTE 不匹配**：不同材料的热膨胀系数不同，在压合和回流焊时可能导致分层或翘曲。
*   **压合参数 (`press cycle`)**：需要找到两种材料都能接受的压合温度和压力曲线。
*   **HILPCB 经验**：我们拥有成熟的 Rogers/FR-4 混压工艺数据库，通过调整 PP 半固化片的流动性和优化 `press cycle`，确保高可靠性的结合。

### 刚柔结合板 (`flex rigid material stackup`)

它将刚性板的承载能力与柔性板的可弯曲性结合起来，常见于相机模组、精密仪器和可穿戴设备。

**设计要点**：
*   **材料**：柔性层通常使用无胶 PI (Polyimide)，刚性部分为 FR-4。
*   **过渡区**：刚柔转换区域的应力最大，需要平滑的走线和补强设计 (Stiffener)。
*   **叠层对称**：柔性部分的叠层也应尽可能对称，以避免弯曲时产生形变。

### MCPCB 与热管理

对于大功率 LED、电源模块等应用，`thermal reliability stackup` 的核心是散热。MCPCB (金属基板) 将电路层直接制作在铝基或铜基上，通过导热绝缘层将热量迅速传导至金属基板。其导热系数远超 FR-4，是解决热问题的终极武器。

## 制造影响：从图纸到实物的“翻译”过程

一个完美的叠层设计，如果脱离了制造现实，就是一张废纸。

*   **树脂流动 (`resin flow`)**：在压合过程中，PP 中的树脂会融化并流动，填充芯板铜箔蚀刻后的空隙。如果设计不当（例如大铜面与密集线路区相邻），可能导致局部树脂缺失或过多，影响介质厚度和阻抗。HILPCB 的 CAM 工程师会通过添加铜箔平衡块来优化此问题。
*   **翘曲 (Warpage)**：非对称的叠层（如 6 层板设计成 S/S/G/P/S/S）或铜箔分布严重不均，是导致板件在热冲击后翘曲的主要原因。请始终保持叠层结构和铜箔分布的对称性。
*   **阻抗 Coupon**：这是验证阻抗控制是否成功的“金标准”。我们会在生产大板的同 一拼板上制作一条包含相同线宽/线距结构的测试条（Coupon），并在生产后使用 TDR (时域反射仪) 进行精确测量，确保阻抗符合你的 ±10% 或更严格的要求。
