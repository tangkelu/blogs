## assembly drawing beginner 需要先理解哪些部件与术语？

**提问：**一块 PCB 电路板到底是由什么构成的？那些行话，比如“过孔”和“焊盘”，又是什么意思？

**解释：**
在我们深入设计之前，必须先熟悉 PCB 的基本解剖结构和通用语言。想象一下，PCB 就像一个为电子元器件建造的多层城市。这个城市有街道（走线）、地基（基板）和建筑物（元器件）。

一块最基础的 PCB 主要由以下几个部分组成：

1.  **基板 (Substrate):** 这是 PCB 的骨架和绝缘层，通常由玻璃纤维环氧树脂制成，最常见的材料就是 FR-4。它为铜箔和元器件提供了机械支撑。
2.  **铜箔 (Copper Foil):** 附着在基板上的薄铜层，通过蚀刻工艺形成导电的“走线”（Traces），用于连接各个元器件，就像城市的电路网络。
3.  **阻焊层 (Solder Mask):** 覆盖在铜箔走线上方的绿色、蓝色或黑色的保护涂层，我们通常称之为“焊油”。它的作用是防止走线氧化，并避免在焊接时出现不必要的“焊桥”（Solder Bridges）。
4.  **丝印层 (Silkscreen):** 在阻焊层之上的文字和符号层，用于标注元器件的编号（如 R1, C1）、轮廓和极性，方便焊接和调试。

**实操：核心术语速查表**

为了让你快速入门，我整理了一张核心术语表。理解这些是阅读原理图和进行 PCB 布局的基础。

| 术语 (Term) | 定义 (Definition) | 对初学者的提示 |
| :--- | :--- | :--- |
| **焊盘 (Pad)** | 用于焊接元器件引脚的裸露铜面区域。 | 这是元器件与电路板连接的“接口”，确保它的大小和形状与你的元器件封装匹配。 |
| **走线 (Trace)** | 连接不同焊盘的导电路径，即铜箔形成的“线”。 | 走线的宽度决定了其可通过的电流量。你需要了解 `trace width calculator intro` 的基本概念，以避免走线烧毁。 |
| **过孔 (Via)** | 用于连接不同铜箔层的电镀孔。 | 它是多层板中的“垂直通道”，让信号可以穿梭于不同层面。 |
| **焊环 (Annular Ring)** | 过孔或焊盘孔周围的铜环。 | 这是 `annular ring definition` 的关键：它确保了孔壁电镀层与铜箔层之间的可靠连接。焊环太小是常见的制造缺陷。 |
| **网络 (Net)** | 一组在电气上相互连接的端点（焊盘）的集合。 | 在设计软件中，所有属于同一个网络的焊盘都会被高亮显示，引导你进行布线。 |
| **封装 (Footprint)** | 元器件在 PCB 上的物理尺寸和焊盘布局。 | 选错封装是初学者最常犯的错误之一，会导致元器件无法安装。务必仔细核对数据手册。 |

<div class="div-style-1">
    <p><strong>知识要点：</strong> PCB 的本质是一个“导体”与“绝缘体”的精密结合体。我们的设计工作，就是规划这些导体的形状和连接关系，同时确保它们在不需要连接的地方保持绝缘。</p>

## 单/双层 PCB 是如何堆叠起来的？

**提问：**我听说过单面板、双面板，它们在结构上有什么不同？“层叠”又是指什么？

**解释：**
“层叠”（Stackup）是指 PCB 各个层面的堆叠顺序和结构。对于初学者来说，最先接触的就是单层和双层板。

-   **单层 PCB (Single-Layer PCB):** 这是最简单的结构，只有一面有铜箔走线。所有元器件都放置在另一面。它的优势是成本极低、生产简单，非常适合 `single layer pcb basics` 的学习项目，例如简单的 LED 闪烁电路。
-   **双层 PCB (Double-Layer PCB):** 基板的两面都有铜箔层。这使得布线密度可以大大增加，因为你可以利用顶层和底层来布线，并通过“过孔”将它们连接起来。这是目前业余爱好者和大多数消费电子产品中最常见的类型。

**实操：一个典型的双层板层叠结构**

让我们来看一个标准的双层板是如何从上到下构建的。这个结构也对应了你最终需要交付给工厂的 Gerber 文件。

| 层面名称 (Layer Name) | 功能描述 (Function) | 对应的 Gerber 文件后缀 (示例) |
| :--- | :--- | :--- |
| **Top Silkscreen** | 顶层丝印，标注元器件信息 | `.GTO` (Gerber Top Overlay) |
| **Top Solder Mask** | 顶层阻焊，保护顶层铜箔 | `.GTS` (Gerber Top Solder Mask) |
| **Top Copper** | 顶层铜箔，主要的走线和元器件焊接层 | `.GTL` (Gerber Top Layer) |
| **Substrate (FR-4)** | 绝缘基板，提供机械支撑 | (不生成 Gerber 文件) |
| **Bottom Copper** | 底层铜箔，另一布线层 | `.GBL` (Gerber Bottom Layer) |
| **Bottom Solder Mask** | 底层阻焊，保护底层铜箔 | `.GBS` (Gerber Bottom Solder Mask) |
| **Bottom Silkscreen** | 底层丝印，通常较少使用 | `.GBO` (Gerber Bottom Overlay) |

当你开始设计更复杂的电路，例如高速信号或紧凑型产品时，就会接触到[多层 PCB](https://hilpcb.com/en/products/multilayer-pcb)，它们在铜箔层之间增加了更多的绝缘层和内层铜箔，以实现更复杂的布线和更好的电气性能。

## 材料、铜厚与焊油：它们影响什么？

**提问：**选择不同的材料会影响我的设计吗？1oz 铜厚是什么意思？

**解释：**
当然会！材料的选择直接关系到 PCB 的性能、成本和可靠性。

-   **基板材料 (Substrate Material):** 对于绝大多数项目，[FR-4 PCB](https://hilpcb.com/en/products/fr4-pcb) 是性价比最高的标准选择。它具有良好的绝缘性和机械强度。但在高频或高温等特殊应用中，则需要使用 Rogers、Teflon 或陶瓷等特殊基材。
-   **铜厚 (Copper Weight):** 这是指每平方英尺铜箔的重量，单位是盎司 (oz)。常见的铜厚有 1oz (约 35µm) 和 2oz (约 70µm)。铜层越厚，走线能承载的电流就越大，散热也越好。对于需要通过大电流的电源电路，通常会选择更厚的铜。
-   **阻焊/焊油 (Solder Mask):** 主要功能是保护电路和防止焊接错误。虽然颜色（绿色、红色、蓝色、黑色、白色等）对电气性能没有影响，但绿色是最常见的，因为它对人眼友好，便于检查。

<div class="div-style-1">
    <p><strong>知识要点：</strong></p>
    <ul>
        <li><strong>材料选择：</strong>决定了 PCB 的电气特性、耐热性和成本。初学者从标准 FR-4 开始即可。</li>
        <li><strong>铜厚选择：</strong>决定了电路的载流能力。电源线通常需要比信号线更宽或更厚的铜。</li>
        <li><strong>焊油颜色：</strong>主要影响外观和人工检查的便利性，对功能无直接影响。</li>
    </ul>
</div>

## 从原理图到 PCB：Netlist、封装与 DRC

**提问：**我用软件画好了电路图，如何把它变成一个可以布线的 PCB 文件？

**解释：**
这个从逻辑设计到物理设计的过程，即 `pcb schematic to layout`，是整个工作的核心。它依赖于几个关键概念的协同工作。

1.  **原理图 (Schematic):** 这是电路的逻辑蓝图。它定义了有哪些元器件，以及它们之间是如何通过“网络 (Net)”连接的。在这一步，你只关心逻辑，不关心物理位置。
2.  **封装 (Footprint):** 每个原理图符号都需要关联一个物理封装。封装定义了元器件在 PCB 上的实际尺寸、引脚间距和焊盘形状。
3.  **网络表 (Netlist):** 当你完成原理图并分配好封装后，设计软件会生成一个网络表。这个文件像一座桥梁，告诉 PCB 布局工具，哪个封装的哪个引脚应该连接到另一个封装的哪个引脚。
4.  **设计规则检查 (DRC - Design Rule Check):** 在开始布局之前，最重要的一步是设置设计规则。这包括最小线宽、线间距、过孔尺寸等一系列制造约束。这一步是 `design rule file setup` 的核心。专业的制造商，如 **HilPCBPCB Factory (HILPCB)**，会提供其制程能力参数，你可以根据这些参数设置 DRC 规则，从源头上避免设计出无法生产的板子。

<div class="div-style-3">
    <h4>操作步骤：从原理图到 PCB 布局的准备工作</h4>
    <ol>
        <li><strong>完成原理图绘制：</strong>确保所有元器件都已添加，并且网络连接正确无误。运行电气规则检查 (ERC) 排除逻辑错误。</li>
        <li><strong>分配封装：</strong>为原理图中的每一个元器件指定一个正确的物理封装。务必对照元器件数据手册进行二次确认。</li>
        <li><strong>导入 PCB 编辑器：</strong>使用软件的“Update PCB from Schematic”或类似功能，将网络表和封装信息导入到 PCB 布局界面。此时，你会看到所有封装和连接它们的“飞线 (Ratsnest)”。</li>
        <li><strong>设置设计规则 (DRC)：</strong>根据你的制造商（如 HILPCB）的工艺能力，设置最小线宽、间距、过孔尺寸、板边距离等关键参数。</li>
    </ol>
</div>

## 布局布线的三步法（模块→走线→检查）

**提问：**所有的元器件和飞线都导入了，我该从哪里下手？感觉一团乱麻。

**解释：**
一个清晰的布局布线思路能让这个过程事半功倍。我推荐遵循“布局→布线→检查”的三步法。

1.  **第一步：布局 (Placement)**
    -   **模块化布局：** 将功能相关的元器件（如电源模块、MCU 核心模块、传感器模块）集中放置。这有助于缩短关键路径，减少干扰。
    -   **先大后小，先特殊后普通：** 首先放置连接器、安装孔、开关等位置和尺寸固定的元器件。然后放置核心 IC，再围绕它放置外围电路。
    -   **考虑信号流和散热：** 尽量让信号流向呈线性，避免交叉。将发热量大的元器件放置在利于散热的位置，并远离热敏感元器件。

2.  **第二步：布线 (Routing)**
    -   **先电源后信号：** 首先布好 VCC 和 GND 等电源网络。使用较宽的走线或覆铜 (Copper Pour) 来确保电源路径的低阻抗。
    -   **关键信号优先：** 优先处理高速信号、时钟信号或模拟信号等敏感线路，让它们尽可能短、直，并远离干扰源。
    -   **布线规范：** 避免 90° 直角走线，推荐使用 45° 或圆弧角，以减少信号反射。根据电流大小，使用走线宽度计算器来确定合适的线宽。

3.  **第三步：检查 (Verification)**
    -   **运行 DRC：** 在布线完成后，务必反复运行设计规则检查 (DRC)，直到所有错误都被修正。
    -   **人工检查：** 检查丝印是否清晰、是否压到焊盘；检查元器件间距是否足够焊接；检查孤立的铜皮 (Dead Copper) 等。
    -   **3D 预览：** 使用软件的 3D 预览功能，检查元器件的实际高度是否会与其他部件产生干涉。

## 输出 Gerber、钻孔、BOM 的标准流程

**提问：**我的设计完成了，要发给工厂哪些文件才能生产？

**解释：**
将设计文件转换为一套标准的制造文件是交付的最后一步，也是至关重要的一步。任何文件的缺失或错误都可能导致生产延误或失败。一个完整的生产资料包通常包含以下内容：

-   **Gerber Files:** 这是 PCB 制造业的通用标准格式。你需要为每一个物理层面（铜层、阻焊层、丝印层、锡膏层等）都生成一个对应的 Gerber 文件。
-   **钻孔文件 (NC Drill File):** 它定义了板上所有孔（过孔、元器件孔、安装孔）的坐标和尺寸。
-   **物料清单 (BOM - Bill of Materials):** 一个详细的清单，列出了项目所需的所有元器件的型号、封装、数量和元件位号。这是采购和贴片组装的依据。
-   **装配图 (Assembly Drawing):** 这份图纸清晰地标明了每个元器件在 PCB 上的位置和方向，是指导焊接（尤其是手工焊接或检查）的关键文件。
-   **坐标文件 (Pick and Place File):** 也叫 Centroid File，它提供了每个元器件中心点的 X/Y 坐标和旋转角度，供自动化贴片机使用。

<div class="div-style-3">
    <h4>操作步骤：标准生产文件导出清单</h4>
    <ol>
        <li><strong>导出 Gerber 文件：</strong>确保单位、精度设置正确，并为所有需要的层面生成文件。</li>
        <li><strong>导出 NC 钻孔文件：</strong>通常与 Gerber 文件一起导出。</li>
        <li><strong>生成 BOM 文件：</strong>从原理图工具导出，并整理成清晰的表格格式（如 .csv 或 .xlsx），确保包含制造商型号 (Manufacturer Part Number)。</li>
        <li><strong>生成坐标文件 (PnP)：</strong>从 PCB 布局工具导出。</li>
        <li><strong>创建装配图：</strong>通常是 PCB 顶层和底层的截图，并清晰标注元件位号，导出为 PDF 格式。</li>
        <li><strong>打包压缩：</strong>将以上所有文件放入一个文件夹，并压缩成一个 .zip 文件，以你的项目名称命名。</li>
    </ol>
