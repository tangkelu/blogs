大家好，欢迎来到 HILPCB 测试实验室。我是你们今天的新人培训讲师。在接下来的时间里，我将带领大家系统地了解 PCB 测试与检验的完整流程。我们的目标是让每位新人都能清晰地理解我们“为什么测”、“如何测”以及“怎么判”，从而确保从我们实验室出去的每一块 PCB 都符合最严格的质量标准。

很多人可能听说过一些复杂的测试，比如今天我们要重点探讨的“salt spray test tutorial”（盐雾测试教程）。盐雾测试是评估产品耐腐蚀性的关键一环，尤其对于那些需要在严苛环境中工作的电子设备至关重要。但在此之前，一块 PCBA（印刷电路板组件）需要经历一系列“关卡”，从基础的电气连接到复杂的功能实现，再到最终的环境可靠性验证。

所以，今天的培训将以盐雾测试为远景目标，但我们会从最基础的测试步骤开始，层层递进，构建一个完整的 PCB 测试知识框架。

## “salt spray test tutorial”要回答哪些质量问题？

在我们深入探讨盐雾测试的具体操作前，首先要理解它在整个质量保证体系中的位置。盐雾测试本身并不孤立，它是“可靠性测试”大家族中的一员，专门用来回答一个核心问题：**“我们的 PCB 产品能否在潮湿、含盐的腐蚀性环境中长期稳定工作？”**

这个问题对于航海设备、沿海地区的通信基站、汽车电子等应用场景至关重要。具体来说，盐雾测试帮助我们验证：

*   **表面处理的有效性**：无论是沉金（ENIG）、有机涂覆（OSP）还是热风整平（HASL），它们的抗腐蚀能力如何？
*   **阻焊油墨的附着力**：在盐雾侵蚀下，阻焊层是否会起泡、脱落，导致铜线暴露？
*   **三防漆（Conformal Coating）的保护能力**：涂覆了三防漆的 PCBA 是否能有效隔绝湿气和盐分？
*   **元器件的耐腐蚀性**：连接器、金属外壳等部件是否会生锈，影响电气连接和结构完整性？

然而，腐蚀只是潜在的失效模式之一。一块合格的 PCB，在考虑其长期可靠性之前，必须首先确保其基本功能是完好的。因此，一个全面的测试流程需要回答更多问题：

*   **制造缺陷**：是否存在短路、开路？（电气测试回答）
*   **装配错误**：元器件是否贴错、漏贴、反向？焊点质量如何？（光学检测回答）
*   **功能实现**：电路板上电后能否按设计要求工作？（功能测试回答）
*   **长期耐用性**：它能否承受温度变化、湿度、振动等环境压力？（可靠性测试回答）

现在，让我们从最基础的电气测试开始，一步步揭开 PCB 测试的全貌。

## 飞针与 ICT：何时使用、如何准备测试点？

电气测试（E-Test）是 PCB 制造和组装后的第一道“大考”，主要目的是检查电路的连通性，找出开路（Open）和短路（Short）等制造缺陷。在 HILPCB，我们主要使用两种工具：飞针测试（Flying Probe Test, FPT）和在线测试（In-Circuit Test, ICT）。

### 飞针测试（FPT）
想象一下，有几根由电脑控制的探针，像蜜蜂采蜜一样在 PCB 上快速移动，逐个接触预设的测试点（Test Pad），这就是飞针测试。

*   **工作原理**：通过高精度的移动探针接触焊盘、过孔或测试点，测量它们之间的电阻值，从而判断是否存在开路或短路。
*   **优点**：无需制作昂贵的专用夹具，程序准备灵活快速，非常适合原型和小批量组装（[prototype assembly](https://hilpcb.com/en/products/prototype-assembly)）。
*   **缺点**：测试速度较慢，因为探针需要逐点移动，不适合大规模量产。

### 在线测试（ICT）
如果说飞针是“散兵游勇”，那 ICT 就是“集团军作战”。ICT 使用一个布满探针的定制夹具，俗称“针床”（Bed of Nails）。这个概念也是我们常说的 **bed of nails tutorial** 的核心。

*   **工作原理**：将被测板（UUT, Unit Under Test）压在针床上，成百上千根探针同时接触板上的所有测试点，在几秒钟内完成所有网络的通断测试，甚至可以测量电阻、电容、电感等元器件的数值。
*   **优点**：测试速度极快，覆盖率高，非常适合大批量生产。
*   **缺点**：需要为每款 PCB 定制昂贵的 **ICT fixture**（ICT夹具），前期投入大，不适用于设计频繁变更的产品。

<div class="div-style-1">

**要点：如何选择 FPT 与 ICT？**

| 特性 | 飞针测试 (FPT) | 在线测试 (ICT) |
| :--- | :--- | :--- |
| **适用阶段** | 原型、小批量、NPI（新产品导入） | 大规模量产 |
| **测试速度** | 慢（分钟级） | 快（秒级） |
| **前期成本** | 低（无需夹具） | 高（需定制昂贵的针床夹具） |
| **灵活性** | 非常高，修改测试程序即可 | 低，设计变更可能需要重做夹具 |
| **覆盖范围** | 主要测开路/短路 | 可测开路/短路、元器件数值、部分功能 |

</div>

**设计要点：测试点的准备**
无论使用哪种方法，良好的可测试性设计（DFT, Design for Testability）都至关重要。工程师在设计 PCB 时，应预留足够的测试点，并遵循以下原则：
*   **尺寸**：测试点直径建议不小于 0.8mm。
*   **间距**：测试点中心间距建议不小于 1.27mm。
*   **位置**：避免放置在大型元器件边缘，方便探针接触。
*   **分布**：均匀分布，避免探针过于集中导致板子受力不均。

在 HILPCB，我们的工程师会与客户紧密合作，提供专业的 DFT 建议，并定制高精度的**飞针/ICT/功能治具**，确保测试的顺利进行。

## 功能/上电测试：脚本、夹具与判据

通过了电气测试，我们能保证“电路是通的”，但它能否“按预期工作”呢？这就是功能测试（Functional Test, FCT）要回答的问题。FCT 模拟 PCB 的实际工作环境，对其进行全面的功能验证。

<div class="div-style-3">

**功能测试（FCT）实施流程**

1.  **定义测试需求**：与客户沟通，明确需要测试哪些功能。例如，一个带 LED 的控制板，需要测试“按下按钮 A，LED B 是否以 1Hz 的频率闪烁”。
2.  **开发测试脚本**：编写自动化软件脚本，通过上位机或单片机向被测板发送输入信号（如模拟按键、输入电压），并读取其输出信号（如串口数据、LED 状态）。
3.  **设计测试夹具**：制作专用的 FCT 夹具。夹具不仅要为 PCB 供电，还要提供所有必需的输入/输出接口，如 USB、以太网、探针等，以连接测试设备。
4.  **执行测试**：将待测板放入夹具，运行测试脚本。脚本会自动执行一系列操作并记录结果。
5.  **判定结果**：将实际输出与预期结果进行比较。所有测试项均符合预期，则为“PASS”；否则为“FAIL”，并记录详细的失败信息。

</div>

**判据（Pass/Fail Criteria）的设定至关重要。** 它必须是量化的、明确的。例如：
*   **电压**：VCC 电压必须在 4.95V 到 5.05V 之间。
*   **信号**：时钟信号频率误差必须在 ±1% 以内。
*   **响应时间**：从接收指令到返回数据的时间必须小于 50ms。
*   **状态**：设备启动后，状态指示灯必须为绿色常亮。

一个清晰的 **test report template**（测试报告模板）会详细列出这些判据和实际测量值，为质量追溯提供依据。

## AOI/SPI/X-Ray：光学检测如何发现缺陷？

在元器件贴装（SMT）过程中，许多缺陷是肉眼难以发现的，例如虚焊、锡珠、元器件偏移等。光学检测技术利用机器视觉，实现了对这些微小缺陷的高效、精准识别。

### 锡膏检查（SPI）
SPI（Solder Paste Inspection）发生在 SMT 流程的最前端，即锡膏印刷之后、元器件贴装之前。它通过 3D 扫描检测每个焊盘上的锡膏体积、面积、高度和偏移量，从源头上杜绝因锡膏问题导致的焊接缺陷。这是保证SMT 组装（[SMT assembly](https://hilpcb.com/en/products/smt-assembly)）质量的第一道防线。

### 自动光学检测（AOI）
AOI（Automated Optical Inspection）通常放置在回流焊之后，通过高分辨率相机拍摄 PCBA 图像，并与预设的合格标准图像进行比对。

<div class="div-style-1">

**AOI 能发现的主要缺陷**

*   **元器件缺陷**：错件、漏件、反向、偏移、立碑。
*   **焊点缺陷**：锡多、锡少、虚焊、连锡（短路）。
*   **异物与污染**：板面的锡珠、残留物等。

AOI 极大地提升了检测效率和一致性，它就像一个不知疲倦的质检员，严格执行着一份数字化的 **visual inspection checklist**（目视检查清单）。

</div>

### X-Ray 检测（AXI）
对于一些“隐藏”的焊点，如球栅阵列封装（BGA）和 QFN 封装，其焊点位于元器件底部，AOI 无法看到。这时就需要 X-Ray 检测。X-Ray 能够穿透元器件，清晰地呈现底部焊球的形状、大小和对齐情况，从而发现气泡、枕头效应（Head-in-Pillow）、短路等深层缺陷。对于[高密度互连板（HDI PCB）](https://hilpcb.com/en/products/hdi-pcb)等复杂产品，X-Ray 检测是必不可少的环节。

## 可靠性入门：热循环、湿热、振动、跌落与盐雾测试

当一块 PCB 确认制造无误、功能正常后，我们还需要评估它在整个生命周期内的耐用性。这就是可靠性测试的范畴，它通过模拟各种严苛的环境来“加速老化”，提前暴露潜在的设计或工艺弱点。HILPCB 的**可靠性仓**配备了多种环境模拟设备。

*   **热循环测试**：将 PCB 在高温和低温之间反复循环，以检验不同材料（如基板、元器件、焊点）因热膨胀系数（CTE）不匹配而可能导致的机械应力失效。
*   **湿热测试**：将 PCB 置于高温高湿环境中（如 85°C / 85% RH），评估其抗湿气侵蚀的能力，防止因吸湿导致的绝缘性能下降或材料分层。
*   **机械冲击与振动测试**：模拟产品在运输或使用过程中可能遇到的颠簸和振动，检查焊点和元器件的机械牢固性。

### 盐雾测试教程（Salt Spray Test Tutorial）
现在，我们回到最初的议题。盐雾测试是评估产品抗大气腐蚀能力，特别是抗海洋性气候腐蚀能力的核心方法。

<div class="div-style-3">

**盐雾测试（ASTM B117 标准）核心流程**

1.  **样品准备**：选择待测样品，进行清洁和干燥。非测试区域（如连接器端口）可能需要用耐腐蚀胶带进行保护。
2.  **盐溶液配置**：通常使用 5% 浓度的氯化钠（NaCl）溶液，并用乙酸或氢氧化钠调节 pH 值至 6.5-7.2 的中性范围。
3.  **试验箱设置**：将试验箱温度设定在 35°C (±2°C)。通过压缩空气将盐溶液雾化，使其在箱内均匀沉降，沉降速率控制在 1-2 mL/h/80cm²。
4.  **样品放置**：将样品以与垂直方向成 15-30° 的角度放置在样品架上，确保盐雾能自由沉降在所有表面，且样品之间不互相接触或遮挡。
5.  **持续暴露**：根据产品标准或客户要求，进行规定时间的连续喷雾暴露，常见时间为 24h, 48h, 96h, 甚至更长。
6.  **评估与记录**：暴露结束后，取出样品，用缓和的流水清洗表面残留的盐分，然后立即进行评估。记录腐蚀现象，如红锈、白锈、起泡、龟裂等，并拍照存档。

</div>

**如何判定盐雾测试结果？**
判定标准通常基于腐蚀出现的时间和腐蚀面积的百分比。国际标准（如 ISO 10289）提供了详细的评级系统。

| 评级 (Rating Number) | 缺陷面积占总面积的百分比 (A) | 描述 |
| :--- | :--- | :--- |
| 10 | 0 | 无腐蚀缺陷 |
| 9 | 0 < A ≤ 0.1 | 非常轻微的腐蚀 |
| 8 | 0.1 < A ≤ 0.25 | 轻微腐蚀 |
| 7 | 0.25 < A ≤ 0.5 | - |
| 6 | 0.5 < A ≤ 1.0 | 中等腐蚀 |
| 5 | 1.0 < A ≤ 2.5 | - |
| 4 | 2.5 < A ≤ 5.0 | 较严重腐蚀 |
| 3 | 5.0 < A ≤ 10.0 | - |
| 2 | 10.0 < A ≤ 25.0 | 严重腐蚀 |
| 1 | 25.0 < A ≤ 50.0 | - |
| 0 | A > 50 | 非常严重的腐蚀 |

例如，一个产品要求“通过 96 小时盐雾测试，评级不低于 9 级”，意味着在 96 小时后，其表面的腐蚀点面积不能超过总面积的 0.1%。

## 测试报告与追溯：记录哪些数据？

“没有记录，就没有发生。” 所有的测试工作最终都必须以一份详尽、准确的测试报告作为收尾。这份报告不仅是产品质量的证明，也是实现 **test data traceability**（测试数据可追溯性）的关键。

<div class="div-style-1">

**一份合格的测试报告应包含以下核心要素：**

*   **产品信息**：产品型号、版本号、唯一的序列号（Serial Number）。
*   **测试环境**：测试日期、时间、操作员、使用的设备型号。
*   **测试项目与标准**：明确列出执行了哪些测试（如 ICT, FCT, Salt Spray），并引用了哪些标准（如 IPC-A-610, ASTM B117）。
*   **测试参数**：记录关键的测试条件，如 FCT 的输入电压、盐雾测试的持续时间等。
*   **测试数据与结果**：详细记录每个测试项的测量值和最终的 PASS/FAIL 判决。
*   **缺陷描述**：对于失败的样品，需附上清晰的缺陷照片、位置描述和初步的失效分析。
*   **原始数据链接**：提供指向原始日志文件或数据库记录的链接，以便深入分析。

</div>

通过建立完善的追溯系统，当市场出现问题时，我们可以迅速根据序列号追溯到该产品的全部生产和测试数据，快速定位问题根源，无论是特定批次的元器件问题，还是某个生产环节的工艺偏差。您可以使用我们的 BOM 查看器（BOM Viewer） 等工具来辅助管理和核对物料信息，确保源头数据的准确性。
