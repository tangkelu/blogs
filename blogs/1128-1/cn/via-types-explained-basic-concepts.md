## 1. 执行摘要：痛点、机会与目标

对于追求快速迭代和产品可靠性的初创研发团队而言，印刷电路板（PCB）设计是硬件开发的核心瓶颈之一。我们发现，团队内部常常因术语不统一、流程缺失、缺乏可制造性（DFM）意识而导致沟通不畅、设计返工和项目延期。尤其在“过孔（Via）”这一基础而关键的设计元素上，混乱的理解——从简单的通孔到复杂的盲埋孔、微孔——直接导致了信号完整性问题、生产成本失控和交付失败。

**核心痛点：**
*   **术语鸿沟：** 研发、工程与制造供应商之间对“盲孔”、“盘中孔”、“深宽比”等概念理解不一，导致需求错配。
*   **流程混沌：** 缺乏从需求分析到生产交付的标准化流程，设计决策随意，评审环节流于形式。
*   **质量黑盒：** 设计质量无法量化，阻抗不匹配、电源网络（PDN）崩溃、热管理失效等问题在样品制成后才暴露。
*   **成本失控：** 滥用高级过孔技术（如HDI堆叠），或因不合理的DFM参数导致制造成本飙升。

**机会与目标：**
本白皮书旨在为您的团队提供一个关于 **via types explained** 的完整基础体系。我们的目标是交付一套可复制、可衡量、可优化的PCB设计框架，将“过孔”作为切入点，辐射整个设计流程。通过建立统一的术语、标准化的流程、清晰的质量门槛和高效的协同方法，您的团队将能够：
*   **提升一次成功率：** 显著降低因设计失误导致的返工次数。
*   **控制研发成本：** 在设计阶段就做出兼顾性能与成本的最佳决策。
*   **加速产品上市：** 缩短从设计到原型、再到量产的周期。
*   **建立团队知识库：** 沉淀设计规范，赋能新成员快速融入。

本白皮书将作为您的行动指南，帮助您搭建从概念到可靠硬件的坚实桥梁。

## 2. 术语与结构图谱：统一您的设计语言

“Via types explained”的第一步是建立通用语言。过孔不仅是连接不同导电层的“垂直通道”，其类型、结构和参数直接决定了PCB的密度、性能、成本和可靠性。下表梳理了核心过孔术语，是团队内部沟通和与供应商协作的基础。

| 名词 (Term) | 定义 (Definition) | 对设计/成本的影响 (Impact on Design/Cost) | 参考标准/场景 (Reference/Scenario) |
| :--- | :--- | :--- | :--- |
| **通孔 (Through-hole Via)** | 贯穿PCB所有导电层的机械钻孔，从顶层连接到底层。 | 最常见，成本最低，工艺最成熟。但占用所有层布线空间，不利于高密度设计。 | 适用于[单双面板](https://hilpcb.com/en/products/single-double-layer-pcb)和大多数标准多层板。 |
| **盲孔 (Blind Via)** | 从PCB的一个外层连接到一个或多个内层，不贯穿整个板。 | 节省内层布线空间，提高布线密度。成本高于通孔，需要额外的钻孔和电镀工序。 | BGA扇出、高密度连接器区域。 |
| **埋孔 (Buried Via)** | 仅连接PCB内部两个或多个导电层，从外部不可见。 | 极大化利用表层和内层空间，是实现高密度互连（HDI）的关键。成本最高，工艺复杂。 | [HDI PCB](https://hilpcb.com/en/products/hdi-pcb)设计、复杂数字系统核心板。 |
| **微孔 (Microvia)** | 通常指直径小于150μm（6mil）的过孔，多采用激光钻孔。 | 实现极高密度布线，改善高频信号性能（寄生参数小）。是HDI技术的核心。 | 智能手机、IC基板、高速通信设备。 |
| **盘中孔 (Via-in-Pad, VIP)** | 将过孔直接放置在SMD元件的焊盘上，通常用树脂填充并电镀填平。 | 缩短信号路径，改善散热，节省空间，是BGA布局的理想选择。增加制造成本和工艺要求。 | 0.5mm间距以下的BGA、QFN封装。 |
| **深宽比 (Aspect Ratio)** | 过孔深度（板厚）与钻孔直径的比值。 | 影响电镀质量的关键参数。比值过高（如>10:1）会导致孔内电镀不均，可靠性下降。 | 制造商通常对不同板厚有最大深宽比限制，设计前必须确认。 |
| **堆叠过孔 (Stacked Via)** | 在不同层压结构中，将盲孔或微孔直接堆叠在一起。 | 提供最直接的垂直互连，空间利用率最高。对对准精度和填充工艺要求极高，成本昂贵。 | 高端HDI设计，用于连接多个层级的微孔。 |
| **交错式过孔 (Staggered Via)** | 在不同层压结构中，将盲孔或微孔错开位置连接。 | 相对于堆叠过孔，工艺难度和成本较低，可靠性更高，是更常用的HDI互连方式。 | 大多数HDI设计中的层间连接。 |
| **散热过孔 (Thermal Via)** | 放置在发热元件焊盘下方或周围，用于将热量传导至内层或背面的散热平面。 | 改善器件的热性能，防止过热。通常需要较大的孔径和多个并联。 | 功率器件、大电流IC、LED。 |

## 3. 流程框架：从需求到交付的可复制路径

一个成功的PCB设计项目依赖于结构化的流程。我们将PCB设计流程分解为六个关键阶段，并明确在每个阶段如何决策和管理过孔。

**阶段一：需求分析 (Requirement Analysis)**
*   **核心任务：** 定义产品的功能、性能、成本和尺寸目标。
*   **过孔决策：**
    *   **信号类型：** 是否包含高速信号（如PCIe, DDR）？这决定了是否需要阻抗控制和微孔。
    *   **元件密度：** 是否使用BGA、QFN等高密度封装？这直接导向盲埋孔或盘中孔的需求。
    *   **成本目标：** 严格的成本限制可能意味着只能使用通孔。
    *   **物理约束：** 产品厚度限制会影响深宽比，进而影响最小孔径。

**阶段二：栈叠设计 (Stackup Design)**
*   **核心任务：** 与制造商协同，定义PCB的层数、材料、层压结构和阻抗模型。这是整个设计的基础。
*   **过孔决策：**
    *   **层压结构：** 确定盲孔和埋孔的实现方式（如1-2层盲孔，2-5层埋孔）。
    *   **材料选择：** 高频/高速设计需要低损耗材料，这会影响钻孔参数。
    *   **协同供应商：** 此阶段必须与 **HilPCBPCB Factory (HILPCB)** 这样的专业制造商沟通。**HILPCB Stackup** 仿真服务可以帮助您在设计初期就确定最佳的层压方案和过孔策略，避免后期因可制造性问题而推倒重来。

**阶段三：布局 (Component Placement)**
*   **核心任务：** 根据电路原理图和信号流，合理放置元器件。
*   **过孔决策：**
    *   **关键器件优先：** 优先布局BGA、连接器等，并规划其扇出（Fanout）所需的过孔类型（如微孔、盘中孔）。
    *   **电源与地：** 规划去耦电容的位置，并预留足够的接地和电源过孔。
    *   **散热考量：** 在功率器件下方预先放置散热过孔阵列。

**阶段四：布线 (Routing)**
*   **核心任务：** 连接所有网络，实现电气功能。
*   **过孔决策：**
    *   **信号完整性：** 高速差分对走线尽量减少过孔使用，若必须使用，应在旁边放置接地过孔以保证回流路径连续。
    *   **电源完整性：** 使用多个过孔连接电源和地平面，以降低PDN阻抗。
    *   **过孔类型应用：** 根据预设规则，在不同区域使用通孔、盲孔或埋孔。

**阶段五：评审 (Review)**
*   **核心任务：** 通过自检、互检和自动化工具检查，确保设计符合所有规则。
*   **过孔检查清单：**
    *   **DRC (Design Rule Check)：** 检查过孔间距、焊盘大小、钻孔到铜的安全距离。
    *   **DFM (Design for Manufacturability)：** 检查深宽比、最小环宽（Annular Ring）、泪滴添加等。
    *   **信号/电源完整性仿真：** 检查过孔对高速信号和电源网络的影响。

**阶段六：交付 (Delivery)**
*   **核心任务：** 生成完整、清晰、无歧义的生产资料包。
*   **过孔相关文档：**
    *   **Gerber Files：** 包含所有铜层、阻焊层、丝印层。
    *   **钻孔文件 (Drill Files)：** 这是最容易出错的地方。必须为不同类型的钻孔（如通孔、盲孔L1-L2、埋孔L2-L3）生成独立的NC Drill文件，并附上清晰的钻孔图纸和说明。
    *   **叠层图 (Stackup Drawing)：** 明确标注每一层的材料、厚度以及盲埋孔的起止层。

## 4. 材料/层叠标准库：构建您的设计基线

为避免重复决策，建议团队建立一个标准化的材料与层叠库。这不仅能加速新项目启动，还能确保设计的一致性和可预测性。

| 层数 (Layers) | 典型应用 (Application) | 推荐过孔策略 (Recommended Via Strategy) | 核心材料 (Material) | 注意事项/成本影响 (Notes/Cost Impact) |
| :--- | :--- | :--- | :--- | :--- |
| **1-2层** | 简单控制板、电源模块、物联网终端 | 仅通孔 (Through-hole only) | FR-4 (Standard Tg) | 成本最低，工艺最简单。是学习 `double layer pcb tutorial` 的理想起点。 |
| **4层** | 微控制器系统、带阻抗控制的简单数字电路 | 优先使用通孔。仅在极高密度区域考虑盲孔。 | FR-4 (Standard/Mid Tg) | 引入地平面和电源平面，极大改善EMI和信号质量。成本适中。 |
| **6-8层** | 复杂嵌入式系统、FPGA核心板、高速接口 | 通孔为主，按需使用盲/埋孔进行BGA扇出。 | FR-4 (High Tg) / 中等损耗材料 | 设计灵活性高，是性能与成本的平衡点。适合大多数[多层PCB](https://hilpcb.com/en/products/multilayer-pcb)应用。 |
| **10+层 (HDI)** | 高速通信、服务器主板、智能手机 | 激光微孔、盲/埋孔、盘中孔成为标配。常采用堆叠或交错结构。 | 低损耗/极低损耗材料 (e.g., Rogers, Megtron) | 成本高，对设计和制造能力要求极高。需与制造商深度合作。 |

## 5. 模块化布局/布线规则

针对不同功能的电路模块，采用差异化的过孔策略是保证性能的关键。

*   **时钟与高速信号模块 (Clock & High-Speed)**
    *   **规则1：** 尽可能减少过孔数量。每个过孔都是一个阻抗不连续点。
    *   **规则2：** 差分对换层时，必须成对使用过孔，并保持对称。
    *   **规则3：** 在高速信号过孔旁放置接地过孔（Stitching Via），为信号提供最短的回流路径。
    *   **工具支持：** 使用 **HILPCB** 的在线阻抗计算器来精确计算过孔对传输线的影响。

*   **电源网络模块 (Power Delivery Network - PDN)**
    *   **规则1：** 在电源输入端、IC电源引脚和去耦电容附近，使用过孔阵列（多个过孔并联）连接电源和地平面，以降低电感。
    *   **规则2：** 避免在电源路径上共享过孔，特别是大电流路径。
    *   **规则3：** 对于发热元件，在其下方的散热焊盘上均匀分布散热过孔，并确保它们连接到大面积的散热铜皮。

*   **模拟信号模块 (Analog Signals)**
    *   **规则1：** 模拟信号路径应短而直，严格避免使用过孔。
    *   **规则2：** 模拟地和数字地应单点连接，通过过孔连接到各自的接地平面，避免噪声耦合。
    *   **规则3：** 敏感的模拟信号走线周围应布设屏蔽地线，并用接地过孔将其缝合到地平面。

*   **大功率/大电流模块 (High Power/Current)**
    *   **规则1：** 使用更大孔径和焊盘的过孔，或多个过孔并联来承载大电流。
    *   **规则2：** 考虑使用填铜（Copper-filled）或塞树脂（Resin-filled）过孔，以增加载流量和散热能力。
    *   **规则3：** 确保大电流过孔的散热路径通畅，避免局部过热。

## 6. 文档清单：确保无歧义的制造交付

清晰的文档是设计意图得以精确实现的保证。这是 `gerber file basics` 的延伸，一份专业的交付包应包含以下内容：

*   **Gerber Files (RS-274X or X2):**
    *   [ ] 所有铜层 (Top, Bottom, Inner Layers)
    *   [ ] 阻焊层 (Top/Bottom Solder Mask)
    *   [ ] 丝印层 (Top/Bottom Silkscreen)
    *   [ ] 锡膏层 (Top/Bottom Solder Paste)
    *   [ ] 边界/轮廓层 (Board Outline)
*   **钻孔文件 (NC Drill Files):**
    *   [ ] `drill_L1-LN.drl` (通孔)
    *   [ ] `drill_L1-L2.drl` (盲孔)
    *   [ ] `drill_L2-L9.drl` (埋孔) - **为每一种盲/埋孔组合生成独立文件**
    *   [ ] 钻孔图纸 (Drill Drawing/Map)，清晰标注孔径、数量和公差。
*   **物料清单 (Bill of Materials - BOM):**
    *   [ ] 包含元件位号、型号、封装、数量和描述。
*   **装配图 (Assembly Drawing):**
    *   [ ] 标注元件位号、方向（特别是极性元件）。
*   **叠层图 (Stackup Drawing):**
    *   [ ] 必须包含层序、材料类型、介电常数(Dk)、厚度、铜厚和目标阻抗。
*   **制造说明 (Fabrication Notes):**
    *   [ ] 明确板材、表面处理、阻焊颜色、公差要求等。
    *   [ ] 特别说明盘中孔的处理方式（如：树脂填充、电镀填平）。

## 7. KPI 仪表：量化您的设计质量

为了持续改进，必须对设计过程和结果进行量化。以下KPI可作为团队的仪表盘。

| KPI 指标 | 定义 (Definition) | 目标 (Target) | 衡量方法 (How to Measure) |
| :--- | :--- | :--- | :--- |
| **一次交付率 (First Pass Yield)** | 设计文件首次提交给制造商，无需任何工程问题(EQ)澄清或修改即通过审核的比例。 | > 95% | (通过审核的项目数 / 总提交项目数) * 100% |
| **DRC/DFM 错误数 (DRC/DFM Error Count)** | 在最终提交前，内部评审或自动化工具发现的关键设计规则错误数量。 | 0 (关键错误) | 运行DRC/DFM检查工具，统计错误报告。 |
| **阻抗命中率 (Impedance Match Rate)** | 生产出的PCB上，受控阻抗线的实际测试值落在规格（如±10%）内的比例。 | > 98% | 查看制造商提供的TDR测试报告（阻抗Coupon）。 |
| **设计迭代次数 (Number of Revisions)** | 从初版设计到最终生产版本，设计文件经历的主要修改次数。 | < 2 次/项目 | 版本控制系统（如Git）的提交历史。 |

## 8. DFM/DFT 清单与 HILPCB 协同服务

将可制造性（DFM）和可测试性（DFT）融入设计是 `pcb design checklist beginner` 的核心。以下清单（超过35项）是您团队的最后一道防线。

| 类别 | 检查项 | 建议/规则 | 优先级 |
| :--- | :--- | :--- | :--- |
| **过孔基础 (Via Basics)** | 最小钻孔尺寸 | 机械钻孔 ≥ 0.2mm，激光钻孔 ≥ 0.1mm (与供应商确认) | 高 |
| | 最小环宽 (Annular Ring) | 外层 ≥ 0.15mm，内层 ≥ 0.175mm (IPC Class 2) | 高 |
| | 深宽比 (Aspect Ratio) | 通孔 < 10:1，盲孔 < 1:1 (与供应商确认) | 高 |
| | 钻孔到铜间距 | ≥ 0.2mm，避免钻孔偏位导致短路 | 高 |
| | 过孔到过孔间距 | ≥ 0.3mm (孔边到孔边) | 中 |
| **盲/埋孔 (Blind/Buried Vias)** | 盲孔起止层 | 必须从外层开始 | 高 |
| | 埋孔起止层 | 必须在内层之间 | 高 |
| | 堆叠微孔 | 避免超过3层堆叠，优先使用交错式 | 高 |
| | 激光钻孔材料 | 确保激光可烧蚀所选介质材料 | 高 |
| **盘中孔 (Via-in-Pad)** | 填充要求 | 必须在制造说明中指定“树脂填充并电镀填平” | 高 |
| | 焊盘表面平整度 | 要求 < 25μm，防止焊接缺陷 | 高 |
| | 适用场景 | 仅用于0.65mm间距以下的BGA/QFN | 中 |
| **散热设计 (Thermal Design)** | 散热过孔孔径 | 0.3mm - 0.5mm，以平衡散热和防锡膏渗漏 | 中 |
| | 散热过孔间距 | 1.0mm - 1.2mm，均匀分布 | 中 |
| | 散热连接 (Thermal Relief) | 接地过孔应全连接到平面，电源过孔可使用十字花焊盘 | 高 |
| | 防焊开窗 (Solder Mask Opening) | 散热过孔阵列上方可开窗以增强散热，但需评估风险 | 低 |
| **DFM通用规则** | 泪滴 (Teardrops) | 在焊盘和过孔与走线连接处添加，防止钻偏断裂 | 高 |
| | 非功能焊盘 (NPTH) | 移除内层未连接过孔的焊盘，增加布线空间 | 中 |
