## component placement basics 需要先理解哪些部件与术语？

**提问：** 在我们开始布局元器件之前，一块空白的 PCB 板到底是由什么构成的？那些专业术语（比如“过孔”、“焊盘”）又是什么意思？

**解释：** 一块看似简单的 PCB，其实是多层材料精密压合的产物。理解它的基本解剖结构，是掌握 component placement basics 的第一步。

- **基板 (Substrate):** 这是 PCB 的骨架，通常是玻璃纤维环氧树脂材料，最常见的就是 **FR-4**。它提供了绝缘和机械支撑。
- **铜箔 (Copper Foil):** 附着在基板上的薄铜层，通过蚀刻工艺形成我们需要的导线（走线）和连接点。
- **阻焊层 (Solder Mask):** 覆盖在铜箔上的一层绝缘保护漆，通常是绿色，但也有红、蓝、黑等多种颜色。它的作用是防止铜线氧化，以及在焊接时避免不必要的焊锡连接（焊锡桥）。
- **丝印层 (Silkscreen):** 在阻焊层之上，用白色或其它颜色的油墨印刷的文字、符号和元器件轮廓。它用于标注元器件的位号（如 R1, C1, U1）、引脚功能和公司 Logo，是组装和调试的重要参考。

这些基础层之上，还有一些关键的连接结构：

- **焊盘 (Pad):** 裸露的铜面，用于焊接元器件的引脚。
- **过孔 (Via):** 用于连接不同导电层的电镀孔。它是实现多层板复杂布线的关键。

**实操：** 为了让你快速进入状态，我们整理了一份核心术语速查表。在后续的设计中，你会频繁遇到它们。

<div class="div-type-1">

**PCB 术语速查表 (PCB Terminology Glossary)**

| 术语 (Term) | 定义 (Definition) | 常见误区/注意事项 |
| :--- | :--- | :--- |
| **走线 (Trace)** | PCB 上的铜质导线，用于连接不同元器件的焊盘。 | 走线宽度并非越细越好，需要根据电流大小来决定。 |
| **网络 (Net)** | 一组电气上相互连接的端点（焊盘）的集合。例如，GND 网络连接了所有需要接地的引脚。 | 在设计软件中，未连接的 Net 会在 DRC 检查时报错。 |

| **过孔 (Via)** | 连接不同铜层的电镀孔。最基本的是通孔 (Thru-hole Via)，贯穿整个板子。 | 过孔会引入额外的电感和电容，在高速信号中需谨慎使用。 |

| **年轮环 (Annular Ring)** | 过孔或焊盘周围的铜环。它确保了钻孔后仍有足够的铜与孔壁连接。 | 年轮环过小可能导致钻孔偏差后开路，是 DFM 检查的重点。 |

## 单/双层 PCB 是如何堆叠起来的？

**提问：** 我听过单面板、双面板和多层板，它们在结构上有什么区别？这对我布局元器件有影响吗？

**解释：** 当然有影响！板子的层数决定了你可以使用的布线空间和设计复杂度。对于初学者来说，从最基础的单/双层板 ([Single/Double Layer PCB](https://hilpcb.com/en/products/single-double-layer-pcb))入手是最合适的。

- **单层 PCB (Single Layer PCB):** 结构最简单，只有一面有铜箔走线。所有元器件和走线都在同一层，设计约束大，常用于遥控器、计算器等简单电路。这是 `single layer pcb basics` 的核心。
- **双层 PCB (Double Layer PCB):** 在基板的两面都有铜箔层，即顶层 (Top Layer) 和底层 (Bottom Layer)。元器件可以放置在任一层，通过过孔 (Via) 将两层的走线连接起来，极大地增加了布线灵活性。这是目前业余制作和大多数消费电子产品的首选。
- **多层 PCB (Multilayer PCB):** 在双层板的基础上，内部增加更多的导电层和绝缘层。例如，四层板通常包含一个顶层、一个底层、一个接地平面和一个电源平面。它能提供更好的信号完整性和更紧凑的布局。

## 材料、铜厚与焊油：它们影响什么？

**提问：** 为什么大多数板子是绿色的？1oz 铜厚是什么意思？这些选择对我的设计重要吗？

**解释：** 材料选择是设计中一个容易被忽略但至关重要的环节。它直接关系到电路板的性能、成本和可靠性。

- **基板材料 (Substrate Material):** [FR-4](https://hilpcb.com/en/products/fr4-pcb) 是性价比最高的通用选择，适用于绝大多数数字和低频模拟电路。如果你的设计涉及高频信号（如 WiFi、射频），则需要考虑 Rogers 或 Teflon 等具有更低介电损耗的特殊材料。
- **铜厚 (Copper Weight):** 通常用“盎司/平方英尺 (oz/ft²)”来表示。1oz 铜厚意味着 1 平方英尺的面积上均匀分布着 1 盎司的铜，厚度约为 35μm。铜层越厚，能承载的电流就越大。标准 PCB 通常使用 1oz 或 2oz 铜厚。电源电路或大功率设备可能需要更厚的铜。
- **阻焊层/焊油 (Solder Mask):** 它的主要功能是保护和绝缘。颜色方面，绿色是最常见、工艺最成熟、成本最低的选择。其他颜色（如红、蓝、黑、白）在功能上没有本质区别，但黑色或白色的板子在自动光学检测 (AOI) 时可能对对比度要求更高。
- **丝印 (Silkscreen):** 丝印的清晰度非常重要。遵循基本的 `silkscreen design rules`，例如确保丝印线宽不小于 6mil (0.15mm)，且与焊盘保持至少 6mil 的间距，可以避免生产时出现模糊不清或印在焊盘上的问题。

<div class="div-type-1">

#### 知识要点：如何做出明智选择

*   **预算有限、常规应用？** 选择标准的 1.6mm 板厚、FR-4 材料、1oz 铜厚、绿色阻焊层。这是最具成本效益的组合。
*   **设计中有大电流通过？** 考虑将电源走线加宽，或选择 2oz 及以上的铜厚。
*   **需要清晰的标识？** 确保你的丝印字体足够大，线条足够粗，并远离焊盘。
*   **不确定如何选择？** 在项目初期，可以向 HILPCB 的工程师咨询，他们会根据你的应用推荐最合适的层叠和材料方案。

## 从原理图到 PCB：Netlist、封装与 DRC

**提问：** 我已经在软件里画好了电路原理图，下一步该如何开始 PCB 设计？

**解释：** 从原理图到 PCB 版图的转换，是整个流程中最关键的“翻译”工作。这个过程被称为 `pcb schematic to layout`，它依赖几个核心概念来确保逻辑正确性。

1.  **原理图 (Schematic):** 这是电路的逻辑蓝图，定义了元器件之间如何连接，但不关心它们的物理位置。在这一步，你需要确保所有元器件都有正确的型号和参数。
2.  **封装 (Footprint):** 每个原理图符号都需要关联一个物理封装。例如，一个电阻的原理图符号是通用的，但它的封装可能是 0603 贴片封装，也可能是 DIP 插针封装。选错封装将导致元器件无法焊接。
3.  **网络表 (Netlist):** 当你将原理图导入 PCB 编辑器时，软件会生成一个网络表。这个文件本质上是一个连接指令清单，它告诉 PCB 编辑器：“U1 的第 3 脚应该和 R1 的第 1 脚连接在一起”。
4.  **设计规则检查 (DRC - Design Rule Check):** 这是你的“语法检查器”。你需要根据制造商的工艺能力，设定一系列规则，如最小线宽、最小间距、过孔尺寸等。设计过程中，DRC 会实时或手动检查你的布局布线是否违反了这些规则。

<div class="div-type-3">

#### 操作步骤：从原理图到 PCB 布局的准备工作

1.  **完成并检查原理图：** 运行电气规则检查 (ERC)，确保没有悬空的引脚或未连接的网络。
2.  **分配所有封装：** 仔细核对数据手册 (Datasheet)，为每一个元器件指定正确的 PCB 封装。建议建立自己的常用封装库。
3.  **定义板框：** 在 PCB 编辑器中，首先画出电路板的物理边界（Board Outline）。
4.  **导入网络表：** 将原理图信息导入 PCB 编辑器。此时，你会看到所有元器件的封装堆叠在一起，并由一些细线（飞线，Ratsnest）连接，这些飞线代表了网络表中的连接关系。
5.  **设置设计规则 (DRC)：** 这是最重要的一步！访问你的制造商网站（例如 HILPCB 的能力参数页面），将最小线宽、间距、过孔尺寸等关键参数填入 DRC 设置中。

</div>

## 布局布线的三步法（模块→走线→检查）

**提问：** 面对一堆由飞线连接的元器件，我该从何下手？有没有一套行之有效的方法？

**解释：** 这正是 component placement basics 的核心。一个混乱的开始会导致后续布线困难重重。我们推荐采用“模块化布局，关键优先布线，持续检查”的三步法。

### 第一步：布局 (Placement) - 城市规划

这是最具创造性也最关键的一步。目标是合理地放置所有元器件，让飞线尽可能短而直接，减少交叉。

- **按功能模块分组：** 将电路中实现特定功能的元器件（如电源模块、MCU 核心模块、传感器接口模块）集中放置。
- **固定位置优先：** 首先放置有位置要求的元器件，如 USB 接口、按钮、LED 指示灯、安装孔等。
- **核心元件居中：** 将核心控制器（如 MCU 或 CPU）放置在板子中央或一个合适的位置，方便与其他模块连接。
- **考虑信号流向：** 尽量让信号从输入到输出形成一个平滑的路径，避免信号路径迂回。
- **热管理：** 将发热量大的元器件（如 LDO、功率MOS管）放置在利于散热的位置，周围留出足够空间，并可以考虑铺设大面积铜皮辅助散热。

### 第二步：布线 (Routing) - 修建道路

当元器件位置基本确定后，就可以开始将飞线变成实际的铜走线了。

- **关键网络优先：** 首先布设最重要的网络。
    - **电源和地线 (Power/Ground):** 尽量使用短而粗的走线，或直接使用大面积的铜平面（Pour/Plane）。
    - **高速信号：** 如时钟线、USB 差分线，应保持短、直，并远离干扰源。
    - **敏感模拟信号：** 同样需要短路径，并与数字信号隔离。
- **走线宽度：** 根据电流大小设置。信号线通常使用 8-10mil，电源线则根据电流大小，可能需要 20mil、50mil 甚至更宽。
- **过孔的使用 (`via types explained`):** 对于双层板，通孔 (Thru-hole Via) 是连接顶层和底层的唯一方式。尽量减少在高速信号路径上使用过孔，因为它会引入阻抗不连续。对于更复杂的 HDI 板，还有盲孔（Blind Via）和埋孔（Buried Via），但初学者暂时无需深入。

### 第三步：检查与优化 (Review & Refine)

布局布线是一个迭代的过程，需要不断检查和优化。

- **频繁运行 DRC：** 每完成一小部分布线，就运行一次 DRC，及时发现并修正问题。
- **丝印整理：** 调整元器件位号的位置，确保清晰可读，没有压在焊盘或过孔上。
- **铺铜 (Copper Pour):** 在空白区域大面积铺设地平面 (GND Pour)。这能提供良好的信号回流路径，增强抗干扰能力，并有助于散热。
- **最终检查：** 从头到尾检查一遍原理图和 PCB，确保完全一致。使用 3D 预览功能检查元器件的实际高度和位置是否会发生干涉。

## 输出 Gerber、钻孔、BOM 的标准流程

**提问：** 我的设计完成了！我应该给工厂发送哪些文件，他们才能帮我生产出来？

**解释：** 为了让任何工厂都能准确无误地理解你的设计，业界建立了一套标准的文件格式，其中最核心的就是 Gerber 文件。你需要从你的 EDA 软件中导出一整套生产资料。

<div class="div-type-1">

#### 生产文件交付清单

*   **Gerber Files (RS-274X 格式):**
    *   `GTL`: 顶层铜箔 (Top Copper)
    *   `GBL`: 底层铜箔 (Bottom Copper)
    *   `GTS`: 顶层阻焊 (Top Solder Mask)
    *   `GBS`: 底层阻焊 (Bottom Solder Mask)
    *   `GTO`: 顶层丝印 (Top Silkscreen)
    *   `GBO`: 底层丝印 (Bottom Silkscreen)
    *   `GKO` 或 `GML`: 板框/机械层 (Board Outline/Mechanical Layer)
*   **钻孔文件 (NC Drill File):**
    *   通常为 `.TXT` 或 `.DRL` 格式，包含了所有孔的坐标和尺寸信息。
*   **物料清单 (Bill of Materials - BOM):**
    *   一个 Excel 或 CSV 文件，列出了所有元器件的位号、型号、封装、数量和描述。这是进行一站式 PCBA 服务 ([Turnkey Assembly](https://hilpcb.com/en/products/turnkey-assembly)) 的必备文件。
*   **坐标文件 (Pick and Place File):**
    *   也叫 Centroid File，提供了每个贴片元件的中心坐标和旋转角度，用于自动化贴片机。
*   **装配图 (Assembly Drawing):** (可选，但推荐)
    *   一个 PDF 文件，清晰地展示了每个元器件在板上的位置和方向。

**实操建议：** 在导出文件后，强烈建议使用第三方的 Gerber 查看器 (Gerber Viewer)（例如 HILPCB 提供的免费在线工具）来逐层检查，确保所有信息都正确无误。

</div>
