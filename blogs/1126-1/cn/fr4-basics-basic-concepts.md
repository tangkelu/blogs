## FR-4 基础与材料特性 (FAQ #1-7)

### 1. 什么是 FR-4？它为什么如此普遍？

*   **问题：** 我总是听到 FR-4，它到底是什么？
*   **场景：** 一位初学者在选择 PCB 基板材料时，发现 90% 的选项都是 FR-4，想知道其原因。
*   **指标/示例：** FR-4 是一种由玻璃纤维布（Fiberglass）浸润环氧树脂（Epoxy Resin）后，在高温高压下固化而成的复合材料。其表面通常覆盖有铜箔。
*   **解决：** FR-4 之所以成为行业标准，是因为它在机械强度、电气绝缘性、耐热性、阻燃性和成本之间取得了绝佳的平衡。它足以满足消费电子、工业控制、通信设备等绝大多数应用的需求，并且制造工艺成熟，价格低廉。
*   **预防 (Checklist)：**
    *   [ ] 确认您的应用是否处于极端温度或超高频环境。
    *   [ ] 对于大多数标准项目，默认选择 FR-4 是最安全、最具成本效益的方案。

### 2. "FR-4" 中的 "FR" 和 "4" 分别代表什么？

*   **问题：** 这个名字听起来很技术化，它有什么特殊含义吗？
*   **场景：** 在阅读技术文档时，工程师希望理解术语的来源，以加深记忆。
*   **指标/示例：** FR = Flame Retardant (阻燃剂)。4 = NEMA (美国国家电气制造商协会) 定义的材料等级。
*   **解决：** "FR" 表示该材料含有溴化阻燃剂，符合 UL94V-0 的防火等级，意味着它在燃烧后能快速自熄，提高了产品的安全性。"4" 是一个等级代号，代表了其具体的材料构成（玻璃纤维布基环氧树脂）。
*   **预防 (Checklist)：**
    *   [ ] 检查产品安全认证要求，确认是否必须使用 UL94V-0 等级的材料。
    *   [ ] 如果您的产品销往有环保限制的地区（如欧盟 RoHS），可能需要考虑[无卤素 FR-4](https://hilpcb.com/en/products/halogen-free-pcb)。

### 3. 什么是玻璃化转变温度 (Tg)，为什么它对 FR-4 很重要？

*   **问题：** 我在材料数据表中看到了 "Tg" 这个参数，它是什么意思？
*   **场景：** 设计一个需要在较高温度下运行的电源模块，工程师需要确保 PCB 不会因受热而失效。
*   **指标/示例：** 标准 FR-4 的 Tg 值通常在 **130-140°C** 之间。中等 Tg 约为 150-160°C，高 Tg 则在 **170-180°C**。
*   **解决：** Tg 是材料从坚硬的玻璃态转变为柔软的橡胶态的温度点。当 PCB 工作温度超过 Tg 时，其机械强度、尺寸稳定性和电气性能会急剧下降，可能导致分层、弯曲或连接失效。因此，PCB 的最高工作温度应远低于材料的 Tg 值（通常建议至少有 20-25°C 的裕量）。
*   **预防 (Checklist)：**
    *   [ ] 评估产品的最大工作环境温度和板上元器件的产热。
    *   [ ] 如果工作温度持续高于 110°C 或进行多次无铅焊接，优先选择[高 Tg FR-4](https://hilpcb.com/en/products/high-tg-pcb)。

### 4. 标准 FR-4 和高 Tg FR-4 有什么区别？我该如何选择？

*   **问题：** 制造商提供了多种 Tg 等级的 FR-4，我该怎么选？
*   **场景：** 一个项目既要考虑成本，又要确保在汽车引擎舱附近工作的可靠性。
*   **指标/示例：**
    *   **标准 Tg FR-4 (≈135°C):** 适用于大多数消费电子产品，成本最低。
    *   **高 Tg FR-4 (≥170°C):** 适用于汽车、工业、高功率 LED 照明等。成本略高。
*   **解决：** 选择依据是**热应力**。高 Tg 材料不仅耐温更高，还具有更好的尺寸稳定性、抗化学性和耐潮湿性。当您的设计满足以下任一条件时，应选择高 Tg FR-4：
    1.  PCB 工作温度长时间接近或超过 120°C。
    2.  需要承受多次高温焊接过程（如复杂的 BGA 返修）。
    3.  高功率密度设计，局部热点温度高。
    4.  对尺寸稳定性要求极高的[多层 PCB](https://hilpcb.com/en/products/multilayer-pcb)。
*   **预防 (Checklist)：**
    *   [ ] 进行热仿真分析，确定板上的最高温度点。
    *   [ ] 与 HILPCB 工程师沟通您的应用场景，获取专业的材料建议。

### 5. 什么是 PCB 的“芯板 (Core)”和“半固化片 (Prepreg)”？

*   **问题：** 在看层叠结构图时，我看到了 Core 和 Prepreg，它们是什么？
*   **场景：** 设计一个四层板，需要理解各层是如何粘合在一起的。
*   **指标/示例：**
    *   **Core (芯板):** 一块双面覆铜的、已经固化好的 FR-4 板材，具有固定的厚度。
    *   **Prepreg (PP片/半固化片):** 未完全固化的 FR-4 玻璃纤维布，呈片状，在高温高压下会熔化并固化，起到粘合和绝缘的作用。
*   **解决：** 在制造多层板时，Core 和 Prepreg 交替堆叠。例如，一个典型的四层板结构是：铜箔 - Prepreg - Core (内层铜) - Prepreg - 铜箔。整个结构在层压机中被压合成一个整体。Core 的厚度和 Prepreg 的层数/型号共同决定了 PCB 的最终厚度和介电性能。
*   **预防 (Checklist)：**
    *   [ ] 在设计文件中明确定义层叠结构，或让 HILPCB 根据您的要求（如总厚度、阻抗）推荐标准层叠。
    *   [ ] 不要假设层间距是均匀的，它由 Prepreg 的厚度决定。

### 6. copper weight explanation：什么是铜重？1oz 铜厚是多少？

*   **问题：** 订单选项里有 1oz、2oz 铜，这是什么单位？
*   **场景：** 设计一个大电流的电机驱动板，需要确保走线不会过热烧毁。
*   **指标/示例：** 铜重（Copper Weight）指的是每平方英尺（ft²）面积上铜箔的重量（盎司，oz）。
    *   **1 oz (标准):** ≈ 35 µm (1.37 mil) 厚。适用于大多数信号线和低功率电源线。
    *   **2 oz:** ≈ 70 µm (2.74 mil) 厚。适用于中等功率应用。
    *   **≥ 3 oz:** 称为[重铜 PCB](https://hilpcb.com/en/products/heavy-copper-pcb)，用于大电流场景。
*   **解决：** 铜重直接决定了走线的载流能力和散热性能。铜越厚，电阻越小，能承载的电流越大。您可以使用 PCB Trace Width Calculator 等工具，根据电流、温升和走线长度来计算所需的线宽和铜厚。
*   **预防 (Checklist)：**
    *   [ ] 评估设计中每条电源路径的最大电流。
    *   [ ] 对于超过 2A 的电流，考虑使用 2oz 或更厚的铜。
    *   [ ] 在制造文件中明确标注内外层的铜重。

### 7. 如何选择合适的 PCB 厚度？标准厚度是多少？

*   **问题：** PCB 可以做多厚？有没有标准？
*   **场景：** 设计一个需要插入标准 PCIe 插槽的板卡，必须符合厚度规范。
*   **指标/示例：**
    *   **最常用标准厚度:** 1.6 mm (0.062 inches)。
    *   **其他常见厚度:** 0.8 mm, 1.0 mm, 1.2 mm, 2.0 mm。
*   **解决：** PCB 厚度的选择取决于多种因素：
    1.  **机械强度:** 薄板更易弯曲，厚板更坚固。
    2.  **标准化接口:** 如 PCIe (1.57mm)、USB-A 插头等有严格的厚度要求。
    3.  **层数和阻抗:** 更多的层数或特定的阻抗控制要求可能需要非标准厚度。
    4.  **重量和空间:** 在便携设备中，更薄的板材可以节省空间和重量。
*   **预防 (Checklist)：**
    *   [ ] 检查是否有任何机械约束或接口标准限制了 PCB 厚度。
    *   [ ] 如果没有特殊要求，选择 1.6mm 可以获得最快的交期和最低的成本。
    *   [ ] 如果需要特殊厚度，请提前与 HILPCB 确认材料可用性。

## PCB 设计与布局 (FAQ #8-15)

### 8. pcb stack layer naming：PCB 层叠的命名规则是怎样的？

*   **问题：** 我在 EDA 软件里看到 L1, L2, L3...，它们通常如何命名？
*   **场景：** 一位工程师正在设置一个四层板项目，不确定如何为内层分配功能和命名。
*   **指标/示例：** 一个经典的四层板命名和功能分配：
    *   **L1:** Top (或 Signal 1) - 主要布线层和元件放置层。
    *   **L2:** GND (或 Inner 1) - 地平面，提供屏蔽和回流路径。
    *   **L3:** Power (或 Inner 2) - 电源平面，分配 VCC、3.3V 等。
    *   **L4:** Bottom (或 Signal 2) - 辅助布线层。
*   **解决：** 清晰的层命名至关重要。虽然没有强制的全球标准，但最佳实践是根据**功能**来命名。例如 `Top_Signal`, `Inner_GND`, `Inner_Power`, `Bottom_Signal`。将电源和地平面放在内层可以为外层的高速信号提供良好的参考平面，改善信号完整性并降低 EMI。
*   **预防 (Checklist)：**
    *   [ ] 在设计开始前就规划好层叠结构和每一层的主要功能。
    *   [ ] 在 Gerber 文件和制造说明中，提供一份清晰的层叠图和命名列表。
    *   [ ] HILPCB 的工程师可以帮助您审核和优化多层板的层叠设计。

### 9. soldermask purpose：阻焊层（绿油）的作用是什么？为什么大多是绿色的？

*   **问题：** PCB 上那层绿色的涂层是什么？可以换成其他颜色吗？
*   **场景：** 设计师希望产品的 PCB 是黑色的以匹配外壳，想确认功能上是否有影响。
*   **指标/示例：** 阻焊层（Solder Mask）是一种环氧树脂基的永久性保护涂层。
*   **解决：** 阻焊层主要有两大作用：
    1.  **定义焊接区域：** 它覆盖所有不需要焊接的铜箔（如走线），只露出需要焊接的焊盘。这在波峰焊或回流焊中可以有效防止焊锡流动到错误的地方，避免“桥连”或短路。
    2.  **环境保护：** 保护铜走线免受氧化、湿气、灰尘和物理划伤。
    **为什么是绿色？** 绿色对人眼最友好，检查人员在显微镜下检查时不易疲劳。同时，绿色的对比度能清晰地突显出走线、焊盘和丝印，便于检查缺陷。不过，HILPCB 也提供多种颜色选择，如红色、蓝色、黑色、白色等，功能上没有差异，但黑色和白色对检查精度要求更高。
*   **预防 (Checklist)：**
    *   [ ] 检查阻焊层开窗（Solder Mask Opening）是否比焊盘略大（通常单边扩大 2-4 mil），以确保焊盘完全暴露。
    *   [ ] 如果选择非绿色阻焊，请与 HILPCB 确认最小工艺能力，特别是对于高密度设计。

### 10. silkscreen design rules：丝印层设计有哪些基本规则？

*   **问题：** 我想在板上印上公司 Logo 和元件位号，有什么要注意的吗？
*   **场景：** 一位硬件工程师在完成布局后，开始添加丝印标识，希望避免制造问题。
*   **指标/示例：**
    *   **最小线宽:** ≥ 6 mil (0.15 mm)
    *   **最小字符高度:** ≥ 32 mil (0.8 mm)
    *   **宽高比:** 约 1:5 到 1:7
*   **解决：** 丝印（Silkscreen）用于在 PCB 表面印刷元件位号、引脚标识、公司 Logo、版本号等信息。遵循以下基本规则可确保清晰可读：
    1.  **禁止上焊盘：** 丝印不能覆盖任何焊盘，否则会影响焊接质量。EDA 软件通常有自动裁剪功能。
    2.  **尺寸清晰：** 字符和线条不能太细太小，否则会模糊不清。
    3.  **方向一致：** 尽量保持元件位号的方向一致，便于阅读和调试。
    4.  **位置明确：** 将位号放在对应元件旁边，避免歧义。
*   **预防 (Checklist)：**
    *   [ ] 在导出 Gerber 前，运行设计规则检查 (DRC)，检查丝印与焊盘的间距。
    *   [ ] 使用 HILPCB 的 Gerber Viewer 预览丝印层的最终效果。
    *   [ ] 避免在过孔或测试点上放置丝印。

### 11. annular ring definition：什么是环形环（Annular Ring）？为什么它很重要？

*   **问题：** DFM 报告里提到了 "Annular Ring Breakout"，这是什么问题？
*   **场景：** 工程师设计的过孔焊盘非常小，制造商警告说可能存在可靠性风险。
*   **指标/示例：** 环形环（Annular Ring）是过孔或元件引脚孔周围的铜环。其宽度 = (焊盘直径 - 钻孔直径) / 2。HILPCB 推荐的最小环形环宽度为 **0.15mm (6mil)**。
*   **解决：** 环形环的作用是确保钻孔后，孔壁与导电层之间有足够面积的铜连接。由于机械钻孔存在公差，如果环形环设计得太小，钻孔偏位可能导致铜环被钻断（Breakout），造成开路或连接不可靠。一个完整的、足够宽的环形环是保证过孔电气和机械连接强度的关键。
*   **预防 (Checklist)：**
    *   [ ] 在 PCB 设计库中，确保过孔和通孔元件的焊盘尺寸足够大。
    *   [ ] 运行 DFM 检查，特别关注环形环宽度是否满足制造商的最小要求。
    *   [ ] 对于关键信号或电源过孔，适当增加焊盘尺寸。

### 12. 什么是过孔（Via）？盲孔、埋孔和通孔有什么区别？

*   **问题：** 我需要在不同层之间连接信号，应该用哪种过孔？
*   **场景：** 在设计一个紧凑的高密度互连 (HDI) PCB 时，工程师需要利用盲孔和埋孔来节省空间。
*   **指标/示例：**
    *   **通孔 (Through-hole Via):** 贯穿整个 PCB，连接所有层。成本最低，最常用。
    *   **盲孔 (Blind Via):** 从外层连接到一个或多个内层，不穿透整个板。
    *   **埋孔 (Buried Via):** 只连接两个或多个内层，从外部完全看不到。
*   **解决：**
    *   **通孔**是标准选择，适用于大多数单双面板和层数不多的多层板。
    *   **盲孔和埋孔**主要用于 HDI 板，可以在不占用其他层空间的情况下增加布线密度。例如，BGA 下方的扇出通常会使用盲孔。然而，它们的制造过程更复杂，成本也更高。
*   **预防 (Checklist)：**
    *   [ ] 除非空间极其有限，否则优先使用通孔以控制成本。
    *   [ ] 如果需要使用盲/埋孔，必须提前与 HILPCB 沟通，确认其工艺能力和层叠设计。

### 13. 什么是表面贴装技术 (SMT) 和通孔插装技术 (THT)？

*   **问题：** 我的元件清单里有些是 SMT 的，有些是 THT 的，它们有什么不同？
*   **场景：** 为一个项目选型时，需要决定是使用传统的 DIP 封装芯片还是现代的 QFP 封装。
*   **指标/示例：**
    *   **SMT (Surface Mount Technology):** 元件直接焊接到 PCB 表面的焊盘上。
    *   **THT (Through-Hole Technology):** 元件引脚穿过 PCB 上的钻孔，在另一面焊接。
*   **解决：** SMT 是现代电子制造的主流，因为它尺寸小、密度高，适合自动化生产（如SMT 贴片组装）。THT 元件尺寸较大，但机械连接更牢固，通常用于连接器、大电容、变压器等需要承受机械应力的元件。一个 PCB 设计通常是两者混合使用。
*   **预防 (Checklist)：**
    *   [ ] 根据元件的电气功能、机械强度要求和成本选择合适的封装类型。
    *   [ ] 在 BOM 中清晰标注每个元件的封装信息。
    *   [ ] 如果需要 HILPCB 提供一站式 PCBA 服务，确保两种类型的元件都已正确指定。

### 14. 什么是阻抗控制？为什么高速电路需要它？

*   **问题：** 制造商问我哪些走线需要做 50 欧姆阻抗控制，这是什么意思？
*   **场景：** 设计一个包含 USB 3.0 接口的电路板，数据手册要求差分线对阻抗为 90 欧姆。
*   **指标/示例：** 常见阻抗值：单端 **50Ω** (如天线、时钟线)，差分 **90Ω** (USB) 或 **100Ω** (Ethernet, HDMI)。
*   **解决：** 当信号频率很高时（通常 >100MHz），PCB 走线不再仅仅是简单的导线，而变成了具有特定特性阻抗的传输线。如果走线的阻抗与源端和终端的阻抗不匹配，就会发生信号反射，导致信号失真、振铃和数据错误。阻抗控制就是通过精确控制走线宽度、介质厚度（FR-4）和参考平面距离，使走线达到一个特定的阻抗值。
*   **预防 (Checklist)：**
    *   [ ] 识别设计中的所有高速信号线（USB, Ethernet, DDR, RF 等）。
    *   [ ] 在原理图和 PCB 布局中标注需要进行阻抗控制的走线及其目标值。
    *   [ ] 使用 HILPCB 的阻抗计算器初步估算线宽，并在制造文件中明确要求阻抗控制测试。

### 15. 什么是 PCB 走线（Trace）？线宽和线距如何确定？

*   **问题：** 走线应该画多宽？线与线之间应该隔多远？
*   **场景：** 在为微控制器布线时，工程师不确定信号线和电源线的宽度应该如何设置。
*   **指标/示例：**
    *   **最小线宽/线距:** HILPCB 的标准工艺为 4mil/4mil (0.1mm/0.1mm)。
    *   **信号线:** 通常 6-10 mil 即可。
    *   **电源线:** 根据电流大小，可从 15 mil 到数百 mil 不等。
*   **解决：**
    *   **线宽 (Trace Width):** 主要由**载流量**决定。信号线电流极小，满足制造工艺即可；电源线需要根据电流大小和可接受的温升来计算，电流越大，线宽越宽。
    *   **线距 (Trace Spacing):** 主要由**电压**和**串扰**决定。电压越高，安全间距要求越大。对于高速或敏感信号，增加线距可以减少线间串扰。
*   **预防 (Checklist)：**
    *   [ ] 设置 EDA 软件中的设计规则 (DRC)，定义不同网络的默认线宽和最小间距。
    *   [ ] 对大电流路径使用在线计算器确定线宽。
    *   [ ] 将模拟信号和数字信号、高速信号和低速信号分开布线，并保持足够间距。

## 制造文件与交付 (FAQ #16-22)

### 16. 什么是 Gerber 文件？它包含了哪些信息？

*   **问题：** 我完成了设计，应该给制造商什么文件？
*   **场景：** 第一次准备生产文件，不确定如何从 Altium Designer 或 KiCad 中导出正确的文件。
*   **指标/示例：** 格式：**RS-274X** (扩展 Gerber)。一个典型的文件包会包含：
    *   GTL (顶层铜), GBL (底层铜), G1, G2... (内层铜)
    *   GTS (顶层阻焊), GBS (底层阻焊)
    *   GTO (顶层丝印), GBO (底层丝印)
    *   GKO (板框/外形层)
    *   DRL/TXT (钻孔文件)
*   **解决：** Gerber 文件是 PCB 制造业的 2D 矢量图像标准格式。它逐层描述了 PCB 的所有图形信息，如铜线路、焊盘、阻焊开窗、丝印等。制造商使用这些文件来制作光绘菲林，并最终生产出物理电路板。您需要为设计的每一层都导出一个对应的 Gerber 文件，并连同钻孔文件一起打包。
*   **预防 (Checklist)：**
    *   [ ] 确保使用 RS-274X 格式，因为它包含了坐标和图形定义，不易出错。
    *   [ ] 导出的所有文件使用清晰、标准的文件扩展名。
    *   [ ] 在提交前，使用 HILPCB 的免费 Gerber 查看器检查所有层是否正确、对齐。

### 17. 什么是物料清单 (BOM)？它与 PCB 制造有什么关系？

*   **问题：** 除了 Gerber，制造商还向我要 BOM，这是什么？
*   **场景：** 客户希望 HILPCB 提供包含元器件采购和焊接的一站式组装服务。
*   **指标/示例：** BOM (Bill of Materials) 是一个 Excel 或 CSV 格式的清单，至少包含以下列：
    *   **Item #:** 序号
    *   **Reference Designator:** 位号 (e.g., R1, C5, U2)
    *   **Manufacturer Part Number (MPN):** 制造商型号 (关键信息！)
    *   **Description:** 元件描述 (e.g., Resistor 10k 1% 0603)
    *   **Quantity:** 数量
*   **解决：** BOM 详细列出了组装一块 PCB 所需的所有元器件。对于 PCB 制造（裸板），BOM 不是必需的。但对于 PCBA（组装），BOM 是采购元器件和指导贴片机操作的核心文件。一个准确、无歧义的 BOM 是确保组装顺利进行的关键。
*   **预防 (Checklist)：**
    *   [ ] 确保每个元件都有唯一的、可采购的 MPN。
    *   [ ] 检查 BOM 中的位号是否与 PCB 丝印和 Pick and Place 文件完全一致。
    *   [ ] 标注任何“Do Not Install (DNI)”或需要特殊处理的元件。

### 18. 什么是可制造性设计 (DFM) 检查？为什么它必不可少？

*   **问题：** HILPCB 提供了免费 DFM 检查，这会检查什么？
*   **场景：** 设计完成后，工程师想在花钱投产前，确保设计没有明显的制造缺陷。
*   **指标/示例：** DFM 检查会审核：最小线宽/线距、最小环形环、钻孔尺寸、阻焊桥宽度、丝印清晰度、板边间距等数十项参数。
*   **解决：** DFM (Design for Manufacturability) 是一个在正式生产前，分析您的设计文件是否符合制造工艺能力的自动化或人工过程。它的目标是提前发现并修复那些可能导致低良率、高成本或功能故障的设计问题。例如，如果您的线距小于 HILPCB 的最小能力，DFM 报告会标记出来，让您在造成损失前进行修改。
*   **预防 (Checklist)：**
    *   [ ] 在设计阶段，从 HILPCB 网站获取其工艺能力参数，并设置到您的 EDA 工具中。
    *   [ ] 始终在下单前利用制造商提供的免费 DFM 服务。
    *   [ ] 认真阅读 DFM 报告，并对其中提出的问题进行修改或确认。

### 19. FR-4 材料的介电常数 (Dk) 和损耗因子 (Df) 是什么意思？

*   **问题：** 在进行信号完整性仿真时，我需要输入 FR-4 的 Dk 和 Df，它们是什么？
*   **场景：** 设计一个 2.4GHz 的蓝牙天线匹配电路，需要精确的材料参数来保证性能。
*   **指标/示例：** 对于标准 FR-4 @ 1GHz:
    *   **Dk (Dielectric Constant / εr):** 约 4.2 - 4.7
    *   **Df (Dissipation Factor / tanδ):** 约 0.015 - 0.025
*   **解决：**
    *   **Dk (介电常数):** 影响电信号在介质中传播的速度和特性阻抗。Dk 值越高，信号传播越慢。对于阻抗控制，Dk 是一个关键的计算参数。
    *   **Df (损耗因子):** 衡量介质材料吸收电磁能量并将其转化为热量的程度。Df 越高，信号在传输过程中的能量损失越大，这在高频应用中尤为重要。
*   **预防 (Checklist)：**
    *   [ ] 注意 Dk 和 Df 都会随频率变化。标准 FR-4 的数据表通常只提供 1MHz 或 1GHz 的值。
    *   [ ] 对于频率超过 5GHz 的高速 PCB，FR-4 的高损耗和不稳定的 Dk 可能不再适用，应考虑 Rogers 或其他高频材料。
    *   [ ] 如果需要精确的 Dk/Df 值，请向 HILPCB 索取您所选板材的详细数据手册。

### 20. FR-4 PCB 的主要局限性是什么？什么时候应该考虑其他材料？

*   **问题：** FR-4 这么好，有没有它不适用的情况？
*   **场景：** 一个毫米波雷达项目，工程师发现使用 FR-4 仿真出的信号衰减非常严重。
*   **指标/示例：** 考虑替代材料的临界点：
    *   **频率:** > 5-10 GHz
    *   **工作温度:** > 150°C
    *   **散热需求:** > 10 W/mK
*   **解决：** 尽管 FR-4 用途广泛，但它在以下三个方面存在局限：
    1.  **高频性能:** FR-4 的 Df 值较高，在高频下信号损耗巨大。此时应选择 Rogers 或 Teflon 等低损耗材料。
    2.  **热性能:** FR-4 的导热系数很低 (约 0.25 W/mK)。对于大功率器件，热量难以散发。此时应选择金属芯 PCB (MCPCB) 或陶瓷 PCB。
    3.  **机械柔性:** FR-4 是刚性材料。对于需要弯曲或折叠的应用，应使用柔性 PCB (FPC)。
*   **预防 (Checklist)：**
    *   [ ] 在项目初期就评估频率、功率和机械要求。
    *   [ ] 如果不确定，请将您的设计需求告知 HILPCB，我们的工程师会为您推荐最合适的材料。

### 21. 如何在设计文件中指定 FR-4 材料的具体型号？

*   **问题：** 我想使用特定品牌的 FR-4，比如 Isola 370HR，该如何告诉制造商？
*   **场景：** 一个高可靠性产品，为了保证性能一致性，要求所有批次的 PCB 都使用同一型号的板材。
*   **指标/示例：** 在制造说明 (Fabrication Notes) 中添加一行文字：“Board material must be Isola 370HR or equivalent. Any substitution requires customer approval.”
*   **解决：** 默认情况下，制造商会使用库存中符合所选 Tg 等级的标准 FR-4 材料（如南亚、生益等）。如果您有特定要求，必须在文件中明确指出。最佳实践是在 PCB 设计图纸的注释区或单独的制造说明文档中，清晰地写明要求的材料品牌和型号。如果可以接受性能相当的替代品，也应注明“or equivalent”。
*   **预防 (Checklist)：**
    *   [ ] 在下单前，与 HILPCB 确认您指定的材料是否有库存，或是否需要特殊采购（可能影响交期和成本）。
    *   [ ] 将材料要求放在 Gerber 文件包中的一个醒目位置，如 README.txt 或板框层。

### 22. FR-4 基板的吸水率对 PCB 性能有何影响？

*   **问题：** 数据手册里有一个“Water Absorption”参数，它重要吗？
*   **场景：** 设计一个需要在户外高湿环境中长期工作的传感器设备。
*   **指标/示例：** 标准 FR-4 的吸水率通常在 **0.10% 到 0.20%** 之间。
*   **解决：** 吸水率衡量材料在潮湿环境中吸收水分的倾向。水分会改变材料的电气性能，特别是介电常数 (Dk) 和损耗因子 (Df)，并可能降低其绝缘电阻。在高频或高阻抗电路中，吸水导致的参数漂移会影响电路性能。虽然 FR-4 的吸水率已经很低，但在极其苛刻或高精度的应用中，选择吸水率更低的材料（如一些高频材料）会更加稳定。
*   **预防 (Checklist)：**
    *   [ ] 评估产品的最终使用环境，特别是湿度条件。
    *   [ ] 对于大多数室内应用，标准 FR-4 的吸水率性能足够。
    *   [ ] 如果设备用于热带、海洋或需要频繁清洗的环境，请考虑进行敷形涂覆 (Conformal Coating) 作为额外的防潮保护。
