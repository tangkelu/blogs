## 1. 摘要：场景、挑战与收益

**场景：** 在现代电子产品设计中，从高速数据中心到新能源汽车的电源管理单元，PCB 不再仅仅是元器件的载体，更是系统性能与安全可靠性的基石。硬件与系统工程师在项目初期就必须面对一个复杂的多变量问题：如何在满足信号完整性、热管理和机械强度的同时，确保产品在全生命周期内的电气安全？

**挑战：** 这一挑战的核心在于材料与叠层（Stackup）的策略选择。错误的材料可能导致信号衰减、阻抗失配，甚至在恶劣环境下（如高压、高湿）引发灾难性的电气击穿。其中，**CTI (Comparative Tracking Index，相对漏电起痕指数)** 作为一个关键的电气安全指标，其重要性日益凸显，但往往在设计早期被忽视。一个全面的 `cti requirement explanation` 不仅关乎合规，更直接影响产品的长期可靠性。

**收益：** 本白皮书旨在提供一个系统化的解决方案。作为 HILPCB 材料实验室的官方指引，我们将交付一套从材料选型到制造验证的完整策略。通过标准化的决策树、叠层模板库、建模方法和 DFM/DFR 清单，本白皮书将帮助您的团队：
- **降低设计风险：** 基于明确的 CTI 要求和性能指标，做出数据驱动的材料决策。
- **加速开发周期：** 利用预验证的叠层模板，快速启动新项目。
- **提升产品可靠性：** 通过科学的建模与验证流程，确保 `thermal reliability stackup` 和电气安全。
- **优化制造成本：** 平衡性能与成本，避免过度设计或因材料不当导致的后期返工。

## 2. 材料决策树：从指标到选型的系统化路径

选择正确的 PCB 材料是叠层设计的第一步，也是最关键的一步。CTI 是安全考量的起点，但必须与其他性能指标（如 Dk、Tg、CTE）协同决策。下表提供了一个简化的决策框架。

<div class="div-type-1">

### CTI Requirement Explanation：核心安全指标

CTI 是衡量绝缘材料在电场和电解液污染下，表面抵抗漏电痕迹形成能力的指标。其等级由 PLC (Performance Level Category) 定义，数值越高，抗电痕能力越强。
- **PLC 0:** CTI ≥ 600V (最高等级，适用于工业、电力、汽车高压系统)
- **PLC 1:** 400V ≤ CTI < 600V
- **PLC 2:** 250V ≤ CTI < 400V
- **PLC 3:** 175V ≤ CTI < 250V (标准 FR-4 材料通常处于此范围)

在设计高压或安规要求严格的产品时，必须明确所需的 CTI 等级，并选择相应材料。

</div>

### 材料选型决策树

| 关键指标 (Key Metric) | 推荐材料/等级 | 典型应用场景 | 设计限制/考量 |
| :--- | :--- | :--- | :--- |
| **CTI ≥ 600V (PLC 0)** | 高 CTI FR-4 (如 S1170G), 酚醛树脂, 部分聚酰亚胺(PI) | 工业电源、逆变器、电动汽车 BMS/OBC、医疗设备 | 成本相对较高；材料选项较少，需提前与 HILPCB 确认库存。 |
| **低介电常数/损耗 (Low Dk/Df)** | Rogers (RO4350B, Dk≈3.48), Megtron 6 (Dk≈3.6), Isola FR408HR | 5G/6G 射频、高速服务器 (≥25Gbps)、`hdmi pcb stackup guide` | 加工工艺要求高（如等离子去钻污）；成本显著高于标准 FR-4。 |
| **高玻璃转化温度 (High Tg > 170°C)** | S1000-2M, IT-180A, TU-768 | 汽车电子、高功率密度模块、多层厚铜板 | 材料更脆，钻孔参数需优化；压合温度（约 190-210°C）更高。 |
| **低 Z 轴热膨胀系数 (Low Z-CTE)** | 低 CTE FR-4, 聚酰亚胺 (Polyimide) | HDI 板、BGA/LGA 密集封装、厚板 (>3.0mm) | 显著提升电镀通孔 (PTH) 可靠性，防止热循环下的 via 裂纹。 |
| **高导热率 (High Thermal Conductivity)** | 铝基板 (MCPCB, 1-3 W/m·K), 陶瓷基板 (AlN, >150 W/m·K) | LED 照明、功率 MOSFET 模块、IGBT 驱动 | 通常为单层或双层结构，布线受限；需要专门的 `thermal reliability stackup` 设计。 |
| **柔性/弯折需求 (Flexibility)** | 聚酰亚胺 (Polyimide, PI) | 可穿戴设备、柔刚结合板、动态弯折应用 | 阻抗控制更复杂，需考虑补强片 (Stiffener) 设计。 |

## 3. 叠层模板库：标准化设计的起点

HILPCB 实验室根据数千个量产项目的经验，总结了以下标准叠层模板。这些模板已通过仿真与实测验证，可作为新设计的可靠起点。

### 标准多层板叠层模板

| 层数 | 结构 (示例) | 描述 | 关键应用 |
| :--- | :--- | :--- | :--- |
| **4 层** | SIG/GND/PWR/SIG | `Core(0.76mm) + PP + Core(0.76mm)` | 消费电子、物联网模块。成本效益高，但 EMI 控制需谨慎。 |
| **6 层** | SIG/GND/SIG/SIG/PWR/SIG | `Core + PP + Core + PP + Core` | 性能与成本的平衡点。提供良好的信号/电源隔离。 |
| **8 层** | SIG/GND/SIG/PWR/GND/SIG/PWR/SIG | `Core + PP + PP + Core + PP + PP + Core` | 高速设计（如 PCIe, USB 3.0）。提供多个参考平面，易于实现 50Ω/100Ω 阻抗控制。 |
| **10 层** | SIG/GND/SIG/GND/PWR/PWR/GND/SIG/GND/SIG | `Core + PP...` | 复杂系统、服务器主板。提供极致的 EMI 屏蔽和电源完整性。 |

### 特殊结构叠层模板

<div class="div-type-3">

#### HDI (High-Density Interconnect) 叠层教程

以一个 **1+N+1 (N=6) HDI** 叠层为例，这是一个典型的 `hdi stackup tutorial` 结构：
- **L1 (SIG):** Microvia (L1-L2)
- **L2 (GND):**
- **L3 (SIG):** Buried Via (L3-L6)
- **L4 (PWR):**
- **L5 (GND):**
- **L6 (SIG):**
- **L7 (GND):**
- **L8 (SIG):** Microvia (L8-L7)

**核心工艺：** 采用“序次压合法”，即先制作 L2-L7 的内层芯板，完成埋孔钻孔与电镀，再在两侧分别压合 L1 和 L8 的介质与铜箔，最后进行激光钻微孔。这种结构极大提升了布线密度。

</div>

#### 柔刚结合 (Rigid-Flex) 与 MCPCB

- **柔刚结合板：** 结构为 `刚性区 (FR-4) - 柔性区 (PI) - 刚性区 (FR-4)`。关键在于过渡区的设计，需确保应力平滑过渡，避免弯折时铜箔断裂。
- **MCPCB (铝基板):** 结构为 `电路层 (铜箔) - 绝缘层 (高导热介质) - 金属基层 (铝)`。核心是绝缘层的导热率和耐压值，是 `thermal reliability stackup` 的关键。

## 4. 阻抗/热/机械指标建模方法

精确的建模是实现设计意图的保障。HILPCB 仿真团队使用以下第一性原理模型进行预分析。

### 阻抗建模 (Impedance Modeling)

对于高速信号，如 HDMI 或 DDR，阻抗控制至关重要，容差通常要求在 **±10%** 以内，甚至 **±5%**。
- **微带线 (Microstrip) - 表层走线:**
  $$ Z_0 \approx \frac{87}{\sqrt{\epsilon_r + 1.41}} \ln\left(\frac{5.98H}{0.8W + T}\right) $$
- **带状线 (Stripline) - 内层走线:**
  $$ Z_0 \approx \frac{60}{\sqrt{\epsilon_r}} \ln\left(\frac{4B}{0.67\pi(0.8W + T)}\right) $$
  *其中：*
  - $Z_0$: 特性阻抗 (Ω)
  - $\epsilon_r$: 介电常数 (Dk)
  - $H$: 介质层厚度
  - $W$: 走线宽度
  - $T$: 铜厚
  - $B$: 两个参考平面间的距离

**示例：** 在一个 `hdmi pcb stackup guide` 中，为实现 100Ω 差分阻抗，使用 Dk 为 4.2 的 FR-4 材料，介质厚度 0.1mm，铜厚 1oz (0.035mm)，通过仿真计算得出所需线宽约为 0.12mm。

### 热建模 (Thermal Modeling)

热可靠性分析的核心是计算热阻路径。
- **一维稳态热阻 (Rth):**
  $$ R_{th} = \frac{L}{k \cdot A} $$
  *其中：*
  - $R_{th}$: 热阻 (°C/W)
  - $L$: 热传导路径长度 (m)
  - $k$: 材料导热系数 (W/m·K)
  - $A$: 热传导横截面积 (m²)

通过叠加计算从芯片结 (Junction) 到 PCB 表面，再到散热器的各层热阻，可以预测芯片工作温度，是 `thermal reliability stackup` 设计的基础。

### 机械建模

主要关注 Z 轴 CTE 错配导致的应力。在 -40°C 到 125°C 的温度循环中，如果材料的 Z-CTE 过高（> 50 ppm/°C），PTH 孔铜会承受巨大张应力，可能导致微裂纹或断裂。选择 Z-CTE < 50 ppm/°C 的材料是高可靠性设计的关键。

## 5. 混压/背钻/特殊结构

为了在成本和性能之间取得最佳平衡，HILPCB 支持多种高级制造工艺。

<div class="div-type-6">

### 混合叠层 (Hybrid Stack)

**Rogers + FR-4 混压：** 这是最常见的 `hybrid stack` 方案。
- **结构：** 将昂贵的 Rogers 材料（如 RO4350B）仅用于承载 RF 信号的表层，而内部的数字和电源层则使用成本较低的高 Tg FR-4。
- **挑战：** 两种材料的 CTE 和压合参数（温度、压力）不同，需要专门的压合程序。同时，FR-4 与 Rogers 材料的结合力较弱，钻孔后需进行等离子处理（Plasma Treatment）以增强孔壁结合力。
- **收益：** 在保证射频性能的同时，可节省 30-50% 的材料成本。

</div>

### 背钻 (Back-Drilling)

- **目的：** 移除高速信号通路中多余的过孔桩 (stub)，减少信号反射和衰减，尤其在速率超过 10 Gbps 时效果显著。
- **流程：** 在 PCB 压合和通孔电镀完成后，从另一侧用一个更大直径的钻头，将指定过孔的多余部分钻掉，深度控制精度要求极高。
- **应用：** 服务器背板、高速交换机、光模块。

### 表面处理工艺对比 (Surface Finish Comparison)

表面处理直接影响焊接可靠性和信号完整性。

| 表面处理 | 优点 | 缺点 | 推荐应用 |
| :--- | :--- | :--- | :--- |
| **HASL (热风焊料整平)** | 成本低，可焊性好 | 表面平整度差，不适用于细间距元件 | 通用消费类产品，对平整度要求不高 |
| **ENIG (化学镍金)** | 表面平整，耐腐蚀，适合 BGA | 成本较高，可能存在“黑盘”风险 | 高速板、BGA/CSP 封装、按键区 |
| **OSP (有机可焊性保护剂)** | 成本低，表面极平整，环保 | 不耐存储，多次回流焊后性能下降 | 消费电子，要求快速周转和低成本 |
| **沉银 (Immersion Silver)** | 平整度好，电性能优异 | 易氧化，对存储和操作环境要求高 | 高频信号、通信设备 |

## 6. 验证流程：从材料来料到最终可靠性

一个稳健的 `stackup strategy` 必须有一个闭环的验证流程来保障。

1.  **材料来料检验 (IQC):** 核对供应商提供的 CTI、Dk、Tg 等参数的规格书与测试报告。关键材料进行抽样测试。
2.  **压合过程监控:** 实时监控压合机的温度、压力曲线，确保与材料规格书要求一致。标准 FR-4 压合温度约 **180°C**。
3.  **特性阻抗 Coupon 测试:** 每批次生产的 PCB 都会附带专门设计的 Coupon 条。使用 TDR (时域反射仪) 精确测量 50Ω/100Ω 等特征阻抗，确保其波动在 **±10%** 的设计容差内。这是 `coupon test` 的核心环节。
4.  **翘曲度 (Warpage) 分析:** 使用光学方法（如 Shadow Moiré）测量 PCB 在回流焊前后的翘曲度，确保其小于 0.75%。
5.  **可靠性测试:**
    - **热冲击测试 (Thermal Shock):** 例如，在 -40°C 和 125°C 之间进行 1000 次循环，然后切片检查 PTH 和微孔的完整性。
    - **互连应力测试 (IST):** 通过对 Coupon 进行快速加热和电流加载，加速模拟产品寿命周期内的热机械应力，评估 via 的可靠性。

## 7. DFM/DFR/DFT 清单

| | 最小激光钻孔尺寸 (HDI) | ≥ 0.10mm | DFM 检查 |
| | 钻孔长径比 (Aspect Ratio) | < 10:1 (推荐), < 12:1 (极限) | DFM 检查 |
| | 孔到铜(线/面)距离 | ≥ 0.20mm | DRC 检查 |

| | 大铜面与焊盘距离 | ≥ 0.25mm | DRC 检查 |
| | 孤岛铜 (Copper Island) 是否移除？ | 移除无电气连接的小铜块 | CAM 软件分析 |
| | 接地过孔 (Stitching Vias) 密度 | 沿板边和高速线附近，间距 < λ/20 | Layout 评审 |
| **阻焊 (Solder Mask)** | 阻焊桥 (Solder Mask Dam) 宽度 | ≥ 0.10mm (4mil) | DFM 检查 |
| | 阻焊开窗尺寸 | 单边比焊盘大 0.05mm (2mil) | DFM 检查 |
| | 过孔是否开窗 (Tenting/Plugging) | 根据测试需求决定，默认盖油 | 设计规范 |
| **丝印 (Silkscreen)** | 丝印线宽 | ≥ 0.15mm (6mil) | DFM 检查 |
| | 丝印与焊盘间距 | ≥ 0.15mm (6mil) | DFM 检查 |
| | 元器件位号和极性是否清晰？ | 确保可读性 | Layout 评审 |
| **可靠性 (DFR)** | 高压区域爬电距离/电气间隙 | 遵循 IEC-62368-1 等标准 | 设计评审 |
| | 板边金属裸露 | 距离板边 ≥ 0.4mm | DFM 检查 |
| | 金手指 (Gold Finger) 倒角 | 30° 或 45° 倒角 | 设计规范 |

| | 基准点 (Fiducial Mark) | 每板至少 3 个，对角线分布 | DFM 检查 |
| | ICT 测试点是否在元器件下？ | 避免 | Layout 评审 |
| | JTAG/SWD 接口是否引出？ | 方便调试 | 设计评审 |
| **其他** | 板厚公差 | ±10% | 设计规范 |
| | 阻抗公差 | ±10% (标准), ±5% (严格) | 设计规范 |
