## 1. 执行摘要：从混乱到可控，构建初创团队的PCB设计体系

对于初创研发团队而言，硬件开发的核心挑战在于如何将创新的电路设计思想，高效、可靠且经济地转化为物理实体。然而，现实往往是残酷的：设计文件与工厂要求不匹配、关键术语理解偏差、制造文档（如 **drill file**）的疏漏，导致了反复的工程问询（EQ）、生产延误、成本超支，甚至产品上市失败。这些痛点源于一个共同的根源：缺乏一个标准化的设计到制造（Design-to-Manufacturing）体系。

**我们的目标是**：
*   **统一语言**：建立清晰的术语图谱，消除团队与供应商之间的沟通壁垒。
*   **固化流程**：提供从需求分析到最终文件交付的六步框架，让每个环节都有章可循。
*   **设定标准**：提供基础材料库、模块化布线规则和详尽的文档清单，确保交付质量。
*   **量化结果**：引入KPI仪表盘，将设计质量从“感觉”变为可衡量的数据。
*   **规避风险**：通过超过35项的DFM/DFT清单，将制造风险前置，在设计阶段就予以消除。

## 2. 术语与结构图谱：PCB制造的通用语言

在深入流程之前，我们必须统一语言。错误的术语理解是导致沟通失误和生产事故的主要原因。下表梳理了从设计到制造最核心的几个概念。

| 名词 (Term) | 定义 (Definition) | 对制造的影响 (Impact) | 参考阶段/工具 |
| :--- | :--- | :--- | :--- |
| **Drill File (钻孔文件)** | 也称为NC Drill File，使用Excellon格式，精确定义了PCB上所有孔（通孔、盲埋孔、安装孔）的坐标、尺寸和类型（PTH/NPTH）。 | **关键**：文件错误或缺失将导致元器件无法插装、过孔连接失效或机械结构无法固定。它是物理连接和结构装配的基础。 | CAM输出 |
| **Gerber File (格柏文件)** | 事实上的行业标准格式（RS-274X），用于描述PCB的每一层图像，如铜层、阻焊层、丝印层、锡膏层等。 | **核心**：Gerber文件是PCB“蓝图”，任何一层信息的错误都会直接导致电路功能的失败，如短路、开路。 | CAM输出 |
| **BOM (物料清单)** | Bill of Materials。详细列出项目所需的所有元器件，包括型号、制造商、封装、位号和数量。 | **直接影响采购与装配**：BOM不准确会导致采购错误、元器件与焊盘不匹配、生产停线。 | 原理图/ERP |
| **Assembly Drawing (装配图)** | 指导PCBA（PCB Assembly）生产的图纸，清晰标示了每个元器件的位号、方向（如极性）和位置。 | **保证装配正确性**：对于有方向性的元器件（二极管、电容、IC），装配图错误会导致元'器件反向，烧毁电路板。 | PCB布局 |
| **Centroid File (坐标文件)** | 也称Pick and Place File，提供SMT贴片机所需的每个表面贴装元器件的精确X-Y坐标和旋转角度。 | **决定自动化贴片效率与精度**：坐标原点或旋转角度错误，会导致元器件贴装偏移、空焊或短路。 | PCB布局 |
| **DRC (设计规则检查)** | Design Rule Check。在EDA软件中根据预设规则（如线宽、间距）自动检查布局布线是否符合制造和电气性能要求。 | **质量门槛**：有效的DRC能发现80%以上的设计硬伤，是保证可制造性的第一道防线。 | PCB布局/布线 |
| **DFM (可制造性设计)** | Design for Manufacturability。一套旨在优化设计，使其更易于制造、成本更低、可靠性更高的设计原则。 | **决定成本与良率**：良好的DFM设计能极大提升生产良率，降低制造成本。例如，合理的焊盘设计能减少焊接缺陷。 | 全流程 |

## 3. 流程框架：从需求到交付的六步法

一个成功的PCB项目不是灵感的瞬间迸发，而是一个结构化、系统化的工程过程。我们提炼出以下六个关键步骤，帮助您的团队建立可复制的工作流。

### **第一步：需求定义 (Requirement Definition)**
这是所有工作的起点。此阶段需要明确：
*   **电气性能**：信号最高频率、阻抗控制要求、电源功率、关键信号的噪声容限。
*   **机械结构**：板框尺寸、厚度、禁用区域、安装孔位置、与其他组件的配合关系。
*   **成本与周期**：目标BOM成本、PCB制造成本、项目交付时间。
*   **合规性**：产品需要满足的安规、环保（RoHS）等标准。

### **第二步：栈叠设计 (Stack-up Design)**
PCB的层叠结构是电路板的“骨架”，决定了其电气性能和成本。
*   **层数选择**：根据信号密度、电源完整性（PI）和信号完整性（SI）需求决定。
*   **材料选择**：FR-4是通用选择，而高速或高频设计可能需要Mid-Loss或Low-Loss材料。HILPCB提供从标准[FR-4 PCB](https://hilpcb.com/en/products/fr4-pcb)到高端Rogers材料的全方位支持。
*   **叠层规划**：合理安排信号层、电源层和地平面。一个经典的四层板叠构是：SIG-GND-PWR-SIG，它能提供良好的屏蔽和阻抗控制。
*   **阻抗计算**：使用**HILPCB的在线阻抗计算器**等工具，根据线宽、介电常数（Er）和介质厚度精确计算目标阻抗。

### **第三步：模块化布局 (Modular Layout)**
布局是“艺术与科学的结合”。遵循模块化原则，将功能相近的电路放在一起。
*   **功能分区**：将模拟电路、数字电路、电源电路、射频电路等物理隔离，减少相互干扰。
*   **关键元件优先**：首先放置连接器、主芯片、时钟源等核心元件。
*   **热管理**：将发热量大的器件放置在利于散热的位置，或为其规划散热铜皮/过孔。
*   **电源路径**：确保从电源输入到各负载的路径短而粗。

### **第四步：关键网络布线 (Critical Routing)**
布线是将逻辑连接变为物理现实的过程。
*   **电源网络 (PDN)**：使用平面或宽走线，确保低阻抗。在IC电源引脚附近放置足够数量和容值的去耦电容。
*   **时钟/高速信号**：走线短、直，远离干扰源，进行严格的阻抗控制和长度匹配（如DDR、USB）。
*   **模拟信号**：使用差分对或保护地（Guard Ring）进行保护，避免跨越数字地分割。
*   **功率信号**：对于大电流路径，考虑使用[重铜PCB](https://hilpcb.com/en/products/heavy-copper-pcb)工艺，并确保足够的线宽以承载电流。

### **第五步：设计评审 (Design Review)**
在交付制造前，内部评审是必不可少的质量关口。
*   **自检**：设计工程师对照DRC报告和DFM清单进行检查。
*   **同行评审**：邀请另一位工程师进行交叉检查，提供“第二双眼睛”。
*   **Checklist驱动**：使用本白皮书提供的DFM/DFT清单，逐项确认。

### **第六步：交付包生成 (Deliverable Generation)**
这是“**drill file tutorial**”的核心实践环节。一个完整的交付包应包含所有制造所需文件，并打包成一个ZIP文件。
*   **Gerber文件**：确保所有层都已导出，并采用RS-274X格式。
*   **Drill文件**：区分PTH（镀通孔）和NPTH（非镀通孔），并提供钻孔图（Drill Map）。
*   **文档**：BOM、装配图、坐标文件、Fab Drawing（制造说明图）。
*   **HILPCB资料检查**：在下单前，您可以使用**HILPCB**的免费Gerber Viewer工具进行初步预览，我们的工程师也会在收到订单后进行专业的人工审核，确保万无一失。

## 4. 材料/层叠标准库：从可靠的基线开始

为避免每次都从零开始设计叠层，建议初创团队建立自己的标准库。以下是三种最常用配置的基线，可作为起点。

*   **单/双层板 (Single/Double Layer)**
    *   **应用**：成本敏感的简单电子产品、电源模块、LED照明。
    *   **典型结构 (双层)**：Top (Signal) - Core (FR-4, 1.6mm) - Bottom (Signal/GND Plane)。
    *   **优势**：成本极低，工艺成熟。
    *   **HILPCB产品**：[单双面板](https://hilpcb.com/en/products/single-double-layer-pcb)服务提供快速、经济的打样和生产。

*   **四层板 (Four Layer)**
    *   **应用**：大多数MCU控制板、物联网设备、消费电子。
    *   **典型结构**：Top (Signal) - Prepreg - L2 (GND Plane) - Core - L3 (Power Plane) - Prepreg - Bottom (Signal)。
    *   **优势**：提供了完整的GND和PWR平面，极大地改善了信号完整性和电源完整性，是性价比最高的[多层板](https://hilpcb.com/en/products/multilayer-pcb)方案。

*   **六层及以上 (Six+ Layers)**
    *   **应用**：高速数字系统（如FPGA、DDR内存）、紧凑型产品。
    *   **典型结构 (六层)**：SIG - GND - SIG - SIG - PWR - SIG。提供了更多布线层，并能通过邻近的GND/PWR层实现更好的阻抗控制。
    *   **HILPCB Stackup服务**：对于复杂的多层板设计，**HILPCB**的工程团队可以提供专业的叠层设计与仿真服务，确保您的阻抗和信号完整性要求得到满足。

## 5. 模块化布局/布线规则

不同类型的电路有不同的“脾气”。以下是针对四类常见电路的布线黄金法则。

*   **时钟网络**：
    1.  **最短路径**：时钟源到负载的走线尽可能短。
    2.  **星型拓扑**：从时钟驱动器到各个接收器采用星型走线，避免菊花链引入的反射。
    3.  **隔离**：与其他信号线保持至少3W（W为线宽）的间距。
    4.  **参考平面**：确保时钟线下方有连续的参考地平面，不要跨分割。

*   **电源网络 (PDN)**：
    1.  **低阻抗路径**：使用电源平面或尽可能宽的走线。
    2.  **去耦电容**：在每个IC的电源引脚旁，就近放置一个0.1uF的陶瓷电容。对于整个电路板，放置一些大容量（10uF-100uF）的钽电容或电解电容作为“水库”。
    3.  **过孔**：在连接电源平面时，使用多个过孔以降低电感。

*   **模拟网络**：
    1.  **物理隔离**：将模拟区域与数字区域分开，避免数字噪声耦合。
    2.  **独立地**：在可能的情况下，为模拟电路提供独立的模拟地，并最终通过磁珠或0欧电阻单点接地。
    3.  **差分对**：对微弱的模拟信号使用差分走线，以抵抗共模噪声。

*   **功率网络**：
    1.  **载流量**：根据IPC-2152标准计算所需线宽，确保能安全承载工作电流。
    2.  **散热**：在走线上开窗（去掉阻焊层）以增加散热，或增加散热铜皮。
    3.  **过孔**：大电流路径上的过孔需要“打满”，即使用多个过孔并联。

## 6. 文档清单：确保制造信息无损传递

一个完整、清晰的制造文档包是实现“一次成功”的最后保障。请确保您的交付包（通常是一个.zip文件）包含以下所有内容。

1.  **Gerber Files (RS-274X)**
    *   `GTL`: Top Copper Layer (顶层铜)
    *   `GBL`: Bottom Copper Layer (底层铜)
    *   `GTS`: Top Solder Mask (顶层阻焊)
    *   `GBS`: Bottom Solder Mask (底层阻焊)
    *   `GTO`: Top Silkscreen (顶层丝印)
    *   `GBO`: Bottom Silkscreen (底层丝印)
    *   `GKO`: Board Outline (板框层)
    *   `GTP`: Top Paste Mask (顶层锡膏)
    *   `GBP`: Bottom Paste Mask (底层锡膏)
    *   *（对于多层板，还需包含所有内层 G1, G2...）*

2.  **Drill Files (Excellon)**
    *   `*.drl` 或 `*.txt`: 包含所有钻孔信息。
    *   **关键**：务必在文件中或附带的ReadMe中明确区分**PTH (Plated Through Hole)** 和 **NPTH (Non-Plated Through Hole)**。通常会生成两个独立的钻孔文件。这是**drill file tutorial**中最容易出错的地方。

3.  **Bill of Materials (BOM)**
    *   格式：Excel (.xls, .xlsx) 或 CSV。
    *   必需列：`Item`, `Quantity`, `Reference Designator`, `Part Number`, `Manufacturer`, `Description`, `Package/Footprint`。
    *   使用**HILPCB的BOM Viewer**可以快速检查您的BOM格式和内容。

4.  **Assembly Drawing**
    *   格式：PDF 或 Gerber。
    *   内容：清晰显示每个元件的位号、轮廓和极性标记（如二极管的阴极、电容的+极、IC的1脚）。

5.  **Centroid / Pick and Place File**
    *   格式：.txt 或 .csv。
    *   必需列：`Reference Designator`, `X-Location`, `Y-Location`, `Rotation`, `Layer (Top/Bottom)`。

## 7. KPI 仪表：量化你的设计质量

无法衡量，就无法改进。为您的PCB设计流程建立以下KPI仪表盘，持续追踪和优化。

| KPI 指标 | 定义 | 目标 (初创团队) | 改进策略 |
| :--- | :--- | :--- | :--- |
| **一次交付成功率 (FPY)** | 提交给制造商的文件无需任何修改即被接受的比例。 | > 85% | 严格执行DFM清单，使用制造商提供的DRC规则文件。 |
| **DRC 错误数量** | 在最终设计中，由EDA软件报告的未解决的DRC错误数量。 | 0 | 在设计过程中定期运行DRC，而不是等到最后。 |
| **阻抗命中率** | 实际生产出的PCB阻抗值与设计目标值的匹配度（通过TDR测试或阻抗Coupon）。 | ±10% 以内 | 使用专业的叠层设计和阻抗计算工具，与HILPCB工程师确认叠层方案。 |
| **设计迭代次数** | 从原型到最终量产版本，硬件设计经历的主要版本修改次数。 | < 3 次 | 加强前期需求评审和仿真分析，减少后期“救火式”修改。 |

## 8. DFM/DFT 清单、HILPCB 协同服务与行动号召

这是您在设计阶段规避制造陷阱的终极武器。在每次提交设计前，请逐项检查。

| 类别 | 检查项 | 规则/指南 | 风险/原因 |
| :--- | :--- | :--- | :--- |
| **钻孔 (Drilling)** | 最小钻孔尺寸 | ≥ 0.2mm | 太小的孔增加断钻风险和成本。 |
| | 最小环宽 (Annular Ring) | ≥ 0.15mm (6mil) | 保证孔壁电镀后与焊盘的可靠连接，防止开路。 |
| | 孔到孔间距 (Hole-to-Hole) | ≥ 0.3mm | 防止钻孔时撕裂基材。 |
| | PTH/NPTH 分离 | 明确分离或在图纸中说明。 | 混淆会导致错误的孔壁处理。 |
| **布线 (Routing)** | 最小线宽/线距 | ≥ 4mil/4mil (0.1mm) | 挑战工厂极限会降低良率，增加成本。 |
| | BGA区域出线 | 避免在焊盘上直接打孔（Via-in-Pad），除非工艺允许。 | 可能导致焊接时焊料流失，造成虚焊。 |
| | 铜皮到板边距离 | ≥ 0.3mm | 防止在V-cut或铣边时铜皮暴露或卷曲。 |
| | 避免锐角/直角走线 | 使用45度角或圆弧走线。 | 锐角在蚀刻时可能造成“酸角”，影响信号完整性。 |
| **焊盘 (Pads)** | 阻焊开窗 (Solder Mask Expansion) | 比焊盘单边大 2-3mil。 | 太小会覆盖焊盘，太大则可能导致相邻焊盘桥连。 |
| | 热风焊盘 (Thermal Relief) | 在连接大铜皮时使用。 | 防止焊接时热量迅速散失，导致冷焊。 |
| | 过孔盖油/塞油 | 根据需求明确指示（Tented/Plugged）。 | 防止焊接时焊料流入过孔。 |
| **丝印 (Silkscreen)** | 最小丝印线宽 | ≥ 6mil | 保证丝印清晰可读。 |
| | 丝印上焊盘 | 绝对禁止。 | 会严重影响元器件的可焊性。 |
| | 丝印与阻焊开窗间距 | ≥ 3mil | 防止丝印印到焊盘上。 |
| **拼板 (Panelization)** | V-Cut 路径 | 必须是直线，贯穿整个拼板。 | V-Cut刀具无法转弯。 |
| | 邮票孔设计 | 孔径0.5-0.8mm，间距1-1.2mm。 | 保证连接强度的同时，易于分板。 |
| | 工艺边 (Tooling Strips) | 宽度 ≥ 5mm，并添加定位孔（Fiducials）。 | SMT设备夹持和光学定位所必需。 |
| **测试 (DFT)** | 测试点 (Test Points) | 关键信号网络应引出测试点。 | 便于生产测试（ICT）和后期调试。 |
| | 测试点尺寸/间距 | 直径 ≥ 0.8mm，间距 ≥ 1.5mm。 | 适应测试探针的要求。 |
