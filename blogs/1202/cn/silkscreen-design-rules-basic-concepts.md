## 1. 执行摘要：从混乱到协同，丝印规则是基石

对于追求快速迭代和产品可靠性的初创研发团队而言，PCB（Printed Circuit Board，印刷电路板）设计流程中的混乱是最大的隐形成本。工程师各自为战，缺乏统一的设计规范，导致原理图与Layout脱节、制造数据频繁被工程提问（EQ）、产线装配效率低下、后期调试与维修困难重重。这些问题的根源，往往被归咎于复杂的布局布线，但一个被严重低估的环节——**丝印（Silkscreen）**，却常常是引发连锁反应的导火索。

**核心痛点：**
*   **沟通模糊：** 丝印信息不清晰、不统一，导致装配、调试、维修人员的误判，浪费大量工时。
*   **制造延迟：** 丝印压焊盘、过孔或超出板框等问题，会触发制造商的DFM（Design for Manufacturability）检查警报，导致生产暂停，需要设计人员反复沟通确认。
*   **质量隐患：** 错误的极性标识、模糊的元器件位号，可能导致元器件反向、错装，造成板卡功能失效甚至烧毁。
*   **知识无法沉淀：** 缺乏标准化的丝印设计规则，团队经验无法固化为可执行的流程，新成员上手慢，重复犯错。

**机遇与目标：**
本白皮书旨在解决上述痛点。通过建立一套围绕“silkscreen design rules”的PCB设计基础体系，我们能将模糊的“经验”转化为清晰、可量化的“标准”。这不仅是关于**what is pcb**设计美学，更是关乎工程效率、产品质量和团队协同的核心实践。

本白皮书的目标是为您的团队交付一个可复制的PCB设计框架，涵盖：
*   **统一术语：** 建立通用语言，消除沟通障碍。
*   **标准化流程：** 固化从需求到交付的最佳实践。
*   **量化质量门槛：** 通过明确的规则和KPI，确保每个设计环节的输出质量。
*   **高效协同：** 赋能团队成员，并与像 **HilPCBPCB Factory (HILPCB)** 这样的专业制造商无缝对接。

最终，您的团队将能够显著提升一次交付成功率，缩短研发周期，并构建起坚实可靠的硬件产品基础。

## 2. 术语与结构图谱：构建通用设计语言

在深入流程之前，我们必须统一对PCB关键元素的认知。下表定义了与丝印设计紧密相关的核心术语，并阐述了其对设计与制造的影响。

| 名词 (Term) | 定义 (Definition) | 对设计/制造的影响 (Impact) | 参考标准/建议 |
| :--- | :--- | :--- | :--- |
| **丝印层 (Silkscreen/Legend)** | PCB表面用于标识元器件位号、极性、外形轮廓、板名版本等信息的油墨层，通常为白色或黑色。 | 直接影响可读性、装配效率和后期调试。丝印错误是常见的DFM问题。 | 字体高度 ≥ 25mil，线条宽度 ≥ 4mil。 |
| **位号 (Reference Designator)** | 分配给每个元器件的唯一标识符，如R1, C1, U1。它连接了原理图、BOM和物理Layout。 | 位号必须清晰、无歧义，且紧邻对应元器件。是装配和维修的“地图”。 | IPC-2612 |
| **极性标记 (Polarity Mark)** | 用于指示有极性元器件（如二极管、电解电容、IC）方向的符号，如“+”、点、斜角或条带。 | 标记错误或缺失会导致元器件反向安装，引发电路故障甚至永久性损坏。 | 遵循元器件数据手册的建议。 |
| **引脚1标识 (Pin 1 Indicator)** | 用于标识集成电路（IC）第一个引脚位置的符号，通常是一个圆点或数字“1”。 | 确保IC安装方向正确，是所有引脚定位的基准。 | 丝印点应放置在IC封装的Pin 1标记附近。 |
| **元器件轮廓 (Component Outline)** | 在丝印层上绘制的元器件物理边界，用于指示其在板上的精确位置和占用空间。 | 帮助手动装配和检查，但对于高密度板，可能会被省略以节省空间。 | 轮廓线应略大于元器件实际尺寸。 |
| **焊环 (Annular Ring)** | 环绕在过孔（Via）或元件孔周围的铜环。**annular ring definition** 指的是钻孔边缘到焊盘边缘的铜宽度。 | 保证钻孔后仍有足够的铜连接，确保电气连接的可靠性。丝印绝不能覆盖焊环区域。 | IPC-6012 |
| **阻焊层 (Solder Mask)** | 覆盖在铜箔上的一层绝缘保护涂层，用于防止焊接时发生桥连，并保护电路免受环境影响。 | 丝印必须印在阻焊层之上。丝印与阻焊开窗（焊盘）之间需要有安全间距。 | 丝印到焊盘的间距 ≥ 3mil。 |
| **基准点 (Fiducial Mark)** | 用于SMT（表面贴装技术）设备光学定位的标记，通常是裸露的铜点，中心有阻焊开窗。 | 确保贴片机能精确识别PCB坐标，实现元器件的精准贴装。丝印不能干扰其光学识别。 | 通常为直径1mm的圆形焊盘，周围有2mm的禁布区。 |

## 3. 流程框架：从需求到交付的丝印规则集成

成功的PCB设计是一个结构化的流程，丝印规则并非孤立存在，而是贯穿于每个环节。建立从 **pcb schematic to layout** 的标准化流程至关重要。

**需求分析 → 原理图设计 → 叠层与阻抗设计 → 布局 → 布线 → 评审 → 生产资料交付**

1.  **需求分析 & 原理图设计 (Input)**
    *   **核心活动：** 定义产品功能、确定关键元器件、绘制原理图。
    *   **丝印关联：**
        *   **位号规范：** 在原理图阶段就建立统一的位号命名规则（如R1xx表示电源部分电阻）。
        *   **元器件信息：** 确保原理图库中的元器件封装带有准确的Pin 1和极性信息，这些信息将自动传递到Layout。

2.  **叠层与阻抗设计 (Stackup)**
    *   **核心活动：** 根据信号速率、电源需求和成本，确定PCB的层数、材料、层压结构和阻抗模型。
    *   **丝印关联：**
        *   **层定义：** 明确顶层丝印（Top Silkscreen/GTO）和底层丝印（Bottom Silkscreen/GBO）的使用。对于只有单面贴片的板，底层丝印可能仅用于标识板名和版本号。
        *   **专业支持：** 此阶段可借助 **HILPCB** 的在线阻抗计算器工具，并咨询其工程师以获得优化的叠层建议，确保生产可行性。

3.  **布局 (Placement)**
    *   **核心活动：** 将元器件从原理图导入PCB，并根据电气性能、热管理和结构要求进行放置。
    *   **丝印关联：**
        *   **位号放置：** 这是丝印设计的关键一步。放置元器件后，立即调整其位号的位置，确保其清晰可读，不会被元器件本体遮挡，且不会与其他位号重叠。
        *   **方向一致性：** 尽量保持位号的读取方向一致（如从左到右，从下到上），避免旋转角度过多，提升可读性。

4.  **布线 (Routing)**
    *   **核心活动：** 连接元器件的引脚，形成完整的电气网络。
    *   **丝印关联：**
        *   **避让规则：** 在布线后，需要再次检查丝印，确保没有任何丝印线条或文字压在裸露的焊盘、测试点或高密度布线区域上。EDA工具通常有自动裁剪（clipping）功能，但仍需人工核查。

5.  **评审 (Review)**
    *   **核心活动：** 设计完成后，进行内部同行评审和DFM/DFT（可制造性/可测试性设计）检查。
    *   **丝印关联：**
        *   **Checklist驱动：** 使用本白皮书第八节的DFM清单，逐项检查丝印的合规性。
        *   **视觉检查：** 打印1:1的PDF或使用Gerber Viewer工具，模拟最终成品，检查文字大小、清晰度和位置是否合理。

6.  **生产资料交付 (Output)**
    *   **核心活动：** 生成Gerber、钻孔文件、BOM、坐标文件等制造所需的全套文档。
    *   **丝印关联：**
        *   **文件完整性：** 确保输出的Gerber文件中包含独立的丝印层文件（如 `.GTO` for Top Overlay, `.GBO` for Bottom Overlay）。
        *   **清晰说明：** 在制造说明文件中，备注丝印颜色（默认为白色）等特殊要求。

## 4. 材料/层叠标准库：构建设计基线

为确保设计的一致性和可制造性，团队应建立一个基础的层叠标准库。这不仅简化了新项目启动过程，也为与制造商沟通提供了清晰的基准。

### 基线一：单/双层板 ([Single/Double Layer PCB](https://hilpcb.com/en/products/single-double-layer-pcb))
这是 **single layer pcb basics** 的核心，适用于成本敏感、功能简单的产品。
*   **材料：** FR-4 (Tg ≈ 130-140°C)
*   **板厚：** 1.6mm (标准) / 1.2mm / 1.0mm
*   **铜厚：** 1oz (35μm)
*   **阻焊颜色：** 绿色 (标准)
*   **丝印颜色：** 白色 (标准)
*   **表面处理：** HASL (有铅/无铅喷锡)
*   **应用场景：** 玩具、简单消费电子、电源适配器。
*   **丝印要点：** 空间充裕，丝印规则可以相对宽松，但仍需保持清晰和规范。可在此类项目中，训练新工程师养成良好的丝印习惯。HILPCB提供高质量的[单双面板](https://hilpcb.com/en/products/single-double-layer-pcb)制造服务，是原型验证的理想选择。

### 基线二：四层板 (Four Layer PCB)
应用最广泛的多层板结构，提供了良好的信号完整性和EMI性能。
*   **材料：** FR-4 (Tg ≈ 150°C)
*   **板厚：** 1.6mm
*   **叠层结构：**
    *   L1: Signal (Top)
    *   L2: GND (Ground Plane)
    *   L3: Power (Power Plane)
    *   L4: Signal (Bottom)
*   **铜厚：** 外层1oz，内层0.5oz
*   **表面处理：** ENIG (沉金)
*   **应用场景：** 物联网设备、微控制器应用、工业控制。
*   **丝印要点：** 元器件密度增加，丝印空间受限。必须严格遵守最小字高、线宽和间距规则。位号可能需要放置在元器件轮廓之外。

### 基线三：更高层数与特殊应用
对于高速、高频或高密度设计，如[HDI PCB](https://hilpcb.com/en/products/hdi-pcb)或[高速PCB](https://hilpcb.com/en/products/high-speed-pcb)，材料和叠层会更加复杂。虽然丝印的基本原则不变，但由于空间极其宝贵，可能会采取以下策略：
*   **简化丝印：** 仅保留关键信息，如Pin 1标识、高压警示和板名。
*   **装配图替代：** 依赖详细的装配图（Assembly Drawing）进行生产和调试，丝印仅作辅助。
*   **选择性丝印：** 只在连接器、测试点等关键区域放置丝印。

与 **HILPCB** 这样的专业制造商合作，可以在项目早期就获得关于材料选择和层叠设计的专业建议，避免后期因可制造性问题导致的设计修改。

## 5. 模块化布局/布线规则中的丝印考量

丝印设计需要根据电路模块的功能特性进行差异化处理。

*   **通用丝印规则 (Global Rules):**
    *   **字体与尺寸：** 推荐使用无衬线矢量字体。最小线宽不低于 **5mil**，最小字高不低于 **30mil**，以保证清晰可辨。
    *   **间距 (Clearance):**
        *   丝印与焊盘间距：≥ 3mil，防止油墨污染焊盘影响焊接。
        *   丝印与板边间距：≥ 10mil，防止被切割掉。
        *   丝印与过孔间距：若过孔开窗，则视为焊盘处理；若过孔盖油，则可忽略。
    *   **位号方向：** 尽量保持0°或90°，避免使用其他角度。整板最多使用两种方向，便于阅读。
    *   **内容规范：**
        *   **必须包含：** 板名、版本号、公司Logo（可选）、UL/RoHS等认证标志。
        *   **严禁出现：** 丝印不能位于元器件下方（除非该元器件是透明的或有足够的高度差）、不能压住任何需要焊接或测试的裸露铜面。

*   **时钟/高速信号模块:**
    *   **考量：** 信号完整性至上。
    *   **丝印策略：** 丝印应远离差分对、时钟线等关键信号线。避免在这些走线附近放置大面积的丝印标识，以减少对阻抗的潜在影响。

*   **电源模块 (Power Supply):**
    *   **考量：** 安全性与易用性。
    *   **丝印策略：**
        *   明确标识输入/输出端口、电压值（如VIN, +5V, GND）。
        *   对于保险丝，明确标识其额定电流（如 F1, 2A/250V）。
        *   在高压区域，使用明确的警示符号（如高压闪电标志）和文字（"DANGER: HIGH VOLTAGE"）。

*   **模拟模块 (Analog Circuit):**
    *   **考量：** 信号敏感，避免干扰。
    *   **丝印策略：** 在敏感的模拟电路区域（如运放输入端），丝印应尽量简化。可以使用丝印框线来标识出“模拟地”和“数字地”的分割区域。

*   **功率模块 (High Power):**
    *   **考量：** 热管理与大电流。
    *   **丝印策略：** 使用丝印标识出大电流路径的方向和区域，提醒注意潜在的高温区域。

## 6. 文档清单：确保与制造商的无缝交接

一份完整、规范的生产资料包是实现高效制造的前提。以下是交付给 **HILPCB** 或任何PCB制造商时必须包含的核心文件清单。

| 文件类型 | 描述 | 关键丝印相关项 |
| :--- | :--- | :--- |
| **Gerber Files (RS-274X)** | 定义了PCB每一层的图形信息，是制造的蓝图。 | 确保包含 `.GTO` (Top Overlay), `.GBO` (Bottom Overlay) 文件。 |
| **Drill File (NC Drill)** | 定义了所有钻孔的位置和尺寸。 | 检查丝印是否与钻孔位置有冲突，特别是过孔。 |
| **BOM (Bill of Materials)** | 列出了所有元器件的型号、封装、数量和位号。 | BOM中的位号必须与丝印层上的位号一一对应，无遗漏、无重复。 |
| **坐标文件 (Pick and Place)** | 提供每个元器件的中心坐标、旋转角度和所在层面，供贴片机使用。 | 坐标文件中的位号必须与BOM和丝印完全一致。 |
| **装配图 (Assembly Drawing)** | PDF格式，直观展示元器件在PCB上的位置、方向和位号。 | 装配图是丝印信息的最终视觉验证，应清晰展示所有丝印内容。 |
| **制造说明 (Fab Notes)** | 文本文档，说明特殊工艺要求。 | 明确指定丝印颜色、板材、厚度、表面处理等要求。 |
| **DFM检查表 (DFM Sheet)** | （可选，但强烈推荐）团队内部的自检清单，确认已符合所有设计规则。 | 包含本白皮书第八节的所有丝印检查项。 |

## 7. KPI 仪表：量化你的设计质量

为了持续改进，必须对设计流程进行量化管理。以下KPI（关键绩效指标）可用于评估和提升团队的PCB设计质量。

| KPI 指标 | 定义 | 目标参考值 | 衡量方法 |
| :--- | :--- | :--- | :--- |
| **一次交付成功率** | 生产资料首次提交给制造商，无需任何修改即可直接投产的比例。 | > 95% | (1 - EQ次数 / 总项目数) * 100% |
| **丝印相关DFM问题数** | 每个项目因丝印问题（压焊盘、字号过小等）被制造商提出的工程问题数量。 | 0 | 统计制造商反馈的EQ记录。 |
| **DRC (设计规则检查) 错误数** | Layout软件中，最终设计文件存在的DRC错误数量。 | 0 | 运行EDA工具的DRC检查报告。 |
| **装配错误率（丝印相关）** | 因丝印标识不清（如极性、位号错误）导致的产线装配错误比率。 | < 0.1% | 统计产线反馈的装配问题报告。 |
| **设计迭代次数** | 从初版Layout到最终交付生产，因内部评审或DFM问题导致的重大修改次数。 | ≤ 2 次 | 项目管理工具中的版本历史记录。 |

## 8. DFM/DFT 清单与 HILPCB 协同服务

这是您的团队在每次设计提交前必须执行的“飞行前检查清单”。它涵盖了从丝印到制造的各个关键节点，是您团队的质量防火墙。这份 **pcb design checklist beginner** 同样适用于经验丰富的工程师，以防疏漏。

### DFM/DFT 自检清单 (≥35项)

| 类别 | 检查项 | 状态 (Y/N) | 备注 |
| :--- | :--- | :--- | :--- |
| **丝印 (Silkscreen)** | 1. 最小线宽是否 ≥ 5mil？ | | |
| | 2. 最小字高是否 ≥ 30mil？ | | |
| | 3. 所有位号是否清晰可辨，未重叠？ | | |
| | 4. 位号是否紧邻对应元器件？ | | |
| | 5. 所有有极性元器件（电容、二极管、LED）是否都有清晰的极性标记？ | | |
| | 6. 所有IC是否都有清晰的Pin 1标记？ | | |
| | 7. 丝印是否已从所有SMD焊盘上裁剪？（检查间距 ≥ 3mil） | | |
| | 8. 丝印是否已从所有THT焊盘和过孔上裁剪？ | | |
| | 9. 丝印是否已从所有测试点和金手指上裁剪？ | | |
