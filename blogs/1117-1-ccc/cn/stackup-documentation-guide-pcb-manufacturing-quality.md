## 制造流程鸟瞰：从 Stackup 文档到物理实体

一份详尽的 `stackup documentation guide` 是制造的起点。它规定了材料、厚度、铜重和公差，而这些参数正是我们设定产线工艺窗口的依据。下表展示了主要制造工序如何与 Stackup 文档中的信息紧密关联。

| 工序 (Process Step) | 核心目标 (Core Objective) | 关键控制参数 (Key Control Parameters) | 关联的 Stackup 信息 |
| :--- | :--- | :--- | :--- |
| **基材准备** | 确保材料符合设计要求 | 材料型号、板厚、铜厚、尺寸精度 | 芯板(Core)与半固化片(PP)型号、Dk/Df值、Tg值、CAF抗性 |
| **内层图形转移** | 精确复制内层线路图形 | 曝光能量、显影时间、对位精度 (±25µm) | 内层铜厚 (e.g., 0.5oz, 1oz)、最小线宽/线距 |
| **层压 (Lamination)** | 将多层结构压合成整体 | 温度、压力、时间曲线、树脂流动控制 | 叠层顺序、PP型号与数量、总厚度公差 (±10%) |
| **钻孔 (Drilling)** | 制造导通孔与安装孔 | 钻嘴转速、进刀速率、孔位精度 (±50µm) | 孔径、孔类型 (PTH/NPTH/盲埋孔)、孔密度、最小环宽 |
| **孔铜/电镀 (Plating)** | 实现层间电气连接 | 整流器电流密度、镀液成分、镀铜厚度 (>20µm) | 孔径比 (Aspect Ratio)、电镀铜厚要求 |
| **外层图形转移** | 精确复制外层线路图形 | 对位精度、蚀刻因子控制 | 外层铜厚、阻抗控制线宽、BGA/QFN焊盘尺寸 |
| **阻焊 (Soldermask)** | 保护线路，定义焊接区域 | 阻焊油墨类型与厚度、曝光精度、硬化条件 | 阻焊颜色、阻焊桥 (Solder Mask Dam) 最小宽度 |
| **表面处理** | 提供可焊性与保护 | 镀层厚度 (e.g., ENIG: Au 0.03-0.08µm)、平整度 | [表面处理类型 (Surface Finish)](/blog/pcb-surface-finish/) (HASL, ENIG, OSP等) |
| **SMT 组装** | 贴装与焊接元器件 | 锡膏量、贴片精度、回流焊温度曲线 | BOM表、元器件封装、焊盘设计 |
| **测试与验证** | 确保电气功能与可靠性 | 测试覆盖率、故障诊断精度 | 测试点布局、网络连接关系 |

## 图形转移、蚀刻与阻焊的精密控制

线路的宽度和间距直接关系到信号完整性和阻抗控制，这些都在 Stackup 文档中被严格定义。制造过程的挑战在于如何将这些数字精确地复刻到物理板上。

### 蚀刻工艺窗口：从数字到现实的转换

蚀刻是“减法”工艺，它移除不需要的铜，留下线路。但蚀刻液不仅向下腐蚀，也会向侧面侵蚀，产生“侧蚀”(Undercut)。为了补偿这种效应，我们会对设计文件进行“蚀刻补偿”，即在生产菲林上预先加宽线路。

<div class="div-style-1">

#### 工艺窗口：线路蚀刻

- **目标**: 实现设计线宽 ±10% 的公差。
- **输入**: Stackup 定义的铜厚 (e.g., 1oz HTE Copper)。
- **关键参数**:
    - **蚀刻补偿**: 1oz 铜通常需要 25-35µm 的补偿量。
    - **蚀刻速率**: 1.2-1.8 m/min，通过调整药液浓度与温度控制。
    - **侧蚀控制**: 采用高精度喷淋系统和专用蚀刻液，将侧蚀量控制在 12µm 以内。
- **检测方法**: 自动光学检测 (AOI) 全检，关键线路使用切片分析测量线宽/线距。
- **DFM 建议**: 在进行[阻抗控制设计](/blog/what-is-impedance-control-in-pcb/)时，应与制造商沟通其蚀刻公差能力，预留足够的设计余量。

</div>

### 阻焊层教程：不止是“绿色油漆”

阻焊层（Solder Mask）是 PCB 的“外衣”，但其作用远超美观。它定义了焊接区域，防止焊接时发生桥连。HILPCB 采用 LDI (激光直接成像) 技术，相比传统菲林曝光，精度更高。

<div class="div-style-3">

#### 阻焊层 LDI 工艺步骤

1.  **板面预处理**: 通过化学清洗和机械研磨，增加板面粗糙度，确保油墨附着力。
2.  **油墨涂覆**: 在无尘车间内，使用全自动丝印机或喷涂设备，均匀涂覆液态感光阻焊油墨，厚度控制在焊盘处 8-15µm，线路处 20-30µm。
3.  **预烘 (Pre-curing)**: 在 75-85°C 下短暂烘烤，使油墨表干，便于后续 LDI 曝光。
4.  **LDI 曝光**: 激光直接在阻焊层上扫描成像，无需菲林，对位精度可达 ±15µm。这是实现精细间距元件（如 0.4mm BGA）之间阻焊桥的关键。
5.  **显影 (Developing)**: 在显影液中冲洗，未被激光曝光的区域被洗掉，露出焊盘。
6.  **终固化 (Final Curing)**: 在 150°C 左右的隧道炉中进行长时间烘烤，使阻焊层完全硬化，具备优异的物理和化学性能。

这个过程是典型的 **soldermask exposure tutorial**，其核心在于能量和精度的控制，确保阻焊桥不断裂、不偏移。

## 钻孔、镀层与孔铜的品质控制

孔（Via）是多层板的“垂直高速公路”，其可靠性至关重要。Stackup 文档定义了孔的类型（通孔、盲孔、埋孔）和尺寸，直接影响钻孔和电镀工艺的选择。

### 钻孔：精度与孔壁质量

高纵横比（板厚/孔径）的孔对钻孔和电镀都是巨大挑战。例如，一个 2.0mm 厚的板，钻一个 0.2mm 的孔，纵横比高达 10:1。

- **钻孔控制**: HILPCB 采用高转速（>150k RPM）气动主轴钻机，并使用 X-Ray 辅助对位，确保多层板的内层连接盘对位精确。对于微孔（<0.15mm），则采用激光钻孔。
- **去钻污**: 钻孔产生的高温会融化树脂，形成钻污（Smear）覆盖在内层铜上，影响电气连接。我们使用等离子去钻污（Plasma De-smear）工艺，有效清除孔壁树脂残留，确保后续镀铜的可靠附着。

### 孔铜（PTH）：可靠性的基石

孔内铜层的厚度和均匀性，是决定 PCB 在热冲击（如焊接）和长期使用中是否可靠的关键。行业标准（如 IPC-6012）通常要求孔铜平均厚度不低于 20µm。

- **控制方法**: 我们通过化学沉铜（Electroless Copper）形成导电底层，再通过图形电镀（Pattern Plating）加厚孔铜和面铜。HILPCB 的电镀线配备了先进的动态电流控制系统和高循环过滤装置，确保即使在高纵横比的孔内，镀层也均匀致密。
- **检测手段**: 定期进行切片分析，使用金相显微镜测量孔铜厚度、检查孔壁质量，确保无空洞（Void）、断裂（Crack）等缺陷。

## SMT 焊接与组装要点

当裸板（Bare Board）制造完成后，就进入了 PCBA（印制电路板组装）阶段。Stackup 文档中的焊盘设计、阻焊层定义，以及元器件布局，都直接影响着 SMT 的成败。

### 钢网设计：锡膏印刷的源头

锡膏印刷是 SMT 的第一步，也是超过 60% 焊接缺陷的来源。一份优秀的 **smt stencil design tutorial** 会强调，钢网（Stencil）的设计是成功的关键。

- **开口设计**: 钢网开口的尺寸和形状决定了锡膏的量。我们遵循面积比（>0.66）和宽高比（>1.5）原则，防止锡膏释放不完全。对于 0201、0.4mm BGA 等精细元件，会采用电抛光或纳米涂层钢网，提升脱模效果。
- **厚度选择**: 钢网厚度（通常为 100-150µm）需根据板上最小间距元件来确定，是设计与工艺结合的典范。

### 回流焊：温度曲线的艺术

回流焊（Reflow Soldering）是将元器件永久固定在 PCB 上的过程。其核心是精确控制温度曲线，以激活助焊剂、熔化焊料并形成可靠的金属间化合物（IMC）。

<div class="div-style-1">

#### 工艺窗口：无铅回流焊温度曲线

- **目标**: 形成光亮、饱满的焊点，无冷焊、虚焊、墓碑等缺陷。
- **输入**: 锡膏规格书 (e.g., SAC305)、PCB 板材与厚度、最大元器件热容。
- **关键参数**:
    - **预热区 (Preheat)**: 150-180°C，60-90s，缓慢升温，防止热冲击。
    - **恒温区 (Soak)**: 180-210°C，60-120s，激活助焊剂，使整板温度均匀。
    - **回流区 (Reflow)**: 峰值温度 240-250°C，液相线 (217°C) 以上时间 45-75s。
    - **冷却区 (Cooldown)**: 冷却速率 < 4°C/s，形成精细的晶粒结构，保证焊点强度。
- **检测方法**: HILPCB 使用 3D SPI（锡膏检测）、3D AOI（自动光学检测）和 X-Ray 检测，对焊接质量进行 100% 监控。

</div>

## 清洗、三防与可靠性处理

对于高可靠性要求的应用（如医疗、汽车、航空），焊接后的清洁度和防护措施至关重要。

- **清洗**: 即使使用“免清洗”助焊剂，在潮湿或高压环境下，其残留物也可能导致电化学迁移（ECM），引发短路。HILPCB 提供水基清洗工艺，并使用离子色谱仪（IC）检测板面清洁度，确保离子残留低于行业标准（如 IPC J-STD-001 要求 <1.56µg/cm² NaCl 等效值）。
- **三防涂覆 (Conformal Coating)**: 在清洗干燥后，选择性地喷涂一层透明的三防漆，可以有效抵御潮湿、盐雾和霉菌的侵蚀，大幅提升产品在恶劣环境下的寿命。

## 测试矩阵：确保每个节点都正确

DFT（可测试性设计）的理念必须贯穿始终。一个无法充分测试的产品，其质量是无法保证的。我们通过一个分阶段的测试矩阵来确保覆盖率。

| 测试阶段 (Test Stage) | 测试方法 (Test Method) | 主要目标 (Primary Goal) | 覆盖范围/缺陷类型 |
| :--- | :--- | :--- | :--- |
| **裸板制造后** | 飞针测试 (FPT) / 测试架 (Bed of Nails) | 验证线路的开路/短路 | 100% 网络连接关系，蚀刻/钻孔缺陷 |
| **SMT 焊接后** | 3D AOI (自动光学检测) | 检查焊接外观质量 | 错件、漏件、极性反、虚焊、桥连、墓碑 |
| **SMT 关键器件** | 3D X-Ray 检测 | 检查不可见焊点 (BGA, QFN) | BGA 焊球短路、空洞、枕头效应 (HoP) |
| **PCBA 电路级** | ICT (在线测试) | 检查元器件参数和电路网络 | 元件值错误、焊接开路/短路、数字逻辑功能 |
| **PCBA 功能级** | FCT (功能测试) | 模拟最终使用环境，验证板卡功能 | 固件烧录、信号输入/输出、通信接口、功耗 |
| **整机/系统级** | 系统级测试 (SLT) / 老化 (Burn-in) | 验证产品在真实应用中的稳定性和性能 | 兼容性问题、间歇性故障、早期失效 |
| **可靠性验证** | HALT/HASS, 温湿度循环, 振动 | 评估产品长期可靠性和设计余量 | 设计薄弱环节、潜在的失效模式 |

这个 **testing matrix** 构成了我们质量保证的骨架，从微观的焊点到宏观的功能，层层把关。

## 质量与可追溯性：数据的力量

在现代制造中，质量不是“检查”出来的，而是“制造”和“管理”出来的。

- **SPC (统计过程控制)**: 我们在蚀刻、电镀、回流焊等关键工序部署了 SPC 监控点，实时追踪过程参数（如药水浓度、炉温），一旦出现偏离趋势，系统会立即预警，让工程师在产生批量缺陷前进行干预。
- **MES (制造执行系统)**: HILPCB 的产线由 MES 系统全面管理。每块 PCB/PCBA 都有一个唯一的二维码，作为其“身份证”。从物料入库（由我们的智能仓储系统管理）到最终出货，所有工序数据、设备参数、人员信息、测试结果都与这个二维码绑定。这意味着，我们可以实现对任何一块产品的**端到端追溯**。如果发现问题，可以在数分钟内定位到影响范围，无论是特定批次的元器件，还是某个时间段的设备参数异常。
