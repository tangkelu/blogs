## 为什么学习 power net zoning 要先锁定需求场景？

**概念 (Concept):**
**Power net zoning** 的策略并非一成不变，它完全由产品需求驱动。一个为高功率服务器设计的电源分区方案，与一个为低功耗物联网传感器设计的方案截然不同。在动手设计前，必须澄清三大核心需求：电气性能、物理约束和成本目标。

**步骤 (Steps):**
1.  **电气需求分析：**
    *   **绘制电源树 (Power Tree)：** 从输入电源开始，梳理出每一路 LDO 或 DCDC 的输出电压、最大电流和目标纹波（例如，FPGA 内核要求 1.0V @ 10A，纹波 < 20mV）。
    *   **识别敏感电路：** 标记出对噪声敏感的电路，如 ADC/DAC、PLL 或 RF 前端。这些区域需要独立的、干净的电源和地。
    *   **定义关键信号：** 明确高速信号（如 PCIe, DDR4）的阻抗要求（例如，单端 50Ω ±5%，差分 90Ω ±7%），这直接影响叠层设计。

2.  **物理与环境约束：**
    *   **板框与高度限制：** 与结构工程师确认 PCB 的尺寸、形状和元器件限高，这是 **ecad mCAD handoff** 的第一步。
    *   **热点识别：** 识别主要发热器件（如 CPU、功率 MOSFET），预估其功耗，为散热设计预留铜皮面积和过孔。
    *   **工作环境：** 考虑产品将在何种温度、湿度或振动环境下工作，这会影响材料选择和机械可靠性。

3.  **成本与供应链目标：**
    *   **目标成本：** 明确 PCB 的目标制造成本，这决定了你可以使用多少层板、何种板材以及工艺复杂度（如 HDI）。
    *   **供应链约束：** 了解合作板厂的工艺能力，例如最小线宽/间距、最小过孔尺寸等。

## 叠层/材料/约束的推导步骤

**概念 (Concept):**
叠层（Stackup）是 PCB 的“骨架”，是 **power net zoning** 的物理基础。一个精心设计的叠层可以提供清晰的信号返回路径、良好的电源/地平面耦合以及天然的 EMI 屏蔽，事半功倍。

**步骤 (Steps):**
1.  **确定层数：** 根据信号密度、电源网络数量和 EMI 要求初步确定层数。一个经验法则是：当 BGA 引脚密度超过 0.8mm 或高速信号（>1Gbps）数量较多时，至少需要 6 层板。
2.  **选择板材：**
    *   **标准 FR-4:** 适用于大多数 <3GHz 的应用。
    *   **中/高 Tg FR-4:** 适用于无铅焊接工艺或工作温度较高的产品，如 [HILPCB 的 High-Tg PCB](https://hilpcb.com/en/products/high-tg-pcb)。
    *   **高速材料 (e.g., Rogers, Megtron 6):** 适用于 >10GHz 的射频或超高速数字信号，损耗更低。
3.  **规划层功能：** 这是核心。目标是让每个信号层都紧邻一个完整的参考平面（GND 或 Power）。
4.  **计算阻抗参数：** 使用阻抗计算器（如 **HILPCB 的在线阻抗计算器**）输入材料的介电常数 (Dk)、损耗角正切 (Df)、介质厚度和目标阻抗，反推出所需的线宽。
5.  **制定 DRC 规则：** 将计算出的线宽、安全间距（通常为 3-5 倍线宽）以及过孔尺寸等参数，转化为 ECAD 工具中的设计规则。

**叠层范例：一个典型的 6 层混合信号板**

| 层 | 类型 | 功能描述 | 关键考量 |
| :- | :--- | :--- | :--- |
| 1 | Signal (Top) | 主要元件层，短距离关键信号 | 避免在敏感模拟电路下方布设高速数字线 |
| 2 | GND | 主接地平面，提供顶层信号的返回路径 | 保持平面完整，仅在必要时做分割 |
| 3 | Signal | 低速控制信号、部分电源走线 | 远离高速信号，避免串扰 |
| 4 | Power | 电源平面，可分割为多个 **power net zoning** 区域 | 确保大电流路径足够宽，避免 IR Drop |
| 5 | GND | 次接地平面，提供底层信号返回路径 | 通过 **via stitching planning** 与 L2 GND 紧密连接 |
| 6 | Signal (Bottom) | 次要元件，非关键信号 | 远离敏感区域，用于连接器等 |

## 布局布线学习路径：从模块到细节

**概念 (Concept):**
布局布线是将逻辑设计转化为物理现实的过程。一个优秀的布局布线方案，应遵循“先整体，后局部；先关键，后次要”的原则，将 **power net zoning** 的规划落地。

**步骤 (Steps):**
1.  **模块化布局 (Floorplanning):**
    *   根据原理图的功能模块（如电源、CPU 核心、DDR 内存、RF 前端），在 PCB 上划分出物理区域。
    *   将接口（如 USB, HDMI, 天线）放置在板边，缩短其走线路径。
    *   将模拟区域和数字区域物理隔离，并规划好它们之间的接口位置。

2.  **关键元件放置：**
    *   **核心器件优先：** 首先放置 CPU/FPGA 等核心 BGA 器件。
    *   **去耦电容就近放置：** 将去耦电容尽可能靠近 IC 的电源引脚，路径越短越好。一般遵循“大电容稍远，小电容贴近”的原则。
    *   **时钟和晶振：** 将晶振及其负载电容紧凑放置，并用地平面包裹，减少噪声辐射。

3.  **电源与地平面处理：**
    *   **大面积铺铜：** 为主电源和地网络使用完整的平面层，以提供低阻抗路径。
    *   **电源平面分割：** 在电源层上，根据 **power net zoning** 规划，分割出不同的电压域。分割的间隙（Gap）至少保持 0.5mm 以上，避免耦合。
    *   **避免跨分割布线：** 这是 **emi safe routing tutorial** 中的黄金法则。任何高速信号线都不能跨越其参考平面的分割区，否则会造成严重的回流路径不连续，产生强烈的 EMI。

4.  **扇出与布线 (Fanout & Routing):**
    *   **制定扇出策略：** 对于 BGA 器件，需要一个清晰的 **fanout strategy tutorial**。例如，使用微孔（Microvia）进行 HDI 设计，或在传统通孔工艺下，规划好过孔的逃逸方向。
    *   **关键信号优先：** 先布时钟、复位、高速差分对等关键信号。
    *   **电源走线：** 对于无法使用平面的次要电源，应使用尽可能短而粗的走线。线宽可根据 IPC-2221 标准或在线工具计算，确保满足载流量要求。

5.  **接地增强与屏蔽：**
    *   **执行 via stitching planning：** 在电路板的GND区域，特别是边缘和分割区附近，大量放置接地过孔（Stitching Vias）。这能有效降低地平面的阻抗，并形成一个法拉第笼，抑制内部噪声向外辐射。推荐的间距是信号波长的 1/20 或更小。

**工具 (Tools):**
*   **ECAD 布局布线工具 (Altium, Cadence, KiCad):** 核心设计平台。
*   **IPC-2152/2221 标准:** 用于计算走线载流量和温升。

## SI/PI/EMC 检查要点与工具组合

**概念 (Concept):**
“无仿真，不设计”。在将设计文件发给 **HILPCB** 这样的制造商之前，通过仿真进行早期验证，可以发现 80% 以上的潜在问题，避免昂贵的硬件重制。

**步骤 (Steps):**
1.  **电源完整性 (PI) 检查：**
    *   **DC 分析 (IR Drop):** 仿真从电源输入端到芯片引脚的直流压降。目标是确保压降在芯片规格允许的范围内（通常为 2-3%）。如果压降过大，需要加宽电源走线/铜皮或增加过孔。
    *   **AC 分析 (PDN Impedance):** 仿真电源分配网络在不同频率下的阻抗。目标是在芯片工作频率范围内，PDN 阻抗足够低，以快速响应瞬态电流需求。去耦电容的容值、ESR/ESL 和布局是优化的关键。

2.  **信号完整性 (SI) 检查：**
    *   **阻抗匹配：** 检查关键信号线的阻抗是否在目标范围内（如 50Ω ±5%）。
    *   **串扰分析 (Crosstalk):** 分析相邻信号线之间的耦合程度，确保噪声不会导致信号误判。
    *   **时序与眼图：** 对于高速总线（如 DDR），进行时序仿真，确保建立/保持时间满足要求，并检查眼图的张开程度。

3.  **电磁兼容性 (EMC) 检查：**
    *   **返回路径检查：** 使用仿真工具高亮显示信号的返回路径，检查是否存在不连续点（如跨分割）。
    *   **热点区域识别：** 识别出潜在的强辐射区域，如开关电源的开关节点、高速信号的连接器端接处。
    *   **屏蔽效能分析：** 评估 **via stitching planning** 的效果，以及机壳接地的有效性。

<div class="div-style-1">
    <h4>知识要点：仿真工具的正确使用</h4>
    <p>仿真工具不是“一键运行”的魔法棒。其结果的准确性高度依赖于模型的质量（如元器件的 S 参数模型、IBIS 模型）和正确的设置。对于初学者，建议从简单的 DC Drop 分析开始，逐步学习更复杂的 AC 和 SI 仿真。HILPCB 的工程师团队也可以在 DFM 审查阶段提供关于电源路径宽度和过孔数量的建议。</p>
</div>

**工具 (Tools):**
*   **专业仿真套件:** Ansys SIwave, Keysight ADS, Cadence Sigrity, HyperLynx。
*   **ECAD 内置工具:** Altium Designer 的 PDN Analyzer 等，适合进行初步的快速检查。
*   **HILPCB DFM 工具:** 在制造前检查物理间距、铜皮连接等基本问题。

## 设计审查、文档与交付物模板

**概念 (Concept):**
一个成功的设计不仅要在技术上可行，还要能被顺利地制造、组装和测试。标准化的审查流程和清晰的交付文档是确保项目成功的最后一道防线。

**步骤 (Steps):**
1.  **内部设计审查 (Peer Review):**
    *   在正式交付前，组织团队成员进行交叉审查。
    *   使用标准化的 Checklist，覆盖原理图、布局、DRC、制造文件等各个方面。
    *   重点关注 **power net zoning** 的实现是否与设计初衷一致。

2.  **ECAD-MCAD 协同：**
    *   这是 **ecad mCAD handoff** 的关键环节。通过 3D 模型（如 STEP 文件）与结构工程师确认 PCB 板、连接器、散热器等部件的装配关系，避免干涉。
    *   使用协同平台（如 Altium 365）可以实时同步设计变更，提高效率。

3.  **生成制造与组装文件：**
    *   **Gerber RS-274X 或 ODB++:** PCB 裸板制造文件。
    *   **NC Drill File:** 钻孔文件。
    *   **Bill of Materials (BOM):** 元器件清单，需包含精确的制造商型号、封装等信息。**HILPCB 的 BOM Viewer** 是一个很好的工具，可以帮助检查 BOM 的完整性。
    *   **Pick and Place File (Centroid File):** 元器件贴装坐标文件。
    *   **Assembly Drawing:** 组装图，清晰标注元器件位号、方向（特别是对于二极管、电解电容等）。

4.  **编写制造说明 (Fab Notes):**
    *   在图纸中明确叠层结构、板材、板厚、铜厚、阻焊颜色、表面处理等要求。
    *   如果有阻抗控制要求，需明确指出哪些走线需要控制，以及目标阻抗和公差，并要求制造商提供阻抗测试条 (Coupon)。

**DFM 审查清单 (部分示例)**

| 类别 | 检查项 | 建议标准/说明 |
| :--- | :--- | :--- |
| **电源/地** | 电源走线宽度 | 根据 IPC-2152 检查载流量，确保有足够裕量 |
| | 孤岛铜皮 (Copper Islands) | 移除所有未连接的孤岛铜皮，或将其连接到 GND |
| | 热焊盘 (Thermal Reliefs) | 检查大面积铺铜上的过孔和焊盘是否有热焊盘连接，便于焊接 |
| **过孔** | 过孔盖油/塞油 | 明确过孔的处理方式（Tenting, Plugging, Filling），特别是 BGA 下方的过孔 |
| | 盘中孔 (Via-in-Pad) | 若使用，需确认制造商支持该工艺（如 VIPPO） |
| **间距** | 铜皮到板边距离 | 通常建议 ≥ 0.3mm，避免在 V-cut 或邮票孔处露铜 |
| | 不同网络间距 | 运行最终 DRC 检查，确保无违规 |

## 实战练习：从评审清单到迭代复盘

**概念 (Concept):**
理论学习必须通过实践来巩固。下面提供几个具体的练习任务，帮助你将前面学到的知识融会贯通，形成自己的 **pcb design roadmap**。

**练习任务 1：重构一个 6 层板叠层**
*   **场景：** 你正在设计一个基于 ARM Cortex-A 核心板，带有 DDR3 内存和千兆以太网。
*   **任务：**
    1.  定义电源层：至少需要 3.3V, 1.8V, 1.5V, 1.0V 四个主要电源。
    2.  规划叠层结构，目标是为 DDR3 和以太网信号提供最佳的返回路径。
    3.  使用在线工具计算 90Ω 差分阻抗和 50Ω 单端阻抗所需的线宽和间距。
    4.  输出一份包含层功能、材料、厚度、线宽参数的叠层设计文档。

**练习任务 2：编写 DRC 规则集**
*   **场景：** 在一个混合信号设计中，你有一个敏感的 24 位 ADC 和一个嘈杂的开关电源。
*   **任务：**
    1.  在你的 ECAD 工具中，创建两个不同的元件类 (Component Class)：`Analog_Sensitive` 和 `Power_Noisy`。
    2.  创建两个网络类 (Net Class)：`Analog_GND` 和 `Digital_GND`。
    3.  设置一条设计规则：`Analog_Sensitive` 元件必须放置在预定义的“模拟区”内。
    4.  设置另一条规则：`Analog_GND` 和 `Digital_GND` 之间的铜皮间距必须大于 1.0mm，并且只允许在某一点通过一个 0Ω 电阻或磁珠单点连接。

**练习任务 3：执行一次设计复盘**
*   **场景：** 你完成了一个设计的初步布局布线。
*   **任务：**
    1.  下载一份通用的 PCB 设计审查 Checklist（网上有很多模板）。
    2.  逐项检查你的设计，特别是关于 **power net zoning**、返回路径和 **via stitching planning** 的部分。
    3.  找出至少 3 个可以改进的地方，并记录下改进方案。例如：“CPU 下方的去耦电容布局可以更紧凑”、“高速 USB 差分线跨越了地平面分割，需要重新布线”。
