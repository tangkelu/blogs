## 1. 摘要与业务价值

<div class="content-section" id="section-1">

**致系统架构师与采购团队：**

在当今高密度、高功率和高可靠性要求的电子产品中，**热循环应力 (Thermal Cycling Stress)** 是导致现场故障的头号隐形杀手。从车载电子到数据中心服务器，再到工业控制模块，设备在运行和环境变化中经历的反复温度波动，会对 PCB 的结构完整性构成严峻挑战。这种应力源于 PCB 内部不同材料（铜、树脂、玻璃纤维、元器件）之间热膨胀系数 (CTE) 的不匹配，最终导致过孔开裂 (barrel cracking)、焊盘翘曲 (pad lifting)、分层 (delamination) 等一系列可靠性问题，造成巨大的保修成本和品牌声誉损失。

本白皮书由 HILPCB 材料实验室撰写，旨在为您提供一个系统性的框架，以应对和管理热循环应力。我们不仅仅是提供一份材料清单，而是交付一套完整的 **PCB 材料与叠层策略 (PCB Material Strategy & Stackup Library)**。

**核心业务价值：**

1.  **加速产品开发 (NPI)：** 通过我们预验证的叠层模板库和材料决策树，您的团队可以跳过繁琐的初期选型和验证过程，将设计周期缩短 15-30%。
2.  **降低供应链风险：** 我们提供主流材料的在库清单、详细的交期数据以及经过验证的替代料方案，确保您的生产计划不受单一供应商或材料短缺的影响。
3.  **控制项目总成本 (TCO)：** 通过选择“恰到好处”的材料而非一味追求最高规格，并利用混压 (Hybrid Stackup) 策略，我们帮助您在性能、可靠性和成本之间找到最佳平衡点。
4.  **提升产品可靠性：** 我们的 DFM/DFR 清单和验证矩阵，将可靠性设计前置，系统性地规避因热应力导致的潜在失效模式，从而显著降低现场故障率。

本白皮书将是您在设计初期进行关键决策的战略地图，确保您的产品从诞生之初就具备卓越的长期可靠性。

## 2. 材料决策树：性能、成本与交期的平衡

选择正确的材料是应对热循环应力的第一道防线。核心目标是选择在 Z 轴（厚度方向）上具有较低 CTE 的材料，以减少对过孔的拉伸应力。同时，高玻璃化转变温度 (Tg) 确保了材料在高温下仍能保持机械稳定性。

下表是 HILPCB 基于数千次量产项目经验总结的材料决策树，帮助您快速定位。

| :--- | :--- | :--- | :--- | :--- | :--- | :--- |

**决策指南：**
*   **从 High Tg (≥170°C) 开始：** 对于大多数需要高可靠性的产品，IT-180A 或同级材料是性价比最高的起点。它们的 Z-CTE 相对较低，能应对大多数热循环挑战。
*   **关注 Z-CTE，而非仅仅是 Tg：** Tg 高不代表 Z-CTE 一定低。在热循环应力管理中，Z-CTE 是更直接、更关键的指标。
*   **咨询 HILPCB 库存：** 在最终确定材料前，请联系我们的团队。我们常备 IT-180A、S1170 等多种 High Tg 材料，可以为您提供最快的打样和量产服务。

## 3. 叠层模板库 (Stackup Template Library)

<div class="content-section" id="section-3">

一个优化的叠层不仅关乎阻抗控制，更是一项精密的应力管理工程。HILPCB 的叠层库基于仿真和实测数据，为不同应用场景提供了可靠的起点。

#### 3.1 高速数字叠层 (High-Speed Digital Stackup)
*   **目标：** 信号完整性、阻抗控制、最小化串扰。
*   **策略：**
    *   **对称结构：** 核心层 (Core) 和半固化片 (PP) 对称分布，以防止翘曲。
    *   **紧耦合参考平面：** 信号层紧邻大面积地平面，提供清晰的回流路径。
    *   **低损耗材料：** 选用 ITEQ IT-968 或更高级别的材料，以支持 28/56/112 Gbps 速率。
    *   **背钻就绪 (Backdrill Ready)：** 预留足够的介质厚度，以便在制造中通过背钻移除过孔残桩 (stub)，减少信号反射。
*   **示例 (12层服务器主板)：**
    1.  SIG (Stripline)
    2.  GND
    3.  SIG (Stripline)
    4.  PWR
    5.  GND
    6.  SIG (Stripline)
    7.  SIG (Stripline)
    8.  GND
    9.  PWR
    10. SIG (Stripline)
    11. GND
    12. SIG (Stripline)

#### 3.2 电源与功率电子叠层 (Power Electronics Stackup)
*   **目标：** 载流能力、热管理、高压隔离。
*   **策略：**
    *   **重铜 (Heavy Copper)：** 内层使用 3oz 或更厚的铜箔，以承载大电流并辅助散热。访问我们的 [重铜 PCB 页面](https://hilpcb.com/en/products/heavy-copper-pcb) 了解更多。
    *   **高 CTI 材料：** 选择相比漏电起痕指数 (CTI) ≥ 600V 的材料，以满足安规要求。
    *   **厚 PP 片：** 在高压层之间使用更厚的半固化片，确保足够的绝缘距离。
    *   **热过孔阵列 (Thermal Via Array)：** 在发热器件下方密集布置热过孔，将热量快速传导至底层或散热器。

#### 3.3 HDI / 精密间距叠层 (HDI Stackup)
*   **目标：** 布线密度、小型化、BGA 可靠性。
*   **策略：**
    *   **低 CTE 材料：** 必须选用 Z-CTE < 50 ppm/°C 的材料，以保护脆弱的微孔 (microvias)。
    *   **堆叠微孔 (Stacked Microvias)：** 采用填铜电镀工艺，支持 1+N+1, 2+N+2 等任意层互联结构。
    *   **树脂填充 (Resin Fill)：** 确保激光钻孔的介质层不含玻璃纤维，以获得完美的微孔形态。
*   **推荐：** 我们的 [HDI PCB 服务](https://hilpcb.com/en/products/hdi-pcb) 结合了先进材料和制程，确保高密度设计的可靠性。

#### 3.4 柔性与刚柔结合叠层 (Flex & Rigid-Flex Stackup)
*   **目标：** 动态弯曲寿命、应力释放。
*   **策略：**
    *   **无胶基材 (Adhesiveless Core)：** 避免使用胶粘剂层，它在热循环中是薄弱环节。
    *   **弯曲区设计：** 信号线在弯曲区域应保持均匀宽度和间距，并垂直于弯曲方向。
    *   **应力释放角 (Tear-drop pads)：** 在刚柔结合区域的焊盘和过渡区增加泪滴，以分散应力。

</div>

## 4. 混压与特殊材料应用

为了在成本和性能之间实现极致平衡，混压 (Hybrid Stackup) 是一种高级策略。HILPCB 在处理复杂混压方面拥有丰富的 NPI 经验。

#### 4.1 Rogers + FR-4 混压
*   **场景：** 射频前端 (RFFE) 与数字控制电路集成在同一块板上。
*   **方案：**
    *   将昂贵的 Rogers 或 Taconic 材料 仅用于表层或特定的核心层，处理高频信号。
    *   其余层使用高性能的 FR-4 材料 (如 IT-180A) 来承载数字信号和电源。
*   **挑战与 HILPCB 解决方案：**
    *   **压合兼容性：** 不同材料的树脂流动性和压合温度曲线不同。HILPCB 压合实验室通过动态机械分析 (DMA) 优化压合程序，防止分层或树脂空洞。
    *   **钻孔参数：** 针对不同材料设置分段钻孔速度和进给，防止钻污和孔壁撕裂。

#### 4.2 陶瓷基板 (Ceramic PCB)
*   **场景：** 极高功率密度、恶劣温度环境（如 SiC/GaN 功率模块）。
*   **优势：**
    *   **极佳的导热性：** 氧化铝 (Al2O3) 或氮化铝 (AlN) 的导热系数是 FR-4 的数十倍。
    *   **极低的 CTE：** 与芯片的 CTE 非常匹配，显著降低了焊接点的热疲劳。
*   **HILPCB 服务：** 我们提供 DBC (直接键合铜) 和 DPC (直接电镀铜) 陶瓷 PCB 解决方案。

#### 4.3 金属芯/基板 (MCPCB / IMS)
*   **场景：** 大功率 LED 照明、电机驱动。
*   **结构：** 由电路层、超薄高导热绝缘层和金属基板（通常是铝）组成。
*   **优势：** 将热量从发热源高效地传导至金属基板，再通过外壳散发。这是最具成本效益的高效散热方案之一。了解我们的 金属芯 PCB。

#### 4.4 埋铜块 (Copper Coin)
*   **场景：** 针对特定大功率芯片（如 FPGA, CPU）的局部散热。
*   **方案：** 在 PCB 压合过程中，将一个实心铜块嵌入叠层中，直接接触芯片底部的散热焊盘。
*   **优势：** 提供了从芯片到 PCB 背面散热器的最短、最低热阻的路径，散热效率远超热过孔阵列。

## 5. 验证矩阵：从仿真到实测的闭环

一个成功的叠层策略需要通过严格的验证来确保其在真实世界中的表现。HILPCB 的阻抗与可靠性实验室提供一站式验证服务。

| 验证项目 | HILPCB 方法与工具 | 关键目标 (针对热循环应力) |
| :--- | :--- | :--- |
| **阻抗验证** | Tektronix TDR, Polar SI9000 仿真 | 确保材料的 Dk/Df 在不同温度下保持稳定，阻抗波动在 ±5% 以内。 |
| **热冲击测试 (TCT)** | 空气-空气热冲击箱 (-65°C to +150°C) | 模拟最严苛的温度变化，通过切片分析，检查 1000 次循环后过孔是否有微裂纹。 |
| **互连应力测试 (IST)** | IST 测试系统 | 通过对测试 Coupon 快速加热和冷却，实时监测过孔电阻变化，精确评估过孔的疲劳寿命。 |
| **翘曲分析** | Shadow Moiré, Strain Gauge | 测量 PCB 在回流焊温度曲线下的动态翘曲。确保翘曲度 < 0.75%，防止 BGA 焊接失效。 |
| **CAF 电阻测试** | 高温高湿偏压箱 (85°C/85%RH) | 评估材料在恶劣环境下抵抗导电阳极丝 (CAF) 生长的能力，防止层间短路。 |
| **背钻质量验证** | X-Ray, 切片分析 | 检查背钻深度精度和残桩 (stub) 长度，确保其符合高速信号要求且不损伤内层。 |

**Coupon 设计实践 (Coupon Design Practice):**
我们强烈建议在每块生产拼板 (Panel) 上都包含专门设计的测试 Coupon。HILPCB 可以协助您设计：
*   **阻抗 Coupon:** 包含单端和差分线，用于 TDR 测试。
*   **可靠性 Coupon:** 包含不同尺寸的过孔链 (daisy chain)，用于 IST 和 TCT 测试。
*   **CAF Coupon:** 用于评估材料的长期绝缘性能。

## 7. 风险控制与 DFM/DFR 清单

预防是最好的策略。下表是我们为系统架构师和 PCB 设计师准备的 DFM (Design for Manufacturability) 和 DFR (Design for Reliability) 清单，专注于规避热循环应力风险。

| 分类 | 检查点 | 风险描述 (若忽略) | HILPCB 推荐实践 |
| :--- | :--- | :--- | :--- |
| **材料选择** | 1. 是否选择了 Tg ≥ 170°C 的材料？ | 低 Tg 材料在高温下软化，Z-CTE 急剧增大，极易导致过孔断裂。 | 除非成本极其敏感，否则 IT-180A 或同级材料应作为基准。 |
| | 2. Z-CTE 是否低于 55 ppm/°C？ | 高 Z-CTE 会在过孔铜壁上施加巨大的拉伸应力。 | 对于厚板或高可靠性要求，应选择 Z-CTE < 50 ppm/°C 的材料。 |
| | 3. 材料的 Td (分解温度) 是否足够？ | Td 低的材料在多次无铅回流焊后可能降解，导致分层。 | 选择 Td ≥ 340°C 的材料。 |
| | 4. 是否考虑了吸湿性？ | 高吸湿性材料在回流焊时会产生“爆米花效应”，导致分层。 | 选择吸湿性 < 0.2% 的材料，并确保生产前充分烘烤。 |
| **叠层设计** | 5. 叠层是否完全对称？ | 不对称叠层会在冷却时因应力不均而导致严重翘曲。 | 铜层、介质层、PP 片的厚度和类型应中心对称。 |
| | 6. PP 片是否选择了合适的型号？ | 树脂含量不足 (Resin Starvation) 会导致内层线路间出现空洞。 | 使用高树脂含量 (High RC) 的 PP 片填充大铜面区域。 |
| | 7. 芯板 (Core) 和 PP 片是否来自同一系列？ | 不同系列的材料 CTE 和压合特性不匹配，存在分层风险。 | 坚持使用同一制造商的同一材料体系。 |
| | 8. 叠层总厚度是否超过 3.2mm？ | 厚板的 Z 轴膨胀总量更大，对过孔的挑战更严峻。 | 对于厚板，必须使用低 Z-CTE 材料，并增大过孔的纵横比。 |
| **过孔与焊盘** | 9. 过孔的纵横比 (Aspect Ratio) 是否 < 10:1？ | 高纵横比的孔难以均匀电镀，中间部分镀层薄，易在热应力下断裂。 | 优选 < 8:1。若必须 > 10:1，需采用特殊的电镀工艺。 |
| | 10. 是否使用了泪滴 (Teardrop)？ | 尖锐的连接点是应力集中区，钻孔偏移时易导致断路。 | 在所有过孔和焊盘上添加泪滴，尤其是在薄的走线上。 |
| | 11. 非功能焊盘 (NFP) 是否被移除？ | NFP 是过孔铜柱上的“赘肉”，会阻碍 Z 轴膨胀，增加应力。 | 在内层移除所有不连接的焊盘，以释放应力。 |
| | 12. BGA 焊盘是否采用 NSMD 设计？ | SMD 焊盘的应力集中在焊点，而 NSMD 将应力分散到阻焊层。 | 推荐使用非阻焊定义 (NSMD) 焊盘，以提高 BGA 焊点的抗疲劳性。 |
| | 13. 过孔是否离 BGA 焊盘太近？ | 过孔会“偷走”焊膏，导致虚焊，并在热应力下影响焊盘。 | 遵循 IPC 标准，保持安全间距，或使用盘中孔 (Via-in-Pad) 工艺。 |
| **铜箔与布线** | 14. 内层铜箔是否均匀分布？ | 大面积铜箔和无铜区并存会导致局部应力不均，引起翘曲。 | 在无布线区域使用网格填充 (Hatching) 来平衡铜箔分布。 |
| | 15. 电源层和地层是否设计为负片？ | 正片设计的平面层可能因蚀刻不均产生应力。 | 尽可能使用负片（反白）设计大面积平面层。 |
| | 16. 走线是否避免 90 度转角？ | 尖角是酸蚀陷阱 (acid trap) 和信号反射点。 | 使用 45 度或圆弧转角。 |
| **可靠性** | 17. 是否设计了测试 Coupon？ | 没有 Coupon 就无法通过实测来验证制造质量和材料性能。 | 在拼板边缘设计专用的阻抗和可靠性 Coupon。 |
| | 18. 是否要求进行热冲击或 IST 测试？ | 这是评估过孔在热循环下可靠性的最直接方法。 | 对于高可靠性产品，应将 TCT 或 IST 测试作为出货标准。 |
| | 19. 是否考虑了 CAF 风险？ | 小间距过孔之间在高温高湿下可能形成导电通路。 | 保持足够的孔到孔间距，并选择抗 CAF 性能好的材料。 |
