## 1. 执行摘要：从混乱到协同，奠定硬件研发基石

对于高速发展的初创研发团队而言，产品迭代速度是生命线。然而，许多团队在硬件开发中最基础的环节——PCB设计——遭遇了无形的瓶颈。工程师各自为战，导致 **pcb stack layer naming** 规范不一、设计文件混乱、与制造商沟通不畅，最终引发了代价高昂的返工、项目延期和产品可靠性问题。这些混乱的根源在于缺乏一个统一、标准化的设计体系。

**痛点：**
*   **术语鸿沟：** 设计、制造、测试团队对“顶层”、“GTL”、“Solder Mask”等术语理解不一，造成沟通障碍。
*   **流程缺失：** 缺乏从需求到交付的标准化流程，设计质量依赖个人经验，难以复制和扩展。
*   **标准真空：** 没有建立内部的材料库、叠层标准和设计规则，每次新项目都从零开始，效率低下且风险高。
*   **交付混乱：** 提交给制造商的 Gerber、BOM、坐标文件等资料包不完整或存在隐性错误，导致生产延误和质量缺陷。

**机会与目标：**
本白皮书旨在解决这些核心痛点。通过建立一套以 **pcb stack layer naming** 为核心的基础体系，初创团队可以抓住以下机会：
*   **建立单一事实来源 (Single Source of Truth)：** 统一的命名和文档标准，确保所有项目成员和供应链伙伴基于同样的信息协同工作。
*   **提升设计可复制性：** 标准化的流程和规则库，使成功的设计经验得以沉淀和复用，加速新项目启动。
*   **降低制造成本与风险：** 强大的 DFM/DFT 清单和评审流程，能在设计阶段发现并修复潜在的制造与测试问题，显著提高一次交付成功率。

本白皮书将作为您的行动指南，由 **HilPCBPCB Factory (HILPCB)** 设计赋能顾问团队为您呈现。我们的目标是帮助您的团队搭建一个从术语、流程到质量门禁的完整PCB设计基础体系，将混乱转化为高效、可靠的协同作业，为您的创新产品奠定坚实的硬件基础。

## 2. 术语与结构图谱：统一PCB设计的“世界语”

精确的沟通始于统一的语言。在PCB设计中，对层、材料和关键特征的命名与理解是所有协同工作的基础。下表梳理了核心术语，构成了我们的 **pcb terminology glossary**，旨在消除歧义，确保设计意图被准确传达。

| 名词 (Term) | 定义 (Definition) | 对设计/制造的影响 (Impact) | 参考标准/最佳实践 |
| :--- | :--- | :--- | :--- |
| **Signal Layer (信号层)** | 用于布设信号走线的铜箔层。通常命名为 L1/Top, L2, L3... Ln/Bottom。 | 决定了布线空间、信号完整性和阻抗控制。层数越多，设计自由度越高，成本也越高。 | IPC-2221 |
| **Plane Layer (平面层)** | 通常整片覆铜，用于分配电源 (PWR) 或作为接地 (GND) 参考。 | 提供低阻抗的电流回路，改善EMI/EMC性能，并有助于散热。分割平面会影响信号完整性。 | 优先使用完整地平面作为高速信号的参考层。 |
| **Solder Mask (阻焊层)** | 覆盖在铜箔上的一层绝缘保护涂层，仅露出需要焊接的焊盘。 | 防止焊接时发生桥连，保护走线免受氧化和物理损伤。阻焊桥的精度是关键制造参数。 | IPC-SM-840 |
| **Silkscreen (丝印层)** | 在阻焊层之上，用于印刷元器件位号、极性标记、公司Logo等信息的图层。 | 辅助手工焊接、调试和维修。清晰的 **silkscreen design rules** 能极大提升可维护性。 | 丝印不应覆盖焊盘，字体大小和线宽需满足制造商的最小要求。 |
| **Paste Mask (钢网层)** | 用于制作SMT钢网的文件层，定义了需要涂敷焊膏的位置和形状。 | 焊膏的量和形状直接影响SMT焊接质量，如虚焊、短路等。通常比焊盘略小（内缩）。 | IPC-7525 |
| **Core (芯板)** | PCB叠层结构中的核心材料，双面覆铜的刚性基板。 | 决定了PCB的基础机械强度和介电性能。其厚度和铜厚是阻抗计算的关键参数。 | FR-4 是最常用的芯板材料。 |
| **Prepreg (PP片/半固化片)** | 由玻璃纤维布浸渍环氧树脂制成的薄片，在层压过程中作为粘合剂和绝缘层。 | 其厚度和介电常数 (Dk) 决定了层间距和阻抗。多层板通过PP片将多个芯板粘合在一起。 | [FR-4 PCB](https://hilpcb.com/en/products/fr4-pcb) 制造中的关键材料。 |
| **Annular Ring (焊环)** | 钻孔周围环绕的铜环，即焊盘减去钻孔的部分。 | **annular ring definition**：提供钻孔与导电图形之间的可靠连接。过小的焊环可能导致钻偏后断裂（破环）。 | IPC-6012 对不同等级的PCB有最小焊环要求。 |
| **Board Outline (板框层)** | 定义PCB物理边界、外形、开槽和挖空区域的图层。 | 制造的第一步就是根据此层进行切割成型。尺寸精度至关重要。 | 通常使用 Keep-Out Layer 或 Mechanical Layer 定义。 |

## 3. 流程框架：从需求到交付的可复制体系

一个成功的PCB设计项目，不是灵感的迸发，而是一套纪律严明的流程。我们将其分解为六个关键阶段，确保每个环节都有明确的输入、输出和质量门槛。

**阶段一：需求分析与技术选型 (Input)**
*   **输入：** 产品需求文档 (PRD)、系统框图、关键器件Datasheet、结构尺寸限制。
*   **活动：**
    1.  确定电路功能、性能指标（如高速信号速率、功耗、散热要求）。
    2.  评估PCB层数、板材（如FR-4, Rogers, 高Tg）、特殊工艺（如HDI、厚铜）。
    3.  建立初步的 **design rule file setup**，定义线宽、间距、过孔等基础约束。
*   **输出：** PCB设计规格书、元器件选型清单 (初步BOM)。

**阶段二：栈叠设计与阻抗规划 (Stackup)**
*   **输入：** PCB设计规格书。
*   **活动：**
    1.  **核心活动：** 定义 **pcb stack layer naming** 规范，例如：`L1_Top_Sig`, `L2_GND_Plane`, `L3_Pwr_Plane`, `L4_Bot_Sig`。
    2.  与制造商（如 HILPCB）沟通，获取可制造的材料参数（Dk, Df, 厚度）。
    3.  使用阻抗计算工具（如 HILPCB Impedance Calculator）规划关键信号线的线宽和层间距，以满足目标阻抗（如50Ω单端，100Ω差分）。
*   **输出：** 确定的PCB叠层结构图、阻抗规则表、更新后的设计规则文件。

**阶段三：布局 (Layout)**
*   **输入：** 原理图网表、叠层结构、封装库、结构文件 (DXF)。
*   **活动：**
    1.  根据信号流向和模块功能进行分区布局（如电源区、模拟区、数字区）。
    2.  放置连接器、安装孔等位置敏感型器件。
    3.  放置核心芯片，并围绕其布置关键外围电路（如时钟、DDR）。
    4.  放置去耦电容，使其尽可能靠近IC的电源引脚。
*   **输出：** 完成元器件布局的PCB文件。

**阶段四：布线 (Routing)**
*   **输入：** 完成布局的PCB文件。
*   **活动：**
    1.  优先布设关键网络：时钟、高速差分对、电源主干路、敏感模拟信号。
    2.  进行大面积覆铜（GND/PWR）。
    3.  完成剩余信号的布线。
    4.  优化走线，消除直角、锐角，调整泪滴，确保信号回流路径完整。
*   **输出：** 完成布线的PCB文件。

**阶段五：设计评审与优化 (Review)**
*   **输入：** 完成布线的PCB文件。
*   **活动：**
    1.  **电气规则检查 (ERC)：** 检查原理图的电气连接性。
    2.  **设计规则检查 (DRC)：** 运行EDA工具的DRC，确保没有违反预设的物理规则。
    3.  **可制造性/可测试性 (DFM/DFT) 评审：** 对照DFM/DFT清单（见第8节）进行人工审查。**HILPCB DFM** 服务在此阶段可提供专业的工程师建议，提前发现制造陷阱。
    4.  **信号/电源完整性 (SI/PI) 仿真：** 对高速或大功率设计进行仿真验证。
*   **输出：** 经过评审和修改的最终PCB设计文件。

**阶段六：生产资料交付 (Delivery)**
*   **输入：** 最终PCB设计文件。
*   **活动：**
    1.  生成并打包所有生产所需文件，理解 **gerber file basics** 是关键。
    2.  **HILPCB 资料检查** 流程会对客户提交的资料包进行二次审核，确保完整性和可制造性。
*   **输出：** 完整的生产资料包（Gerber、钻孔文件、BOM、坐标文件等）。

## 4. 材料/层叠标准库：构建设计的“乐高积木”

标准化的叠层库是提升设计效率和可靠性的关键。团队应建立一个内部标准库，而不是每次都重新设计。以下是三个基线标准，可作为您团队的起点。

**基线一：单/双层板 (Cost-Effective Baseline)**
*   **结构：**
    *   单层板：`L1_Top_Sig` + Substrate
    *   双层板：`L1_Top_Sig` - Core - `L2_Bot_Sig`
*   **材料：** 标准 FR-4, Tg130-140, 1.6mm 板厚, 1oz (35μm) 铜厚。
*   **应用：** 成本敏感型产品、简单控制器、LED驱动、物联网终端传感器。
*   **产品链接：** [Single/Double Layer PCB](https://hilpcb.com/en/products/single-double-layer-pcb)

**基线二：四层板 (Performance Baseline)**
*   **结构 (经典)：** `L1_Top_Sig` - Prepreg - `L2_GND_Plane` - Core - `L3_PWR_Plane` - Prepreg - `L4_Bot_Sig`
*   **优势：**
    *   完整的GND平面为顶层高速信号提供了极佳的回流路径和屏蔽。
    *   专用的PWR平面提供了低阻抗的电源分配。
    *   信号层与平面层相邻，易于实现可控阻抗。
*   **应用：** 大多数消费电子、工业控制、嵌入式系统。这是性价比最高的[多层PCB](https://hilpcb.com/en/products/multilayer-pcb)方案。

**基线三：六层及以上 (High-Speed/High-Density Baseline)**
*   **结构示例 (六层)：** `L1_Top_Sig` - Prepreg - `L2_GND_Plane` - Core - `L3_Inner_Sig` - Prepreg - `L4_Inner_Sig` - Core - `L5_PWR_Plane` - Prepreg - `L6_Bot_Sig`
*   **优势：**
    *   提供更多布线层，应对高密度BGA等器件。
    *   可以实现“信号-地-信号-电源-地-信号”的对称结构，提供更好的信号完整性和EMI性能。
*   **应用：** [高速PCB设计](https://hilpcb.com/en/products/high-speed-pcb)、通信设备、服务器、复杂嵌入式系统。

## 5. 模块化布局/布线规则：特定电路的设计心法

不同的电路模块有不同的设计要求。将这些规则模块化，有助于在布局布线阶段做出正确决策。

*   **时钟与高速信号模块：**
    *   **规则1 (阻抗)：** 严格按照叠层规划的线宽进行布线，确保阻抗匹配。
    *   **规则2 (邻近性)：** 差分对应保持等长、等距、紧密耦合。
    *   **规则3 (参考平面)：** 确保高速信号下方有连续的参考平面（通常是GND），避免跨分割。
    *   **规则4 (隔离)：** 与其他信号保持足够间距（3W原则），避免串扰。

*   **电源分配模块 (PDN)：**
    *   **规则1 (路径)：** 电源路径应短而粗，从电源输入 -> 大电容 -> IC -> 去耦电容。
    *   **规则2 (平面)：** 尽可能使用电源平面而非走线来分配主电源，以降低压降和阻抗。对于[大电流PCB](https://hilpcb.com/en/products/heavy-copper-pcb)，铜厚和宽度是关键。
    *   **规则3 (去耦)：** 每个IC的电源引脚旁都应放置一个高频去耦电容 (0.1uF)，并尽可能通过最短的路径连接。

*   **模拟信号模块：**
    *   **规则1 (隔离)：** 模拟地与数字地应进行单点连接或磁珠隔离，防止数字噪声干扰模拟电路。
    *   **规则2 (分区)：** 物理上将模拟电路与数字电路、电源电路分开布局。
    *   **规则3 (屏蔽)：** 敏感的模拟信号线可以使用地线进行屏蔽（GND Guarding）。

## 6. 文档清单：确保无损交付的“护照”

一份完整、清晰的生产资料包是设计与制造之间的桥梁。任何缺失或错误都可能导致生产停滞。以下是交付给制造商（如HILPCB）的必备文件清单。

*   **Gerber Files (RS-274X or X2 Format):**
    *   `GTL`: Top Copper Layer
    *   `GBL`: Bottom Copper Layer
    *   `GTS`: Top Solder Mask
    *   `GBS`: Bottom Solder Mask
    *   `GTO`: Top Silkscreen
    *   `GBO`: Bottom Silkscreen
    *   `GTP`: Top Paste Mask
    *   `GBP`: Bottom Paste Mask
    *   `GKO`/`GM1`: Board Outline/Mechanical Layer
    *   `Gxx`: Inner Signal/Plane Layers (for multilayer boards)

*   **Drill Files (Excellon/NC Drill Format):**
    *   Plated Through-Hole (PTH) Drill File
    *   Non-Plated Through-Hole (NPTH) Drill File
    *   Drill Drawing/Map (可选，用于人工核对)

*   **Bill of Materials (BOM):**
    *   `Item Number`: 序号
    *   `Reference Designator`: 位号 (e.g., R1, C1, U1)
    *   `Manufacturer Part Number (MPN)`: 制造商料号 (最关键)
    *   `Description`: 器件描述
    *   `Quantity`: 数量
    *   `Package/Footprint`: 封装

*   **Pick and Place File (Centroid/XY File):**
    *   `Reference Designator`: 位号
    *   `X-coordinate`: X坐标
    *   `Y-coordinate`: Y坐标
    *   `Rotation`: 旋转角度
    *   `Layer`: 所在层面 (Top/Bottom)

*   **Assembly Drawings:**
    *   清晰标注元器件位号、极性（如二极管、电解电容）、安装方向（如连接器）的PDF文件。

*   **Fabrication Notes / DFM Report:**
    *   特殊工艺要求说明（如阻抗控制、盲埋孔、金手指等）。
    *   DFM检查报告，确认所有已知风险已评估。

## 7. KPI 仪表：量化设计团队的卓越表现

无法量化，就无法改进。建立一套关键绩效指标 (KPI) 仪表盘，可以帮助团队客观评估设计质量和流程效率。

*   **一次交付成功率 (First Pass Yield, FPY):**
    *   **定义：** 首次打样即功能完好、无需修改设计的项目比例。
    *   **目标：** > 90%。这是衡量整个设计体系有效性的最终指标。

*   **DRC/DFM 错误数量 (DRC/DFM Error Count):**
    *   **定义：** 在提交制造前，通过内部评审和 **HILPCB DFM** 工具发现的设计规则和可制造性问题数量。
    *   **目标：** 持续降低。追踪此数据有助于发现常见的错误模式，并针对性地进行培训。

*   **阻抗命中率 (Impedance Hit Rate):**
    *   **定义：** 对于有阻抗控制要求的PCB，实际生产出的阻抗测试条 (Coupon) 值与目标值的符合率。
    *   **目标：** > 95% 的测试值在 ±10% 的公差范围内。这反映了叠层设计和与制造商沟通的准确性。

*   **平均设计迭代次数 (Average Design Iterations):**
    *   **定义：** 每个项目从启动到最终定版所需的硬件版本数量（Spin Count）。
    *   **目标：** < 2.0。更少的迭代意味着更快的上市时间和更低的前期研发成本。

## 8. DFM/DFT 清单与 HILPCB 协同服务

这份清单是您团队的最后一道防线，应在每次交付前逐项检查。它融合了可制造性 (DFM) 和可测试性 (DFT) 的核心要点。

| 类别 | 检查项 | 描述与标准 |
| :--- | :--- | :--- |
| **通用/板规** | 板厚公差 | 是否在制造商标准公差内（通常±10%）？ |
| | 最小线宽/线距 | 是否满足制造商的工艺能力（如4/4mil, 5/5mil）？ |
| | 最小钻孔尺寸 | 机械钻孔/激光钻孔是否符合能力限制？ |
| | 焊环 (Annular Ring) | 所有PTH焊盘的焊环是否满足IPC-6012 Class 2标准？ |
| | 铜到板边距离 | 导电图形距离V-cut线或铣槽边缘是否≥0.3mm？ |
| | 阻焊开窗 | 所有SMD和THD焊盘是否都有对应的阻焊开窗？开窗是否比焊盘略大（如单边2mil）？ |
| | 阻焊桥 | QFP/QFN等细间距器件的引脚之间是否有足够的阻焊桥（≥3mil）？ |
| | 丝印清晰度 | 丝印线宽/字高是否满足最小要求（如5mil/30mil）？ |
| | 丝印上焊盘 | 是否有丝印覆盖在任何需要焊接的焊盘上？ |
| | 金手指设计 | 金手指是否按要求设计了倒角？是否远离板边？ |
| **钻孔** | 孔密度 | 局部区域的钻孔是否过于密集，可能导致板材断裂？ |
| | 孔到孔间距 | PTH到PTH，NPTH到NPTH，PTH到NPTH的中心距是否足够？ |
| | 非金属化孔 (NPTH) | NPTH孔壁是否无铜？其焊盘是否已移除？ |
| | 过孔盖油/塞油 | BGA区域的过孔是否按要求设计为盖油或塞油填平？ |
