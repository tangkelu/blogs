大家好，欢迎来到 HILPCB 测试实验室的新人培训。我是你们的培训讲师。今天，我们将深入探讨一个对 PCB 可靠性至关重要的核心概念——`cleanliness ionics basics`（离子清洁度基础）。

你可能觉得奇怪，一块看起来光洁如新的电路板，为什么在客户那里用了几个月就莫名其妙地失灵了？很多时候，罪魁祸首并非设计缺陷或元器件问题，而是那些肉眼无法看见的“隐形杀手”——离子残留物。本教程将带你完整梳理从电气测试、光学检验到最终的可靠性验证，让你明白我们“为什么测”、“如何测”以及“怎么判”，确保从 HILPCB 出厂的每一块板子都经得起时间的考验。

## “cleanliness ionics basics”要回答哪些质量问题？

首先，我们必须理解什么是离子污染物。在 PCB 制造和组装过程中，助焊剂残留、蚀刻液、电镀盐、甚至是操作员手上的汗渍，都可能在板面上留下氯离子、溴离子等活性离子。`cleanliness ionics basics` 这门学问，核心就是要回答以下几个关键的质量问题：

1.  **电路板是否足够“干净”以防止未来发生短路？**
2.  **在潮湿或高温环境下，电路板能否长期稳定工作？**
3.  **我们的清洗和焊接工艺是否得到了有效控制？**

这些问题的答案直接关系到产品的最终寿命。离子污染物在有湿气和电压的环境下，会引发灾难性的后果。

<div class="div-type-1">
<h5>离子污染的主要风险</h5>
<ul>
    <li><strong>电化学迁移 (ECM)</strong>：这是最主要的失效模式。带电的金属离子（如铜离子）会在电场作用下，从正极向负极迁移，并形成树枝状的金属沉积物（枝晶）。这些枝晶会不断生长，最终在相邻的导线之间形成短路，导致电路永久性损坏。</li>
    <li><strong>腐蚀</strong>：氯离子等腐蚀性离子会侵蚀铜走线、焊盘和元器件引脚，导致开路或接触不良。</li>
    <li><strong>漏电流增加</strong>：离子残留物会降低板材的表面绝缘电阻 (SIR)，在高阻抗或高灵敏度的模拟电路中，这会导致信号失真或功能异常。</li>
</ul>
</div>

虽然有专门的离子色谱法 (IC) 或溶剂萃取电阻率 (ROSE) 测试来直接量化离子残留，但我们接下来要讲的每一项标准测试，都与发现或预防由离子污染导致的失效息
息相关。

## 飞针与 ICT：何时使用、如何准备测试点？

在给电路板通电之前，我们必须进行电气测试，确保没有基本的制造缺陷，如短路、开路。飞针测试 (FPT) 和在线测试 (ICT) 是完成这项任务的两大主力。

#### 飞针测试 (Flying Probe Test, FPT)
想象一下几根由电脑控制的探针，像蜜蜂采蜜一样在电路板上快速移动，逐点接触焊盘和过孔，测量它们之间的连通性和电阻、电容值。这就是飞针测试。

*   **工作原理**：无需制作专门的夹具，直接使用 CAD/Gerber 数据驱动探针移动。
*   **适用场景**：非常适合[原型和小批量组装](https://hilpcb.com/en/products/prototype-assembly)，因为它的启动成本几乎为零，程序切换灵活。
*   **与清洁度的关系**：如果测试点上有绝缘的助焊剂残留，探针可能无法稳定接触，导致误报为“开路”。因此，一个干净的测试点是飞针测试准确的前提。

#### 在线测试 (In-Circuit Test, ICT)
ICT 则像一个定制的“钉床”，我们称之为 ICT 治具。这个治具上有成百上千根对准了 PCB 测试点的探针。测试时，电路板被压到治具上，所有探针同时接触，几秒钟内就能完成对所有网络和元器件的测试。

*   **工作原理**：通过高密度探针阵列一次性完成所有测试点的接触和测量。
*   **适用场景**：大批量生产的理想选择。虽然 `ict fixture introduction`（ICT治具导入）成本高昂，但其极高的测试速度可以大幅摊薄单板成本。
*   **与清洁度的关系**：与飞针类似，ICT 对测试点的清洁度要求更高。任何残留物都可能导致大面积的接触不良，造成整批次的误判和生产中断。此外，进行 ICT 前，通常需要进行 `warp flatness check`（翘曲平整度检查），因为过度弯曲的板子无法在“钉床”上实现均匀接触。

#### 如何准备测试点？
无论是 FPT 还是 ICT，可靠的测试都依赖于良好的 `test point design rules`（测试点设计规则）。

*   **尺寸**：测试点直径通常建议不小于 0.8mm，以确保探针有足够的接触面积。
*   **间距**：相邻测试点的中心距应大于 1.27mm，防止探针短路。
*   **位置**：避免将测试点放在高大元器件旁边，并均匀分布在板上，以平衡治具压力。
*   **表面处理**：测试点应进行表面处理（如沉金、OSP），且不能被阻焊油墨覆盖。在设计阶段，你可以使用 HILPCB Gerber Viewer 等工具来仔细检查测试点的位置和阻焊层开窗是否正确。

<br>

| 特性 | 飞针测试 (FPT) | 在线测试 (ICT) |
| :--- | :--- | :--- |
| **核心原理** | 2-6个可移动探针，顺序测试 | 定制化“钉床”治具，并行测试 |
| **测试速度** | 较慢（分钟级） | 极快（秒级） |
| **治具成本** | 无 | 高（数千至数万美元） |
| **适用批量** | 原型、小批量 | 中到大批量生产 |
| **程序修改** | 灵活，软件修改即可 | 困难，通常需要修改治具 |
| **缺陷定位** | 精确到具体坐标点 | 精确到具体网络 |
| **覆盖率** | 较高，但对压接器件等无能为力 | 非常高，可测电容、电阻、电感等 |

## 功能/上电测试：脚本、夹具与判据

通过了电气测试，证明电路板“硬件”上没有明显的短路或开路。但它能正常“工作”吗？这就是功能测试 (FCT) 要回答的问题。FCT 模拟电路板的最终使用环境，对其进行全面的功能验证。

<div class="div-type-3">
<h5>功能测试流程</h5>
<ol>
    <li><strong>定义测试计划</strong>：与客户一同明确需要测试哪些功能，例如：LED 是否按指令闪烁？串口能否正常收发数据？电源管理单元的输出电压是否在规格范围内？</li>
    <li><strong>设计测试夹具</strong>：在 HILPCB，我们为客户定制专用的功能测试治具。这些治具不仅提供电源和必要的输入信号，还能连接到输出端口，捕捉响应数据。</li>
    <li><strong>编写测试脚本</strong>：开发自动化软件，按预设流程向待测板 (UUT) 发送指令，并读取返回数据。例如，脚本会发送一个“点亮红色LED”的命令，然后通过一个光传感器或摄像头来验证红色LED是否真的亮了。</li>
    <li><strong>设定明确判据</strong>：每一个测试步骤都必须有清晰的“通过/失败”标准。例如，“读取到的 ADC 电压值必须在 4.95V 到 5.05V 之间”，任何超出范围的结果都将被判定为失败。</li>
</ol>
</div>

**与清洁度的关系**：离子污染在这里会扮演“性能杀手”的角色。例如，在高阻抗的模拟信号采集电路中，由离子残留引起的微小漏电流，就可能导致测量值产生严重偏差，从而在 FCT 阶段被判定为不合格。这种“软故障”是 ICT 很难发现的。

## AOI/SPI/X-Ray：光学检测如何发现缺陷？

电气测试和功能测试关注的是“电”的性能，而自动化检测则关注“形”的质量。它们利用光学和 X 射线技术，像侦探一样寻找制造过程中的物理缺陷。

*   **焊膏检测 (SPI)**：在贴装元器件之前，SPI 设备会 3D 扫描印刷在焊盘上的焊膏，检查其体积、面积、高度和偏移是否达标。这是预防焊接缺陷的第一道防线。
*   **自动光学检测 (AOI)**：在元器件贴装和回流焊之后，AOI 设备通过高清摄像头拍摄电路板图像，并与预设的良品图像进行比对。它可以快速发现诸如元件缺失、错位、极性反转、虚焊、连锡等问题。一份好的 `visual inspection checklist`（目视检查清单）是配置 AOI 程序的基础。
*   **自动 X 射线检测 (AXI)**：对于 BGA、QFN 这类焊点隐藏在器件底部的元件，AOI 无能为力。AXI 利用 X 射线的穿透能力，可以清晰地看到这些隐藏焊点的形态，检查是否存在气泡、短路或开路。

**与清洁度的关系**：虽然这些设备不直接测量离子，但它们是过程控制的得力助手。例如，AOI 可能会发现焊点周围有异常的、大面积的助焊剂残留（“白斑”），这本身就是一个强烈的视觉信号，表明清洗过程可能存在问题，离子污染的风险很高。同时，精确的光学检测也依赖于板子的物理尺寸稳定性，这与 `dimensional inspection guide`（尺寸检验指南）的要求紧密相关。

## 可靠性入门：热循环、湿热、振动、跌落

通过以上所有测试的电路板，可以确保它在出厂时是完好的。但它能在严酷的实际环境中（如汽车引擎舱、室外基站）稳定工作数年吗？可靠性测试就是为了回答这个问题，它通过模拟和加速老化的方式，提前暴露潜在的设计和制造缺陷。

`cleanliness ionics basics` 在这里显得尤为重要，因为离子污染引发的失效模式，在标准室温环境下可能潜伏数月甚至数年，但在可靠性测试中会迅速暴露。

*   **热循环测试 (Thermal Cycling)**：将电路板置于 HILPCB 的可靠性温变箱中，在极端高低温（例如 -40°C 至 +125°C）之间反复循环数百次。这会考验不同材料（如芯片、基板、焊料）因热胀冷缩系数不同而产生的应力，暴露虚焊、PCB 分层等问题。
*   **湿热测试 (Damp Heat)**：这是对离子清洁度的终极考验。我们将电路板置于高温高湿环境（如 85°C / 85% RH）中，并施加工作电压（称为 THB 测试）。这种环境为电化学迁移（ECM）创造了完美条件。如果板上有离子残留，枝晶会迅速生长，导致短路。一块通过了所有前期测试但在清洗上偷工减料的板子，在这一关几乎必败无疑。对于需要在恶劣环境中工作的 [High-Tg PCB](https://hilpcb.com/en/products/high-tg-pcb) 等产品，这项测试是强制性的。
*   **振动与跌落测试**：模拟产品在运输和使用中可能遇到的机械应力。通过将电路板固定在振动台上或进行受控跌落，我们可以检查较大元器件的焊接强度和整体结构的稳固性。

## 测试报告与追溯：记录哪些数据？

测试的最后一步，也是同样重要的一步，是记录和报告。没有数据的测试等于没有测试。一份详尽的测试报告是质量追溯和流程改进的基石。

每一块通过 HILPCB 测试流程的电路板，都会有一个唯一的序列号，并关联着其完整的测试数据档案。这份档案至少包含：

*   **基本信息**：产品型号、序列号、测试日期、操作员。
*   **测试流程记录**：经过了哪些测试站（ICT, FCT, AOI 等）。
*   **测试结果**：每个测试项的通过/失败状态。
*   **详细数据（针对失败项）**：如果是 ICT 失败，会记录是哪个网络开路/短路，测量值是多少；如果是 FCT 失败，会记录是哪个功能步骤出错，返回的错误代码是什么；如果是 AOI 失败，会附上缺陷位置的图像。

这种精细化的数据追溯能力，使我们能够在出现问题时，迅速定位到是特定批次的元器件问题，还是某台机器的参数漂移，从而实现快速有效的根本原因分析。
