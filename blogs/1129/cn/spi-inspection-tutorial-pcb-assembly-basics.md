## 1. 摘要：从装配痛点到卓越制造的价值路径

对于成长型企业而言，将一个创新的电子产品设计转化为可靠的、可量产的成品，是一条充满挑战的道路。PCB 组装（PCBA）环节的瓶颈尤为突出：不稳定的良率、模糊的质量标准、频繁的返工以及不可预测的交付周期，这些问题不仅侵蚀利润，更会延误产品上市，错失市场良机。其根本原因在于，企业缺乏一个标准化的“装配准备”框架，导致设计与制造之间存在巨大的信息鸿沟。

本白皮书旨在成为您的“**spi inspection tutorial** Assembly Readiness Whitepaper”，为您提供一套系统化的方法论。我们将深入探讨从设计到生产的每一个关键节点，重点剖析以 3D 锡膏检测（SPI）为起点的全流程质量控制体系。通过建立清晰的流程蓝图、物料与文档标准、关键工艺控制点、全面的检测矩阵以及可量化的 KPI，您可以将混乱的生产过程转变为一个可预测、可控制、可追溯的标准化流程。这不仅是关于如何进行 SPI 检测的教程，更是关于如何利用 SPI 及后续检测手段，构建一个稳健、高效且符合成本效益的 PCBA 制造体系，最终实现产品卓越与商业成功。

## 2. PCBA 流程蓝图：标准化的装配路线图

一个成功的 PCBA 项目始于一个清晰、协同的流程蓝图。它定义了每个阶段的任务、职责、输入和输出，确保从工程设计到最终发货的每个环节都无缝衔接。下表展示了一个标准的 PCBA 流程，并突出了关键的质量控制节点。

| 阶段 (Stage) | 核心任务 (Key Task) | 输入 (Input) | 输出 (Output) | 关键控制点 (Key Control Point) |
| :--- | :--- | :--- | :--- | :--- |
| **1. NPI & 工程准备** | DFM/DFA 分析、工艺路线确认、生产资料转换 | Gerber, BOM, Centroid, 装配图 | 生产工单 (Work Order), 钢网文件, 贴片程序 | DFM/DFA 报告的完整性与准确性，确保所有工程问题在生产前关闭。 |
| **2. 物料准备** | 元器件采购、PCB 裸板入库、IQC (来料检验) | 供应商清单, BOM, PCB 规格书 | 检验合格的物料套料 (Kitting) | IQC 覆盖率，元器件MSD（湿敏等级）管控，遵循 **esd handling guide**。 |
| **3. SMT 表面贴装** | **锡膏印刷 (Solder Paste Printing)** | 钢网, 锡膏, PCB | 印刷锡膏的 PCB | 锡膏的型号、粘度、回温时间；印刷参数（速度、压力、脱模）。 |
| | **锡膏检测 (SPI Inspection)** | 印刷锡膏的 PCB | SPI 检测报告, 合格/不合格板 | SPI 程序准确性，检测标准（体积、高度、面积、偏移）的设定。 |
| | **元件贴装 (Pick and Place)** | 贴片程序, Centroid, Feeder | 已贴装元件的 PCB | 贴装精度与速度，**smt component polarity** 的正确性，抛料率控制。 |
| | **回流焊接 (Reflow Soldering)** | 回流焊温度曲线 (Profile) | 完成焊接的 SMT 板 | 温度曲线的精确性（预热、恒温、回流、冷却），N₂ 环境控制。 |
| **4. THT 通孔插装** | 元件成型、手动/自动插件 | THT 元器件, 装配图 | 已插件的 PCB | 元件插入深度、方向、极性；防错料机制。 |
| | **波峰焊/选择焊** | 焊接温度曲线, 助焊剂 | 完成焊接的 THT/混装板 | 焊接温度、传送带速度、喷锡高度、助焊剂涂覆均匀性。 |
| **5. 清洗与后处理** | **Cleaning after soldering** (如需要) | 清洗剂, DI 水 | 洁净的 PCBA | 清洗工艺的选择（水洗/免洗），离子污染物残留测试。 |
| **6. 检测与测试** | AOI, X-Ray, ICT, FCT | 测试治具, 测试程序 | 测试报告, 合格品 | 检测矩阵的覆盖率，测试程序的准确性，缺陷数据的收集与分析。 |
| **7. 终检与包装** | FQC (最终质量控制), 烧录, 三防漆涂覆 | 客户规格书, 包装材料 | 待发货成品 | FQC 检验标准，包装的 ESD 防护与物理防护。 |

## 3. 物料与文档标准：卓越制造的基石

“Garbage in, garbage out.” 这句古老的工程谚语在 PCBA 领域尤为适用。不规范、不完整、不清晰的生产资料是导致延误、错误和成本超支的首要原因。建立严格的物料与文档标准，是实现装配准备的第一步。

### 3.1. 物料清单 (BOM - Bill of Materials)
一个“制造友好”的 BOM 应当是明确且无歧义的。
- **核心字段**: 必须包含位号 (Reference Designator)、制造商零件编号 (MPN)、制造商 (Manufacturer)、描述 (Description)、封装 (Footprint/Package) 和数量 (Quantity)。
- **替代料**: 为关键或长交期物料提供 1-2 个经过验证的替代选项，以增强供应链的韧性。
- **DNI/DNP**: 明确标注“Do Not Install/Populate”的元器件，避免混淆。
- **格式**: 推荐使用 Excel 或 CSV 格式，便于自动化处理。您可以使用 HILPCB BOM Viewer 工具在下单前检查您的 BOM 格式与完整性。

### 3.2. 坐标文件 (Centroid File)
Centroid 文件（也称 Pick-and-Place 文件）是贴片机编程的直接输入，其准确性至关重要。
- **核心字段**: 必须包含位号 (Ref Des)、X/Y 坐标、贴装层 (Top/Bottom) 和旋转角度 (Rotation)。
- **原点**: 明确定义坐标系的原点，通常是 PCB 的左下角或板中心。
- **单位**: 统一使用公制单位（毫米）。
- **旋转**: 明确定义 0 度旋转方向，并确保所有元件的旋转角度都遵循 IPC-7351 标准。

### 3.3. 装配图与 Gerber 文件
装配图是人工操作和质量检验的重要参考。
- **清晰的位号**: 确保丝印层的位号清晰可辨，不被焊盘或过孔遮挡。
- **极性标记**: 对于二极管、电解电容、IC 等有极性要求的元件，必须有清晰的标记（如“+”号、点、斜角）。这是避免 **smt component polarity** 错误的关键。
- **特殊说明**: 在图纸上标注任何特殊工艺要求，如压接件、点胶、特定焊接标准等。
- **Gerber 文件**: 提供完整的 Gerber RS-274X 或 ODB++ 文件集，包括所有铜层、阻焊层、丝印层和锡膏层。使用 HILPCB Gerber Viewer 可以帮助您在投产前预览和验证文件。

## 4. SMT/THT/混装能力与关键控制

掌握核心组装工艺的控制点，是确保质量稳定性的关键。HILPCB 的自动化产线通过精密的设备和严格的流程控制，确保每个环节都达到最高标准。

### 4.1. SMT (表面贴装技术)
SMT 是现代电子组装的主流，其质量始于锡膏印刷。
- **锡膏印刷**: 这是缺陷的源头，超过 60% 的 SMT 缺陷与锡膏印刷有关。我们通过激光切割钢网确保开口精度，并使用全自动印刷机控制印刷速度、压力和脱模，确保锡膏成型一致。
- **SPI (Solder Paste Inspection)**: 这是我们质量控制的第一道“防火墙”。作为本 **spi inspection tutorial** 的核心，SPI 使用 3D 光学技术精确测量每个焊盘上锡膏的体积、面积、高度和位置。任何超出预设阈值（如体积在 50%-150% 之间）的印刷缺陷都会被立即标记，操作员会及时干预，调整印刷参数或清洁钢网，从而防止缺陷流入下一环节。
- **元件贴装**: 高速贴片机能够处理小至 01005 的元件，并通过视觉系统校准位置和方向。我们严格遵循 **esd handling guide**，所有操作环境和人员均采取静电防护措施，保护敏感元器件。
- **回流焊接**: 我们为每款产品定制优化的回流焊温度曲线，通过多温区（通常为 8-12 个）氮气回流焊炉，精确控制预热、浸润、回流和冷却的斜率与时间，确保形成可靠的焊点，同时避免对元件造成热损伤。

### 4.2. THT (通孔插装技术)
尽管 SMT 普及，但 THT 在连接器、大功率器件等领域仍不可或缺。
- **选择性波峰焊**: 对于高密度的混装板，我们采用选择性波峰焊，通过可编程的喷嘴，仅对 THT 焊点进行焊接，避免对邻近的 SMT 元件造成二次加热。
- **手工焊接**: 对于异形件或小批量产品，我们拥有经过 IPC-A-610 认证的焊接技术员团队，确保手工焊接的质量与一致性。

### 4.3. 混装技术
混装板（同时包含 SMT 和 THT 元件）的工艺流程更为复杂。通常先完成所有 SMT 工序，再进行 THT 插装和焊接。关键在于保护已贴装的 SMT 元件在 THT 焊接过程中不受影响，这需要精密的载具（carrier）设计和精确的焊接工艺控制。

## 5. 检测矩阵：构建全方位的质量防火墙

单一的检测方法无法覆盖所有潜在缺陷。一个强大的 PCBA 制造商会部署一个多层次的检测矩阵，在流程的不同阶段捕获不同类型的缺陷。

| 检测技术 | 部署阶段 | 主要目的 | 检测缺陷类型 | 优势/局限性 |
| :--- | :--- | :--- | :--- | :--- |
| **SPI** | 锡膏印刷后 | **过程控制**：在缺陷源头发现并纠正问题。 | 锡膏体积/高度/面积/偏移、桥连、缺失。 | **优势**: 成本效益极高，防止批量缺陷。 **局限**: 仅检测锡膏。 |
| **AOI (自动光学检测)** | 贴片后/回流焊后 | **缺陷检测**：快速检查元件和焊点外观。 | 元件缺失、偏移、错件、极性反、虚焊、桥连、锡珠。 | **优势**: 速度快，适合大批量生产。 **局限**: 无法检测隐藏焊点（如 BGA）。 |
| **X-Ray (X射线检测)** | 回流焊后 | **无损透视检测**：检查不可见焊点。 | BGA/QFN 虚焊、气泡、桥连、枕头效应 (HIP)，通孔填充不足。 | **优势**: 唯一能有效检测 BGA 的方法。 **局限**: 成本高，速度慢，通常用于抽检或关键器件全检。 |
| **ICT (在线测试)** | 组装完成后 | **电气性能验证**：检查元器件值和电路连接。 | 开路、短路、元器件值错误、功能性问题。 | **优势**: 测试覆盖率高，速度快。 **局限**: 需要设计测试点 (Test Point)，测试治具成本高。 |
| **FCT (功能测试)** | 组装完成后 | **最终功能验证**：模拟最终使用环境，验证产品功能。 | 软件、硬件协同工作是否正常，产品性能是否达标。 | **优势**: 最接近真实使用场景的测试。 **局限**: 测试方案开发复杂，通常由客户提供。 |

这个矩阵构成了我们质量保证体系的核心。通过在流程早期（如 SPI）发现问题，**HILPCB 的质量团队** 能够利用 **MES (制造执行系统)** 的数据，快速进行根源分析和过程调整，而不是等到最终测试才发现问题，从而大大降低了返工成本和时间。

## 6. KPI 与持续改进：用数据驱动卓越

没有衡量，就没有改进。建立一套清晰的关键绩效指标 (KPI) 是将质量管理从被动反应转变为主动预防的关键。

- **首次通过率 (First Pass Yield, FPY)**: 这是衡量生产过程稳定性的核心指标。FPY = (一次性通过所有测试的板卡数量 / 投入生产的总板卡数量) × 100%。一个强大的 SPI 流程能显著提升 FPY，因为它在早期就消除了大量潜在的焊接缺陷。
- **缺陷百万机会数 (Defects Per Million Opportunities, DPMO)**: 一个更精细的质量度量，用于衡量过程控制的能力。它考虑了每块板上可能出现缺陷的机会点数量。
- **准时交货率 (On-Time Delivery, OTD)**: 衡量制造商的可靠性和计划能力。
- **客户退货率 (Customer Return Rate)**: 最终衡量产品在客户端表现的指标。
- **根本原因分析 (RCA) 响应时间**: 从发现问题到完成根本原因分析并实施纠正措施所需的时间。HILPCB 的 **MES 系统** 实现了全面的 **可追溯性**，可以追溯到每个元件的批次、每块板的生产设备和操作员，这为快速 RCA 提供了强大的数据支持。

通过持续监控这些 KPI，管理团队可以识别流程中的薄弱环节，并采取针对性的改进措施，形成一个“计划-执行-检查-行动”(PDCA) 的持续改进循环。

## 7. DFM/DFA 清单：从源头设计卓越

最好的质量控制始于设计阶段。一个考虑了可制造性 (DFM) 和可装配性 (DFA) 的设计，可以从根本上避免许多生产问题。以下是一个包含超过 35 个检查点的 **design for assembly checklist**，供您的设计团队参考。

| 类别 | 检查项 | 建议/标准 |
| :--- | :--- | :--- |
| **PCB 布局** | 1. 元件间距 | 遵循 IPC 标准，为返修和测试留出足够空间。 |
| | 2. 元件到板边距离 | 通常建议 ≥ 3mm，避免在 V-cut 或邮票孔附近放置元件。 |
| | 3. 高/重元件布局 | 均匀分布，避免集中在一侧导致应力或焊接问题。 |
| | 4. 测试点 (Test Points) | 为关键信号设计易于接触的测试点，直径 ≥ 1mm。 |
| | 5. 方向一致性 | 尽可能使相似元件（如 IC, 二极管）方向一致，减少贴装错误。 |
| **元件选择** | 6. 封装标准化 | 优先选用标准、通用的封装，如 0402, 0603。 |
| | 7. 避免异形元件 | 尽可能避免需要手工处理的异形元件。 |
| | 8. 元件可获得性 | 在设计阶段确认关键元件的供货周期和替代选项。 |
| | 9. 封装与焊盘匹配 | 确保元件库中的封装尺寸与 PCB 焊盘设计精确匹配。 |
| **拼板 (Panelization)** | 10. 拼板方式 | 根据板形和元件布局选择 V-cut 或邮票孔。 |
| | 11. 工艺边 (Tooling Strips) | 添加宽度 ≥ 5mm 的工艺边，用于轨道传送和支撑。 |
| | 12. 基准点 (Fiducials) | 每块单板和拼板工艺边上至少放置 3 个全局基准点。 |
| | 13. 邮票孔设计 | 孔径和间距合理，确保易于分板且边缘光滑。 |
| **焊盘与掩膜** | 14. 阻焊桥 (Solder Mask Dam) | 在细间距引脚之间保留阻焊桥，防止桥连，建议宽度 ≥ 4mil。 |
| | 15. 钢网开口 (Paste Mask) | 通常与焊盘 1:1，但可根据元件类型微调（如 BGA 焊盘适当缩小）。 |
| | 16. 过孔处理 | 避免将过孔直接设计在焊盘上（Via-in-Pad），除非采用填孔工艺。 |
| | 17. BGA 焊盘设计 | 采用 NSMD (非阻焊定义) 设计，以获得更好的焊接可靠性。 |
| | 18. 热焊盘 (Thermal Relief) | 在连接到大铜皮的焊盘上使用热焊盘，避免焊接时散热过快。 |
| **丝印 (Silkscreen)** | 19. 位号清晰度 | 丝印清晰，不被元件或焊盘遮挡。 |
| | 20. 极性标记 | 清晰标注二极管、电容、IC 的第一引脚等。 |
| | 21. 丝印线宽 | 建议线宽 ≥ 6mil，以保证印刷质量。 |
| | 22. 避免上焊盘 | 确保丝印油墨不会印刷到任何焊盘上。 |
| **其他** | 23. ESD 防护设计 | 在电路设计中考虑 ESD 防护器件。 |
| | 24. 清洗要求 | 设计时考虑是否需要 **cleaning after soldering**，避免使用不兼容的元件。 |
| | 25. 散热设计 | 为大功率器件设计足够的散热铜皮或散热孔。 |
| | 26. 避免底部元件过高 | 如果需要过波峰焊，板底 SMT 元件高度需受限。 |
| | 27. 连接器位置 | 考虑线缆连接和操作空间。 |
| | 28. 安装孔 | 确保安装孔尺寸、位置正确，并与外壳匹配。 |
| | 29. 射频/高速信号 | 遵循阻抗控制和信号完整性设计规则。可使用 HILPCB 阻抗计算器。 |
| | 30. 柔性板 (Flex PCB) | 设计加强筋 (stiffener) 和弯折区域，避免在弯折区放置元件。 |
| | 31. 元件高度限制 | 确认产品外壳内部的高度限制。 |
| | 32. 避免阴影效应 | 在回流焊和波峰焊中，高元件不应遮挡矮元件。 |
| | 33. BGA 逃逸布线 | 为 BGA 设计合理的逃逸布线通道。 |
| | 34. 晶振布局 | 靠近主控芯片，并有地平面保护。 |
| | 35. 文件命名规范 | 使用清晰、一致的文件命名规则。 |
