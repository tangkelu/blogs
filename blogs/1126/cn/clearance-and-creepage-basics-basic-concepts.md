## 引言

在印刷电路板（PCB）设计领域，尤其是在处理高压或电源应用时，“电气间隙（Clearance）”和“爬电距离（Creepage）”是两个至关重要的安全概念。它们直接关系到产品的安全性、可靠性以及能否通过安规认证。忽视这些基础规则可能导致设备短路、电弧、甚至火灾。本文将通过 20 个常见问题，深入浅出地解析 **clearance and creepage basics**，帮助您从入门到精通，确保您的设计既功能强大又绝对安全。

### **1. 什么是 PCB 设计中的“电气间隙 (Clearance)”？**

-   **场景**：您正在设计一个电源适配器，需要确保 230V 交流输入端的火线（L）和零线（N）之间不会发生空气击穿。
-   **指标/示例**：电气间隙是指两个导电体之间在空气中测量的最短直线距离（line-of-sight）。它用于防止高压导体之间因空气电离而产生电弧（Arcing）。例如，根据 IEC 62368-1 标准，对于一个工作在 250V RMS、污染等级 2 的环境，基本绝缘所需的最小电气间隙约为 2.0 mm。
-   **解决方案**：在您的 EDA（电子设计自动化）软件中，为高压网络设置一个明确的“Clearance”设计规则。例如，在 Altium Designer 中，您可以为“HV_Net_Class”设置一个 2.0 mm 的间隙约束。
-   **预防 Checklist**：
    -   [ ] 在项目初期确定最大工作电压和瞬态过压。
    -   [ ] 查阅适用的安全标准（如 IEC 62368-1）以获取最小间隙值。
    -   [ ] 在 EDA 软件中将这些值设置为强制性设计规则。

### **2. 什么是 PCB 设计中的“爬电距离 (Creepage)”？**

-   **场景**：您的设备需要在潮湿或多尘的环境中运行。您担心绝缘板表面会因积聚的污染物而导电，导致高压和低压电路之间发生短路。
-   **指标/示例**：爬电距离是指两个导电体之间沿绝缘材料表面的最短路径。它用于防止因表面污染和湿气导致的“电痕”（Tracking）现象，即在绝缘材料表面形成导电通路。对于同样的 250V RMS、污染等级 2、材料组 IIIa（CTI 175-399）的应用，IEC 62368-1 要求最小爬电距离为 2.5 mm。
-   **解决方案**：确保高压导体之间在 PCB 表面的距离足够。如果布局空间有限，可以通过在导体之间切割一个槽（Slot）或铣一个沟（Groove）来人为地延长表面路径。
-   **预防 Checklist**：
    -   [ ] 评估产品的最终使用环境，确定污染等级。
    -   [ ] 了解所用 PCB 基板的 CTI（比较性漏电起痕指数）值。
    -   [ ] 在布局时优先保证爬电距离，必要时使用开槽等技巧。

### **3. 为什么电气间隙和爬电距离如此重要？**

-   **场景**：一个初级设计师为了节省空间，将高压走线和信号走线靠得非常近，导致产品在安规测试中失败，或在用户使用中发生故障。
-   **指标/示例**：这两个参数是产品电气安全的核心。**不合规的间隙**可能导致瞬时电弧，损坏元件；**不足的爬电距离**则可能在长期使用中因灰尘和湿气积累形成永久性短路，引发火灾。全球几乎所有的电子产品安全认证（如 UL、CE、TÜV）都强制要求满足特定的间隙和爬电距离标准。
-   **解决方案**：将安全设计置于最高优先级。在设计初期就根据目标市场的安规标准，计算并规划好所需的距离。利用 EDA 工具的 DRC（设计规则检查）功能来自动捕捉违规。
-   **预防 Checklist**：
    -   [ ] 将安规要求纳入产品需求文档。
    -   [ ] 对设计团队进行定期的安全标准培训。
    -   [ ] 在设计审查（Design Review）中将间隙和爬电距离作为关键检查项。

### **4. 电气间隙和爬电距离的根本区别是什么？**

-   **场景**：一位工程师混淆了这两个概念，错误地认为只要两个焊盘在 PCB 表面相距 3mm，就万事大吉，却忽略了它们在空气中通过元件本体的距离可能只有 1mm。
-   **指标/示例**：
    -   **Clearance**：穿过**空气**的最短路径。想象一根拉紧的线。
    -   **Creepage**：沿着**绝缘表面**的最短路径。想象一只蚂蚁在板上爬行的最短路线，它需要绕过障碍物（如槽）。
    -   **示例**：两个相邻的变压器引脚，它们在空气中的直线距离是 Clearance。而电流需要沿着 PCB 表面从一个引脚“爬”到另一个引脚的路径长度是 Creepage。
-   **解决方案**：始终同时考虑这两个维度。对于表面贴装器件（SMD），Clearance 和 Creepage 通常是相同的。但对于通孔器件（THT），元件本体上方的空气间隙可能远小于 PCB 表面的引脚间距。
-   **预防 Checklist**：
    -   [ ] 绘制草图，清晰地标出空气路径和表面路径。
    -   [ ] 特别注意那些跨接高低压隔离带的元件（如光耦、变压器）。
    -   [ ] 使用 3D 模型检查元件之间的空间距离。

### **5. 哪些行业标准定义了电气间隙和爬电距离的规则？**

-   **场景**：您正在开发一款销往全球的医疗设备，需要知道应该遵循哪个标准来设计电源部分。
-   **指标/示例**：不同的产品和地区有不同的标准，最常见的有：
    -   **IEC 62368-1**：音视频、信息和通信技术设备的最新安全标准，正在取代 IEC 60950-1 和 IEC 60065。
    -   **IEC 60601-1**：医疗电气设备的安全标准，要求通常更严格。
    -   **IPC-2221B**：PCB 设计的通用标准，提供了基于电压的通用间隙指导。
    -   **UL 840**：定义了绝缘配合的要求。
-   **解决方案**：在项目启动时，与合规团队或认证机构确认产品需要遵循的主要安全标准。然后，获取该标准的最新版本，并依据其中的表格和公式来确定具体要求。
-   **预防 Checklist**：
    -   [ ] 确定产品类别（ITE、医疗、工业等）。
    -   [ ] 确定目标销售市场（北美、欧洲、中国等）。
    -   [ ] 将标准名称和版本号记录在设计文档中。

### **6. 电压如何影响电气间隙和爬电距离的要求？**

-   **场景**：您不确定是应该根据 120V 的交流输入电压，还是根据其峰值电压（约 170V）来计算间隙。
-   **指标/示例**：要求与电压直接相关，且通常是非线性的。您需要考虑：
    -   **工作电压（Working Voltage）**：通常是 RMS（均方根）值或直流电压。
    -   **峰值电压（Peak Voltage）**：对于交流电，峰值电压约为 RMS 值的 1.414 倍。
    -   **瞬态过电压（Transient Overvoltage）**：由雷击、开关操作等引起的瞬时高压。标准通常会根据设备的安装类别（Overvoltage Category I, II, III, IV）来规定需要承受的瞬态电压。
-   **解决方案**：查阅标准时，要仔细核对表格是基于 RMS 电压、峰值电压还是瞬态电压。通常，**Clearance 主要由峰值电压和瞬态过电压决定**，而 **Creepage 主要由 RMS 工作电压决定**。
-   **预防 Checklist**：
    -   [ ] 分析电路，确定所有导体间的最大 RMS 和峰值电压。
    -   [ ] 根据产品应用确定其过电压类别。
    -   [ ] 使用最坏情况下的电压值来查表计算。

### **7. 什么是 CTI (比较性漏电起痕指数)，它如何影响爬电距离？**

-   **场景**：为了降低成本，您选择了一款标准的 [FR-4 PCB](https://hilpcb.com/en/products/fr4-pcb) 板材，但后来发现它无法满足高压部分的爬电距离要求。
-   **指标/示例**：CTI (Comparative Tracking Index) 是衡量绝缘材料在电压和表面污染作用下抵抗电痕形成能力的指标。CTI 值越高，材料的抗电痕能力越强，在相同电压下所需的爬电距离就越小。
    -   **材料组 I**：CTI ≥ 600
    -   **材料组 II**：400 ≤ CTI < 600
    -   **材料组 IIIa**：175 ≤ CTI < 400 (标准 FR-4 通常属于此类)
    -   **材料组 IIIb**：100 ≤ CTI < 175
-   **解决方案**：在设计高压产品时，应主动选择高 CTI 的板材（如 CTI 600 的 FR-4）。虽然成本稍高，但可以显著减小 PCB 尺寸，或避免使用开槽等复杂工艺。请与您的 PCB 供应商（如 HILPCB）确认他们提供的板材 CTI 等级。
-   **预防 Checklist**：
    -   [ ] 在物料清单（BOM）中明确指定 PCB 基板的 CTI 要求。
    -   [ ] 如果无法使用高 CTI 材料，则在设计中为更大的爬电距离预留空间。
    -   [ ] 咨询 **HILPCB** 获取关于[高Tg PCB](https://hilpcb.com/en/products/high-tg-pcb) 或其他特殊材料的建议。

### **8. 运行环境（污染等级）如何影响这些距离？**

-   **场景**：您设计的产品将用于工业车间，那里充满了金属粉尘和油污，而您的设计是按照办公室环境（污染等级2）的标准进行的。
-   **指标/示例**：污染等级（Pollution Degree）描述了设备运行环境中干燥污染和湿气凝结的程度。
    -   **PD 1**：无污染或仅有干燥、非导电性污染。通常指密封在盒子里的电路。
    -   **PD 2**：仅有非导电性污染，但偶尔会有因凝露导致的暂时导电。办公室、家庭环境。
    -   **PD 3**：有导电性污染，或干燥的非导电性污染因凝露而变得导电。工业环境。
    -   **PD 4**：持续的导电性，如雨雪天气。户外设备。
-   **解决方案**：污染等级越高，所需的爬电距离越大。必须根据产品的最终用途来选择正确的污染等级。如果无法确定，应选择更严格的等级以确保安全。
-   **预防 Checklist**：
    -   [ ] 在产品规格书中定义预期的运行环境。
    -   [ ] 如果产品可能用于多种环境，按最恶劣的情况设计。
    -   [ ] 考虑使用外壳、密封或敷形涂层来降低板级的污染等级。

### **9. 敷形涂层（Conformal Coating）对电气间隙和爬电距离有何影响？**

-   **场景**：您的 PCB 布局非常紧凑，无法通过开槽来满足爬电距离要求。您正在寻找替代方案。
-   **指标/示例**：敷形涂层是在 PCB 表面覆盖一层薄薄的绝缘聚合物膜（如丙烯酸、聚氨酯、硅胶）。这层涂层可以有效地将电路与环境中的湿气和污染物隔离开来。
-   **解决方案**：正确应用的敷形涂层可以**显著降低爬电距离的要求**，因为它将污染等级从 PD 2 或 PD 3 降低到了 PD 1。然而，它**通常不影响电气间隙**的要求，因为电弧仍然可以击穿涂层和空气。使用涂层前，必须确保其厚度、均匀性和附着力都符合标准。
-   **预防 Checklist**：
    -   [ ] 确认所选涂层材料的绝缘强度和认证情况。
    -   [ ] 在制造文件中明确规定涂层的类型、厚度和覆盖区域。
    -   [ ] 确保连接器、测试点等不需要涂覆的区域被正确遮蔽。

### **10. 功能绝缘、基本绝缘和加强绝缘之间有什么区别？**

-   **场景**：您正在设计一个 USB 充电器，需要确保交流电侧和 5V USB 输出侧之间有足够的安全隔离。
-   **指标/示例**：这是描述绝缘等级和安全保护水平的术语：
    -   **功能绝缘 (Functional Insulation)**：仅为电路正常工作所必需的绝缘，不提供防电击保护。例如，同一电路中两条信号线之间的绝缘。
    -   **基本绝缘 (Basic Insulation)**：提供基本防电击保护的单层绝缘。
    -   **加强绝缘 (Reinforced Insulation)**：相当于两层独立基本绝缘的单层绝缘系统，提供高级别的防电击保护。
    -   **双重绝缘 (Double Insulation)**：由基本绝缘和补充绝缘（Supplementary）组成。
-   **解决方案**：对于连接到电网的设备，其高压侧（Primary）和用户可接触的安全低压侧（Secondary）之间，通常要求**加强绝缘**或**双重绝缘**。这意味着它们之间的电气间隙和爬电距离要求会比基本绝缘严格得多（通常是两倍左右）。
-   **预防 Checklist**：
    -   [ ] 绘制系统框图，识别出不同的安全边界（如 Primary-Secondary）。
    -   [ ] 为跨越这些边界的每个路径确定所需的绝缘等级。
    -   [ ] 在布局中用隔离带（Isolation Barrier）物理地隔开这些区域。

### **11. 如何在我的 EDA 软件（如 Altium, KiCad）中设置电气间隙和爬电距离规则？**

-   **场景**：您希望在布线过程中，软件能自动提示并阻止您违反安全间距。
-   **指标/示例**：现代 EDA 工具都有强大的 DRC（设计规则检查）引擎。
    -   **电气间隙 (Clearance)**：可以直接在“Clearance Constraint”规则中设置。您可以为不同的网络类（如“HV”和“LV”）设置不同的规则。例如，设置“HV”网络类内部间隙为 2mm，而“HV”与“LV”之间的间隙为 5mm。
    -   **爬电距离 (Creepage)**：EDA 软件通常不能直接计算爬电距离。您需要通过设置一个足够大的 Clearance 规则来间接满足爬电距离要求。如果使用了开槽，DRC 工具可能无法识别增加的距离，需要手动检查。
-   **解决方案**：
    1.  创建网络类，将不同电压等级的走线分配进去。
    2.  在规则矩阵中，为这些网络类之间设置最小电气间隙。
    3.  对于需要满足爬电距离的地方，将 Clearance 规则设置为目标爬电距离值。
    4.  对于开槽设计，手动测量并验证。
-   **预防 Checklist**：
    -   [ ] 在开始布局前就设置好所有关键的 DRC 规则。
    -   [ ] 定期运行 DRC 检查，而不是等到设计结束。
    -   [ ] 利用 HILPCB 的免费 Gerber 查看器 在提交前进行最终检查。

### **12. 有哪些常见的 PCB 布局技巧可以用来增加爬电距离？**

-   **场景**：您的设计空间非常宝贵，两个高压焊盘之间的直线距离无法满足爬电距离要求。
-   **指标/示例**：当物理空间受限时，可以通过改变表面拓扑来延长路径。
    -   **开槽/铣削 (Slotting/Milling)**：在两个导体之间铣掉一条 PCB 基板，强制电流必须绕着槽的边缘走，从而显著增加爬电距离。一个 1mm 宽的槽可以轻松将 2mm 的表面距离延长到 4mm 以上。
    -   **V型槽/刻痕 (V-Groove)**：在导体之间刻一个 V 形的沟，也能起到延长路径的作用。
    -   **绝缘隔板 (Barrier)**：在 PCB 上安装一个垂直的绝缘片，增加爬电距离。
-   **解决方案**：开槽是最常用且有效的方法。在设计中，用一个非金属化的槽（通常在机械层或板框层绘制）放置在需要隔离的导体之间。确保槽的宽度和位置在制造文件中清晰标注。
-   **预防 Checklist**：
    -   [ ] 确保槽的宽度至少为 1mm，以保证制造可靠性。
    -   [ ] 槽的边缘距离任何导体至少要有 0.5mm，防止加工误差。
    -   [ ] 在制造说明中明确指出哪些是绝缘槽。

### **13. 过孔（Vias）和焊盘会违反间隙规则吗？如何修复？**

-   **场景**：您的 DRC 报告了大量“Pad-to-Via”或“Pad-to-Trace”的间隙违规，尤其是在密集的 BGA 或 QFP 封装周围。
-   **指标/示例**：是的，过孔和焊盘都是导电体，它们与其他所有导电元素（走线、覆铜、其他过孔/焊盘）之间的距离都必须满足 Clearance 规则。一个常见的错误是，焊盘本身的尺寸或阻焊层开窗过大，导致其边缘与邻近的走线距离过近。
-   **解决方案**：
    1.  **减小焊盘/过孔尺寸**：在不影响焊接质量和载流量的前提下，适当减小焊盘或过孔的直径。
    2.  **调整走线路径**：将违规的走线绕开，保持安全距离。
    3.  **使用泪滴（Teardrops）**：为焊盘和过孔添加泪滴可以平滑连接，但要注意泪滴本身不能引入新的违规。
    4.  **检查阻焊层膨胀（Solder Mask Expansion）**：确保阻焊层开窗的膨胀值不会过大，导致裸露的铜皮区域超出预期。
-   **预防 Checklist**：
    -   [ ] 在创建或使用 `netlist and footprints` 库时，严格审查封装的焊盘尺寸和间距。
    -   [ ] 为不同类型的过孔（例如，`via types explained` 中的微孔、盲孔）设置特定的间隙规则。
    -   [ ] 在高密度区域使用更精细的设计规则。

### **14. 如何在同一块 PCB 上管理高压电路和低压电路的间隙？**

-   **场景**：您的设计中既有 230V 的交流电源部分，也有 3.3V 的微控制器（MCU）和传感器部分。
-   **指标/示例**：关键在于**隔离（Isolation）**。高压区（Primary）和安全低压区（Secondary）之间必须有明确的物理隔离带。这个隔离带的宽度由加强绝缘的电气间隙和爬电距离要求决定，通常在 4mm 到 8mm 之间。
-   **解决方案**：
    1.  **物理分区**：在布局时，将 PCB 明确划分为高压区和低压区，中间留出足够宽的“无人区”。
    2.  **禁止布线**：在这个隔离带内，任何层的走线、覆铜、过孔都是绝对禁止的。
    3.  **跨接元件**：所有需要跨越隔离带的信号都必须通过隔离元件，如光电耦合器、变压器或安规电容。
    4.  **开槽**：在隔离带下方开一个或多个绝缘槽是增强隔离性能的最佳实践。
-   **预防 Checklist**：
    -   [ ] 在布局初期就画出隔离边界。
    -   [ ] 确保所有团队成员都了解并遵守隔离区的布线规则。
    -   [ ] 仔细检查跨接元件的封装，确保其自身的引脚间距满足隔离要求。

### **15. 丝印层（Silkscreen）会影响电气间隙或爬电距离吗？**

-   **场景**：为了美观，您将元件的丝印标识直接印在了焊盘上。
-   **指标/示例**：通常情况下，标准的非导电丝印油墨不会影响电气性能。然而，存在两个风险：
    1.  **导电油墨**：在某些特殊应用中（如电容触摸按键），可能会使用导电油墨。如果误用，后果严重。
    2.  **覆盖焊盘**：丝印覆盖在即将焊接的焊盘上会严重影响焊接质量，可能导致虚焊。
-   **解决方案**：最佳实践是遵循良好的 `silkscreen design rules`。确保丝印层与阻焊层开窗（即裸露的焊盘）之间有足够的间隙（通常为 0.1-0.2mm）。这样可以防止油墨污染焊盘。
-   **预防 Checklist**：
    -   [ ] 在 EDA 工具中设置丝印到裸铜的间隙规则。
    -   [ ] 确认使用的丝印油墨是绝缘的。
    -   [ ] 在最终生成 Gerber 文件前，仔细检查丝印层，确保没有覆盖任何焊盘。

### **16. PCB 层叠结构（Stackup）和层命名如何影响高压设计？**

-   **场景**：您不确定是将高压走线放在外层还是内层更好。
-   **指标/示例**：层叠结构对高压设计有显著影响。
    -   **外层 vs. 内层**：外层（Top/Bottom）直接暴露在环境中，受污染等级影响大，爬电距离要求严格。内层（Internal）被 FR-4 介质完全包裹，环境相当于污染等级 1，因此相同电压下内层的间隙要求通常比外层宽松得多。
    -   **介质厚度**：相邻内层之间的介质（Prepreg）厚度决定了它们之间的垂直绝缘能力。
-   **解决方案**：如果空间允许，将高压走线布置在内层可以利用更小的间隙，从而节省空间。但要注意，内层散热较差。一个清晰的 `pcb stack layer naming` 约定（如 `L1_Signal`, `L2_GND`, `L3_HV_Power`）有助于团队理解设计意图。
-   **预防 Checklist**：
    -   [ ] 与 **HILPCB** 这样的制造商沟通，确定一个既满足电气性能又易于制造的层叠方案。
    -   [ ] 在设计文档中详细说明层叠结构，包括每层的功能、铜厚和介质厚度。
    -   [ ] 优先将高压网络布置在内层，但要进行热仿真以评估散热。

### **17. 像 HILPCB 这样的制造商如何验证我的电气间隙和爬电距离设计？**

-   **场景**：您已经完成了设计，并将其发送给 HILPCB 进行生产。您想知道他们会如何检查您的文件。
-   **指标/示例**：制造商会进行 DFM（Design for Manufacturability）检查。对于间隙，他们会使用专业的 CAM 软件自动分析您的 Gerber 文件，检查所有铜皮元素之间的距离是否满足他们工厂的最小制造能力（例如，最小线隙/线宽为 3/3 mil）。
-   **解决方案**：HILPCB 的 DFM 检查主要关注**制造可行性**，而非**安规符合性**。他们会确保能造出您设计的板子，但不会判断您的 5mm 间隙是否满足 IEC 62368-1 的要求。因此，安规符合性的责任最终在设计者。不过，您可以向 HILPCB 的工程师寻求 **DFM 咨询**，他们可以根据经验对您的设计提出潜在风险的建议。
-   **预防 Checklist**：
    -   [ ] 在下单前，了解并遵循 HILPCB 的制造能力参数。
    -   [ ] 不要完全依赖制造商的 DFM 来发现安规问题。
    -   [ ] 在提交文件时，附上清晰的制造说明，特别是关于绝缘槽等特殊要求。

### **18. 需要提供哪些文档来向工厂清晰传达我的间隙和爬电距离要求？**

-   **场景**：您的设计中有一个关键的绝缘槽，您担心工厂会忽略它。
-   **指标/示例**：清晰的文档是成功制造的关键。
    -   **Gerber 文件**：确保包含一个独立的机械层（Mechanical Layer）或板框层（Keep-Out Layer），在其中精确绘制出所有需要铣削的槽或区域。
    -   **制造图纸 (Fabrication Drawing)**：在图纸中明确标注这些槽的尺寸、公差，并用文字说明其用途是“用于高压隔离”。
    -   **Readme.txt 文件**：在提交的压缩包中附上一个文本文档，再次强调特殊要求，如“请注意，GML 层上的轮廓为绝缘槽，必须铣空，非金属化”。
-   **解决方案**：过度沟通胜过沟通不足。使用多种方式（图纸、层文件、文本说明）来传达关键信息，确保制造工程师不会误解您的设计意图。
-   **预防 Checklist**：
    -   [ ] 创建一个专门的制造说明文档模板。
    -   [ ] 在图纸上使用清晰的标注和剖面图。
    -   [ ] 在下单后，与 HILPCB 的客服或工程师再次确认他们已理解所有特殊要求。

### **19. 如果我的设计因间隙问题未能通过制造商的 DFM 检查，会发生什么？**

-   **场景**：您提交订单后，收到了来自 HILPCB 的一封邮件，标题是“工程问题 (Engineering Question - EQ)”。
-   **指标/示例**：当 DFM 软件发现您的设计违反了工厂的最小制造能力时（例如，您设计的间隙为 3 mil，而工厂最低只能做 4 mil），订单会被暂停。工程师会向您发送一封 EQ 邮件，其中会附上截图，清晰地指出问题所在。
-   **解决方案**：
    1.  **仔细阅读 EQ**：理解问题的具体位置和原因。
    2.  **评估影响**：判断是否可以接受工厂的建议（例如，将间隙扩大到 4 mil）。
    3.  **修改设计**：如果需要，修改您的源文件，重新生成 Gerber，并回复 EQ 邮件附上新文件。
    4.  **书面确认**：如果同意工厂的修改建议，直接在邮件中书面授权他们进行调整。
-   **预防 Checklist**：
    -   [ ] 在设计前就从制造商网站获取其最新的 DFM 规则。
    -   [ ] 在 EDA 工具中设置与制造商能力相匹配的 DRC 规则。
    -   [ ] 保持沟通渠道畅通，以便及时响应 EQ。

### **20. 设计重铜板（Heavy Copper PCB）时，对电气间隙有什么特殊考虑？**

-   **场景**：您正在设计一个大功率逆变器，使用了 4oz 的铜厚。
-   **指标/示例**：重铜板的制造工艺与标准 PCB 不同。由于铜层很厚，蚀刻过程中的侧蚀（Undercut）会更严重，导致走线的横截面呈梯形。这会影响到实际的最小间隙。例如，一个设计为 8 mil 的间隙，在 4oz 铜上最终可能只剩下 6 mil。
-   **解决方案**：为重铜板设计更大的间隙余量。经验法则是，铜厚每增加 1oz，最小间隙应比标准板增加 1-2 mil。务必与您的制造商（如 HILPCB，他们有丰富的[重铜 PCB](https://hilpcb.com/en/products/heavy-copper-pcb) 制造经验）确认他们的具体制造能力和设计建议。
-   **预防 Checklist**：
    -   [ ] 在项目初期就与制造商沟通重铜设计规则。
    -   [ ] 在 DRC 中为重铜层设置更宽松的间隙规则。
    -   [ ] 考虑使用圆形焊盘和 45 度走线，以减少蚀刻过程中的尖角问题。

### **21. 元件封装库（Footprints）如何影响电气间隙？**

-   **场景**：您从网上下载了一个高压 MOSFET 的封装，但在布局后发现其引脚间距不满足安全要求。
-   **指标/示例**：`netlist and footprints` 是设计的基石。一个不准确或不合规的封装库是灾难的根源。例如，一个 SOT-23 封装，其引脚中心距为 0.95mm，如果焊盘设计得过宽，焊盘边缘的实际间隙可能只有 0.4mm，这对于高压应用是完全不够的。
-   **解决方案**：
    1.  **验证封装**：切勿盲目相信下载的封装。对照元件的数据手册（Datasheet）仔细核对每一个尺寸。
    2.  **创建自己的库**：建立并维护一个经过验证的、符合 IPC-7351 标准的公司内部元件库是最佳实践。
    3.  **使用 Courtyard**：在封装中定义一个“Courtyard”区域（元件的边界外加一个安全距离），确保元件之间不会靠得太近。
-   **预防 Checklist**：
    -   [ ] 对所有用于高压电路的元件封装进行 100% 审查。
    -   [ ] 检查数据手册中关于 PCB 布局的建议部分。
    -   [ ] 确保焊盘尺寸和阻焊开窗的设置合理。

### **22. 柔性电路板（Flex PCB）的电气间隙和爬电距离有何不同？**

-   **场景**：您正在设计一款需要反复弯折的[柔性 PCB](https://hilpcb.com/en/products/flex-pcb)，上面承载着高压信号。
-   **指标/示例**：柔性板的基材（通常是聚酰亚胺，PI）和覆盖膜（Coverlay）的绝缘性能与刚性板的 FR-4 和阻焊油墨类似。因此，在静态（不弯折）状态下，其间隙和爬电距离的计算方法是相同的。
-   **解决方案**：主要的挑战在于**动态弯曲区域**。反复的弯曲和拉伸会对铜箔和覆盖膜造成机械应力，可能导致微小裂纹，从而降低绝缘性能。因此，在设计动态弯曲区域时，应采取以下措施：
    1.  **增加间隙**：在弯曲区域的走线间隙应比设计标准留出更多余量。
    2.  **避免过孔**：绝对禁止在动态弯曲区域放置过孔。
    3.  **弧形走线**：使用平滑的弧形走线，避免 90 度转角。
-   **预防 Checklist**：
    -   [ ] 明确标识出 FPC 的静态区和动态弯曲区。
    -   [ ] 在动态弯曲区采用更保守的设计规则。
    -   [ ] 与 HILPCB 讨论关于补强板（Stiffener）和覆盖膜的选项，以增强机械强度。
