## 引言：从混乱到清晰的 PCB 设计文档

在快节奏的电子产品开发中，一份清晰、完整且准确的 PCB 设计文档是连接设计与制造的唯一桥梁。然而，许多团队常常因为文档疏漏、设计细节模糊或沟通不畅而导致生产延误、成本超支甚至产品失效。本篇 **pcb documentation tutorial** 旨在通过一个全面的 FAQ 清单，系统性地解决从叠层设计到最终交付的各个环节中的常见问题。

本文将深入探讨 20 个核心问题，涵盖叠层阻抗、布局布线、电源 EMC 以及审核交付四大领域。每个问题都将遵循“**问题→现象→根因→解决方案→预防清单**”的结构，为您提供可立即应用的实践指导。此外，我们还准备了详尽的 DFM 检查清单和分级学习路径，帮助您的团队建立标准化的设计与文档流程，将潜在风险扼杀在摇篮之中。

## 第一部分：叠层与阻抗控制 FAQ

#### 1. 问题：为什么我设计的 50Ω 阻抗，实测出来偏差超过 10%？

-   **现象**：使用 TDR (时域反射计) 测试阻抗 Coupon，发现特性阻抗为 56Ω 或 44Ω，远超 ±5% 的行业标准。
-   **根因**：
    1.  **叠层信息不准**：设计时使用的介质厚度 (H)、介电常数 (Dk) 与板厂实际使用的材料参数不符。
    2.  **铜厚误差**：线路的最终铜厚 (T) 因电镀工艺而增加，未在仿真时考虑。
    3.  **线宽补偿**：板厂在蚀刻过程中会对线宽 (W) 进行补偿，但补偿量与设计预期不符。
-   **解决方案**：
    1.  在设计初期就与板厂沟通，获取他们常用板材的精确 Dk/Df 值和层压结构。
    2.  使用专业的阻抗计算工具（如 Polar Si9000）或 EDA 软件内置的工具进行仿真。
    3.  在制造文件 (Fabrication Drawing) 中明确要求阻抗控制，并指定公差范围（如 50Ω ±5%），要求提供 TDR 测试报告。
-   **预防清单**：
    -   [ ] 设计前与 **HILPCB** 等专业制造商沟通，获取推荐的 `pcb stackup tutorial` 和材料参数。
    -   [ ] 在设计文件中包含详细的叠层结构图，标明每层厚度、材料、铜厚。
    -   [ ] 要求板厂在生产前进行叠层结构的二次确认 (Pre-production confirmation)。
    -   [ ] 要求对关键信号线制作阻抗 Coupon 并随板交付测试报告。

#### 2. 问题：高速信号跨越参考平面分割处会发生什么？

-   **现象**：信号眼图严重恶化，出现大量振铃和过冲；系统 EMI 测试失败，辐射超标。
-   **根因**：当信号线跨越其参考平面（GND 或 PWR）的分割区域时，其回流路径被迫绕一个大圈，形成一个巨大的电流环路。这个环路相当于一个天线，会向外辐射能量，同时也会引入严重的电感，破坏信号完整性。
-   **解决方案**：
    1.  **重新布线**：调整走线，避免跨越分割。这是最佳方案。
    2.  **使用缝合电容 (Stitching Capacitor)**：在分割处附近放置一个 0.1uF 或 1nF 的小电容，为高频回流电流提供一个低阻抗的跨接路径。
    3.  **使用桥接**：如果分割是不可避免的，可以在分割处用一小段铜皮将两个平面连接起来，信号线从桥上通过。
-   **预防清单**：
    -   [ ] 在布局规划阶段，就确定好高速信号的关键路径，确保其下方有连续的参考平面。
    -   [ ] 对 `mixed signal pcb layout` 进行严格分区，但尽量保持地平面的完整性。
    -   [ ] 使用 EDA 工具的信号完整性分析功能，检查回流路径是否连续。

#### 3. 问题：为什么我的 10Gbps+ 信号在长距离传输后性能很差？

-   **现象**：高速 SerDes 链路误码率 (BER) 高，眼图张开度不足，时序抖动 (Jitter) 严重。
-   **根因**：**玻纤效应 (Fiber Weave Effect)**。PCB 的 FR-4 基材是由玻璃纤维布和环氧树脂构成的。玻璃纤维 (Dk ≈ 6) 和树脂 (Dk ≈ 3) 的介电常数不同。如果差分对中的一根线大部分时间在玻纤束上，另一根在树脂区，两者感受到的有效 Dk 不同，导致传输延迟不一致，产生差分到共模的转换和时序偏移。
-   **解决方案**：
    1.  **Z-axis 旋转布线**：将走线相对于玻纤编织方向旋转一个小角度（如 10-15 度），让差分对交替地经过玻纤和树脂区，以平均 Dk。
    2.  **锯齿形或波浪形布线**：强制走线路径变化，同样为了平均 Dk。
    3.  **使用平坦型玻纤布**：选择 1067, 1086 等开纤更均匀的玻纤布，或机械平坦化的材料。
    4.  **使用高速材料**：选择 Dk 值更低且更均匀的高速板材，如 Megtron 6 或 Rogers 系列。
-   **预防清单**-
    -   [ ] 对于 >5Gbps 的信号，在设计规范中明确板材和玻纤布类型。
    -   [ ] 在布线时，有意识地让高速差分对走斜线，避免与板边平行或垂直。
    -   [ ] 与板厂确认是否可以指定布线相对于玻纤的方向。

#### 4. 问题：对于一个简单的 4 层板，最佳的叠层结构是什么？

-   **现象**：设计的 4 层板 EMC 性能不佳，容易受到外部干扰，或自身辐射严重。
-   **根因**：不合理的叠层结构导致参考平面不佳、信号与电源层耦合。例如，GND-SIG-SIG-PWR 的结构，两层信号之间没有地平面隔离，容易产生串扰。
-   **解决方案**：
    -   **最佳推荐**：**SIG - GND - PWR - SIG**。
        -   顶层和底层走信号线。
        -   中间两层为完整的地平面和电源平面。
        -   **优点**：信号层与参考平面紧密耦合，提供了良好的阻抗控制和回流路径。GND 和 PWR 平面紧邻，形成了一个天然的板级去耦电容，降低了 PDN 阻抗。
    -   **次优选择**：GND - SIG - SIG - PWR。
        -   **缺点**：内层信号线之间没有隔离，需要特别注意串扰问题。外层是平面，屏蔽效果好，但不利于元器件布局和调试。
-   **预防清单**：
    -   [ ] 除非有特殊理由，否则 4 层板设计默认采用 SIG-GND-PWR-SIG 结构。
    -   [ ] 在 `pcb stackup tutorial` 文档中明确定义此结构，并解释其设计意图。

## 第二部分：布局与布线 FAQ

#### 5. 问题：如何正确处理模数混合电路的布局与接地？

-   **现象**：模拟信号（如音频、传感器读数）中混入了数字噪声（时钟谐波），导致 ADC 采样精度下降或运放输出失真。
-   **根因**：数字电路的高速开关电流通过共用地平面或电源平面，在上面产生了压降（地弹），这些噪声电压耦合到了敏感的模拟回路上。
-   **解决方案**：
    1.  **物理分区**：在 PCB 上明确划分出数字区、模拟区和电源区。所有模拟元件放在模拟区，数字元件放在数字区。
    2.  **单点接地 (Star Ground)**：将模拟地 (AGND) 和数字地 (DGND) 在 PCB 上分离布线，最后只在一点连接起来，通常是在 ADC/DAC 芯片下方或电源入口处。
    3.  **注意回流路径**：即使地平面是统一的，也要确保数字信号的回流路径不会流经模拟区域下方。
-   **预防清单**：
    -   [ ] 在 `mixed signal pcb layout` 阶段，首先进行模块化布局规划。
    -   [ ] 确保模拟信号线下方是模拟地，数字信号线下方是数字地。
    -   [ ] 避免任何信号线（尤其是高速数字线）跨越模数地之间的分割间隙。

#### 6. 问题：差分对布线的基本原则是什么？

-   **现象**：差分信号的眼图很差，远端串扰 (FEXT) 和近端串扰 (NEXT) 严重，无法通过一致性测试。
-   **根因**：违反了 `differential pair basics` 的核心原则，导致差分阻抗不连续和模式转换。
-   **解决方案**：
    1.  **等长 (Matched Length)**：差分对的两条线 (P/N) 长度必须严格匹配。对于高速信号（如 PCIe Gen3），容差可能在 5mil 以内。使用 EDA 工具的等长调整功能（通常是“蛇形线”）。
    2.  **等距 (Constant Spacing)**：P/N 线之间的间距必须在整个路径上保持恒定，以维持稳定的差分阻抗（如 100Ω）。
    3.  **同层布线 (Same Layer Routing)**：尽量在同一层内完成布线，避免因换层导致过孔引入的阻抗不连续。
    4.  **对称布线 (Symmetry)**：进入和离开焊盘、过孔或弯折时，路径应保持对称。
-   **预防清单**：
    -   [ ] 在 EDA 工具中将差分对定义为“Differential Pair Class”。
    -   [ ] 设置严格的布线规则：线宽、间距、最大长度差。
    -   [ ] 避免在差分对附近布线高噪声信号（如时钟、开关电源）。

#### 7. 问题：为什么我的差分对在弯折处性能下降？

-   **现象**：TDR 测试显示，每当差分对拐弯时，阻抗就会出现一个明显的下降尖峰。
-   **根因**：在直角或锐角弯折处，外侧走线的路径比内侧长，导致局部相位失配。同时，弯折处外侧的线宽实际上变窄了，而内侧的耦合更强，导致阻抗变化。
-   **解决方案**：
    1.  **使用弧形走线**：这是最佳方案。采用 45° 或圆弧走线可以最大程度地减少阻抗突变。
    2.  **补偿间距**：如果必须使用锐角弯折，可以在弯折区域适当增加 P/N 线之间的间距，以补偿增强的耦合效应。
    3.  **添加补偿拐角**：在内侧走线的拐角处添加一个小的凸起，以补偿其较短的路径长度。
-   **预防清单**：
    -   [ ] 在设计规则中，将差分对的走线模式设置为“圆弧”或“45度角”。
    -   [ ] 避免在差分对路径上使用 90 度直角弯。

#### 8. 问题：什么时候需要考虑过孔残桩 (Via Stub) 的影响？

-   **现象**：在非常高的频率下（通常 > 10Gbps），信号通过过孔后衰减严重，S21 参数在高频段出现急剧下降的“陷波”。
-   **根因**：当信号从某一层通过过孔换到另一层时，过孔未被使用的那部分（从信号层到 PCB 另一端的延伸部分）就形成了“残桩”。这个残桩就像一个小的天线，会在特定频率（1/4 波长）产生谐振，严重影响信号质量。
-   **解决方案**：
    1.  **背钻 (Back-drilling)**：在 PCB 制造完成后，从另一面将过孔多余的铜柱钻掉。这是最常用且成本效益高的方法。
    2.  **使用盲/埋孔 (Blind/Buried Vias)**：只在需要的层之间打孔，从根本上消除残桩。成本较高。
    3.  **优化布线**：尽量减少高速信号的换层次数。
-   **预防清单**：
    -   [ ] 对于速率超过 10Gbps 的信号，评估是否需要背钻。
    -   [ ] 在制造文件中明确标注需要背钻的过孔、背钻深度和公差。

#### 9. 问题：如何为换层信号提供良好的回流路径？

-   **现象**：信号换层后，边沿变缓，出现振铃，甚至导致逻辑错误。
-   **根因**：信号电流总是以环路形式流动。在同一层布线时，回流电流会紧邻信号线在其下方的参考平面上流动。当信号通过过孔换层时，如果新的参考平面与旧的参考平面（例如，从 GND 参考层换到 PWR 参考层）没有低阻抗的连接，回流电流就需要绕远路，形成一个大的环路。
-   **解决方案**：
    -   **放置接地缝合过孔 (Stitching Via)**：在信号过孔旁边紧挨着放置一个或多个接地过孔。这些过孔将不同层的地平面连接起来，为回流电流提供了一条直接、短小的路径。
-   **预防清单**：
    -   [ ] 养成习惯：每当一个高速信号换层时，就在其旁边放置一个接地过孔。
    -   [ ] 对于差分对换层，应在每条信号线旁边都放置一个接地过孔，保持对称性。

#### 10. 问题：保护环 (Guard Trace) 是不是总能隔离噪声？

-   **现象**：为敏感的模拟信号添加了保护环，但噪声问题依然存在，甚至更糟。
-   **根因**：`guard trace design` 如果不正确，会弊大于利。
    1.  **保护环悬空**：如果保护环没有良好接地，它会像天线一样收集噪声，再耦合到被保护的信号线上。
    2.  **单端接地**：如果保护环很长，只在一端接地，它本身可能成为一个谐振结构。
    3.  **接地方式不当**：保护环应该连接到安静的模拟地，而不是嘈杂的数字地。
-   **解决方案**：
    1.  **多点接地**：保护环必须通过一系列过孔密集地连接到下面的参考地平面。建议每隔一段距离（如 1/20 波长）就放置一个接地过孔。
    2.  **正确间距**：保护环与被保护信号线的间距，以及与噪声源的间距，应遵循经验法则，如 2W 或 3W（W 为信号线宽）。
    3.  **谨慎使用**：在大多数情况下，一个完整的地平面提供的屏蔽效果比不正确的保护环要好得多。只有在没有连续地平面或需要隔离特定强噪声源时才考虑使用。
-   **预防清单**：
    -   [ ] 在使用保护环之前，首先确认是否可以通过优化布局和地平面来解决问题。
    -   [ ] 如果使用，确保其通过密集的过孔连接到正确的地平面。

## 第三部分：电源与 EMC FAQ

#### 11. 问题：去耦电容应该如何放置？

-   **现象**：芯片工作不稳定，在高速运行时复位或出错，电源轨上充满了高频噪声。
-   **根因**：去耦电容的放置位置和连接方式不当，导致其等效串联电感 (ESL) 过大，无法有效地为芯片提供瞬时电流和滤除高频噪声。
-   **解决方案**：
    1.  **紧靠引脚**：将电容尽可能地靠近芯片的电源 (VCC) 和接地 (GND) 引脚。
    2.  **路径最短**：从电源平面到电容焊盘，再到芯片引脚的路径应尽可能短而宽。最佳实践是让电流先流经电容焊盘，再进入芯片引脚。
    3.  **过孔优化**：使用多个过孔连接到电源和地平面，以降低电感。将过孔直接打在焊盘上（Via-in-Pad）是最佳选择，但成本较高。
    4.  **容值搭配**：使用多个不同容值的电容组合（如 1uF, 0.1uF, 10nF, 1nF），以在更宽的频带上提供低阻抗。小容值电容（物理尺寸也小，ESL 更低）应最靠近引脚。
-   **预防清单**：
    -   [ ] 遵循芯片数据手册中关于去耦电容布局的建议。
    -   [ ] 在原理图阶段就规划好每组电源引脚的去耦电容组合。
    -   [ ] 在布局时，优先处理去耦电容的放置，而不是将其塞在最后。

#### 12. 问题：如何理解并最小化电流环路面积？

-   **现象**：产品无法通过辐射发射 (RE) 测试，在特定频率上出现尖峰。
-   **根因**：根据法拉第电磁感应定律，变化的磁场会产生电场，反之亦然。一个载有高频电流的环路会产生一个变化的磁场，从而向外辐射电磁能量。辐射强度与环路面积、电流大小和频率的平方成正比。因此，最小化高频电流环路面积是 EMC 设计的核心。
-   **解决方案**：
    1.  **紧邻参考平面**：确保所有高速信号线下方都有一个连续的参考平面。回流电流会自然地选择路径最短、环路面积最小的路径，即紧贴在信号线下方。
    2.  **优化去耦**：去耦电容的环路（从电容到芯片引脚再返回）是主要的高频电流环路，必须最小化。
    3.  **检查 I/O 接口**：连接器处的信号和地线应相邻，避免信号线从连接器一端进入，地线从另一端进入，形成大环路。
-   **预防清单**：
    -   [ ] 在设计评审中，专门检查关键信号和电源的环路面积。
    -   [ ] 使用 3D 场求解器工具可以直观地看到电流密度和回流路径。

#### 13. 问题：我应该分割地平面吗？

-   **现象**：在 `mixed signal pcb layout` 中，对于是否要将模拟地 (AGND) 和数字地 (DGND) 分割开来感到困惑。
-   **根因**：分割地的初衷是为了隔离数字噪声，防止其污染敏感的模拟电路。然而，不恰当的分割会带来更严重的问题，比如信号跨分割导致的回流路径中断。
-   **解决方案**：
    -   **优先选择统一地平面**：对于大多数现代设计，一个设计良好的统一地平面是更好的选择。通过严格的物理分区（将模拟和数字电路放在 PCB 的不同区域），可以引导回流路径，使其互不干扰。
    -   **何时考虑分割**：
        1.  **医疗或高精度仪器**：对噪声极其敏感，需要物理隔离。
        2.  **安规要求**：如高压和低压电路之间的隔离。
        3.  **特定芯片要求**：一些 ADC/DAC 厂商会明确建议分割地。
    -   **如果必须分割**：确保分割的“护城河”两端通过一个“桥”连接起来，所有跨越模数区域的信号线都必须从这个桥上通过，以保证回流路径的连续性。
-   **预防清单**：
    -   [ ] 在决定分割地之前，问自己：“我是否能通过分区和布局来解决问题？”
    -   [ ] 如果分割，仔细检查每一条跨越分割的走线。

<div class="div-style-4">
    <h4>常见踩坑提醒：过度依赖默认 DRC 规则</h4>
    <p>许多工程师认为只要 DRC (Design Rule Check) 检查通过，设计就没有问题。这是一个巨大的误区。默认的 `drc rule template pcb` 只能检查基本的物理间距和连接性，无法发现信号完整性、电源完整性或 EMC 问题，如不连续的回流路径、过大的环路面积或阻抗失配。成功的 PCB 设计需要将 DRC 规则与深入的设计知识和评审流程相结合。</p>
</div>

#### 14. 问题：20H 规则是什么，还有用吗？

-   **现象**：PCB 板边辐射严重，导致 EMC 测试失败。
-   **根因**：电源平面和地平面在 PCB 边缘会产生边缘场效应，向外辐射噪声。
-   **解决方案**：
    -   **20H 规则**：要求电源平面比相邻的地平面内缩一段距离，该距离等于两个平面间介质厚度 (H) 的 20 倍。这样可以使大部分边缘场被限制在两个平面之间，减少向外的辐射。
    -   **是否还有用**：在多层板设计中，如果电源和地平面紧密耦合，20H 规则的效果会减弱，但仍然是一种有益的实践，尤其是在板边有敏感电路或连接器时。
-   **预防清单**：
    -   [ ] 在布局时，为电源层设置一个比板框小的内缩区域。
    -   [ ] 更好的方法是确保 PCB 的最外层是地平面，并用一圈接地过孔墙将整个板卡屏蔽起来。

#### 15. 问题：如何设计一个低阻抗的电源分配网络 (PDN)？

-   **现象**：FPGA 或 CPU 在高负载下工作时，核心电压跌落 (IR Drop) 严重，导致系统崩溃。
-   **根因**：PDN 的阻抗过高，无法满足芯片在纳秒级时间内对大电流的需求。PDN 阻抗包括了从 VRM 到芯片的整个路径的电阻和电感。
-   **解决方案**：
    1.  **使用电源/地平面**：用完整的平面代替走线来传输电源，可以极大地降低直流电阻和电感。
    2.  **分层去耦**：在板上、封装上和芯片上都布置去耦电容。板级去耦电容负责提供中低频电流，封装和片上电容负责高频部分。
    3.  **目标阻抗法**：根据芯片的电流需求和允许的电压纹波，计算出 PDN 在目标频率范围内的最大允许阻抗（Z_target = ΔV / ΔI），然后通过放置合适的电容组合来满足这个目标。
-   **预防清单**：
    -   [ ] 对于高性能芯片，进行 PDN 仿真分析。
    -   [ ] 遵循芯片厂商的电源设计指南，它们通常提供了详细的去耦策略。

## 第四部分：审核与交付 FAQ

#### 16. 问题：我的 DRC 检查全部通过，为什么生产出来的板子还是有问题？

-   **现象**：板卡出现短路、开路或性能不佳，但 EDA 软件的 DRC 报告是干净的。
-   **根因**：DRC 规则集不完整。标准的 DRC 只能检查物理间距（线到线、线到焊盘等），但无法检查更高级的问题，如：
    -   **酸角 (Acid Traps)**：锐角连接处可能导致过度蚀刻。
    -   **孤岛铜 (Copper Slivers)**：细小的、未连接的铜皮可能在生产中脱落并造成短路。
    -   **阻焊层开窗问题**：BGA 焊盘的阻焊层定义不当（NSMD vs. SMD）。
    -   **丝印上焊盘**：丝印覆盖在焊盘上，影响焊接。
-   **解决方案**：
    1.  **建立全面的 `drc rule template pcb`**：与板厂合作，根据他们的制造能力（DFM - Design for Manufacturability）来定制 DRC 规则。
    2.  **进行 DFF/DFA 检查**：除了 DRC，还要进行可制造性 (DFF) 和可装配性 (DFA) 检查。
    3.  **人工评审**：Checklist 驱动的人工视觉检查是不可或缺的最后一道防线。
-   **预防清单**：
    -   [ ] 定期更新公司的 DRC 规则文件。
    -   [ ] 将 DFM 检查作为设计发布的强制步骤。

#### 17. 问题：Gerber、BOM 和坐标文件之间最常见的冲突是什么？

-   **现象**：SMT 工厂报告元件贴反、贴错位置或使用了错误的元件。
-   **根因**：这三个核心交付文件的数据来源不统一，或在手动修改过程中出现错误。
    -   **元件位号不匹配**：BOM 中的 R10 在坐标文件中不存在。
    -   **封装不匹配**：原理图和 BOM 中是 0402 封装，但 PCB 库用的是 0603。
    -   **元件旋转角度错误**：坐标文件中的元件旋转角度与 PCB 库中的 0 度方向定义不一致。
-   **解决方案**：
    1.  **单一数据源**：所有制造文件都应从最终确认的 PCB 设计文件中自动生成。避免手动编辑导出的文件。
    2.  **标准化元件库**：建立统一、经过验证的元件库，确保原理图符号、PCB 封装和 3D 模型一一对应。
    3.  **使用 ODB++ 或 IPC-2581 格式**：这些现代格式将所有设计和制造信息打包在一个文件中，从根本上消除了文件不一致的问题。
-   **预防清单**：
    -   [ ] 在发布前，使用 Gerber 查看器将 Gerber、钻孔文件和坐标文件叠加检查。
    -   [ ] 随机抽查几个元件，核对 BOM、坐标和 PCB 上的信息是否完全一致。

#### 18. 问题：如何有效地管理 PCB 设计的变更？

-   **现象**：团队成员对当前哪个是最新版本感到困惑，导致生产了过时的设计。
-   **根因**：缺乏正式的工程变更控制 (ECO) 流程和版本管理系统。
-   **解决方案**：
    1.  **版本控制系统**：使用 Git 或 SVN 等工具来管理原理图和 PCB 文件。
    2.  **ECO 流程**：建立一个正式的工程变更单流程。任何修改都必须有记录，包括变更原因、内容、审批人和生效日期。
    3.  **清晰的版本命名**：在文件名和 PCB 板上丝印上使用清晰的版本号（如 `Project_V1.2`）。避免使用 `_final`, `_new` 等模糊的后缀。
-   **预防清单**：
    -   [ ] 对所有设计文件进行版本控制。
    -   [ ] 每次发布制造文件时，都附带一份变更说明。

#### 19. 问题：我的制造说明 (Fab Notes) 中经常遗漏哪些关键信息？

-   **现象**：提交 Gerber 后，板厂会发来一长串问题，反复沟通导致生产周期延长。
-   **根因**：制造说明过于简单，没有提供足够的信息让板厂准确无误地生产。
-   **解决方案**：
    -   **创建一个标准模板**，至少包含以下信息：
        1.  **板材要求**：FR-4、Tg 值、Dk/Df（如果关键）。
        2.  **叠层结构图**：详细的层压结构，包括每层铜厚、介质厚度和材料类型。
        3.  **阻抗控制要求**：哪些线需要做阻抗控制，目标值和公差，以及测试方法。
        4.  **外层表面处理**：ENIG（化金）、HASL（喷锡）等。
        5.  **阻焊层和丝印颜色**。
        6.  **钻孔和外形公差**。
        7.  **特殊工艺**：如背钻、盲/埋孔、金手指、盘中孔 (VIPPO) 等。
-   **预防清单**：
    -   [ ] 将制造说明模板化，并作为 `pcb documentation tutorial` 的一部分进行团队培训。
    -   [ ] 每次下单前，都对照模板检查一遍。

#### 20. 问题：为什么板厂有时会擅自修改我的叠层设计？

-   **现象**：收到的板子虽然通过了电气测试，但高速性能不达标。追溯发现，板厂为了使用库存中现有的芯板 (Core) 和半固化片 (PP)，修改了原始的叠层结构。
-   **根因**：设计文件中没有明确禁止修改，或者没有强调阻抗控制的重要性。板厂出于成本和效率考虑，会使用最接近的材料组合来“凑”出总厚度。
-   **解决方案**：
    1.  **在制造说明中明确**：“此叠层结构为阻抗控制设计，未经设计方书面许可，严禁修改。如有任何材料替换，必须提供等效仿真报告并获得批准。”
    2.  **要求阻抗 Coupon 和 TDR 测试报告**：这是验证板厂是否遵循了阻抗设计的最终手段。
    3.  **与可靠的供应商合作**：像 **HILPCB** 这样的专业制造商，会严格遵循客户的设计文件，并在生产前对叠层进行工程确认 (EQ)，确保最终产品的性能符合预期。
-   **预防清单**：
    -   [ ] 在 Fab Notes 中加入“禁止修改叠层”的条款。
    -   [ ] 将 TDR 测试作为验收的必要条件。
