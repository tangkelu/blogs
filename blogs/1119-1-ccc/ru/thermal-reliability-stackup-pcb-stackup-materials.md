## 1. Резюме: сценарии, вызовы и выгоды

**Сценарий:** С быстрым развитием 5G/6G, ИИ‑вычислений, автомобильной электроники и HPC проектирование PCB сталкивается с беспрецедентной плотностью мощности и скоростью передачи данных. Это делает **thermal reliability stackup (тепломеханически надежный stackup)** не «опцией», а фундаментом успеха изделия. Неправильный stackup может отказать при тепловых циклах, механических напряжениях или длительной работе даже при корректной электрической функциональности.

**Вызовы:**
*   **Парадокс выбора материалов:** высокоскоростные материалы (низкий Dk/Df) часто имеют менее благоприятные тепломеханические свойства (например, более высокий CTE), тогда как материалы с высокой теплопроводностью не всегда подходят для ВЧ‑применений. Как сбалансировать производительность, стоимость и надежность?
*   **Многофизическая связность:** stackup — это связанная задача электрических, тепловых и механических характеристик. Контроль импеданса влияет на SI, а несоответствие CTE напрямую приводит к расслоениям, трещинам via и др.
*   **Разрыв между проектом и производством:** проектные предпосылки (например, целевой Dk) могут сместиться при ламинации из‑за течения смолы и шероховатости меди, что вызывает импеданс‑рассогласование и деградацию.

**Выгоды:** Эта белая книга дает командам системного и аппаратного проектирования стандартизированную **stackup strategy**. Используя дерево выбора материалов, библиотеку шаблонов и процесс верификации, команда сможет:
*   **Повысить надежность:** предотвращать полевые отказы из‑за CTE‑несоответствий и hot‑spots на уровне источника.
*   **Сократить цикл разработки:** применять проверенные материалы и шаблоны, уменьшая число итераций.
*   **Оптимизировать TCO:** балансировать начальную стоимость и долговременную надежность, снижая риски дорогостоящих отзывов и ремонтов.

<div class="div-type-1">
    <p><strong>Ключевая идея:</strong> Успешный stackup — это соблюдение SI/PI, дополненное точной материаловедческой и конструктивной инженерией, чтобы обеспечить тепломеханическую стабильность PCB на всем жизненном цикле.</p>

## 2. Дерево выбора материалов: от метрик к выбору

Выбор правильных материалов — первый шаг к `thermal reliability stackup`. Таблица ниже дает «решающую рамку» по ключевым метрикам.

| Ключевая метрика (Key Metric) | Рекомендуемая серия (Recommended Material) | Типичные применения (Typical Application) | Ограничения/заметки (Key Limitations/Considerations) |
| :--- | :--- | :--- | :--- |
| **Чувствительно к стоимости, универсальные применения** | **FR‑4 среднего Tg (Tg: 130–150°C)** | Consumer, IoT, маломощные модули | Низкая стойкость к тепловым напряжениям, не подходит для бессвинцовой пайки/высокой мощности. Средние Dk/Df. |
| **Повышенная тепловая стабильность** | **FR‑4 высокого Tg (Tg: ≥170°C)** | Серверы, промконтроль, automotive, силовые платы | Дороже среднего Tg FR‑4. Dk (~4.2–4.7) может ограничивать ультравысокие скорости. |
| **Высокоскоростные сигналы (>10 Gbps)** | **Low Dk/Df (например Rogers RO4000, Megtron 6, Isola I‑Speed)** | Backplane, оптика, RF | Высокая цена, строгие требования процесса (например plasma desmear). CTE часто выше FR‑4 — нужен аккуратный hybrid stack. |
| **Экстремальное тепловое управление** | **Высокотеплопроводные (IMS/MCPCB, CEM‑3)** | LED, DC/DC, приводы | Обычно 1–2 слоя, сложная трассировка ограничена. Теплопроводность диэлектрика 2–10 W/m·K >> FR‑4 (~0.25). |
| **Гибкость/высокая надежность** | **Полиимид (PI)** | FPC, rigid‑flex, aerospace | Высокая гигроскопичность, требуются строгие bake‑процессы. Дорого, размерная стабильность ниже жестких плат. |
| **Согласование низкого CTE** | **Низко‑CTE материалы (например Isola 370HR, Nelco N4000‑13)** | BGA/CSP substrate, HDI | Нацелено на CTE кристалла (~3–7 ppm/°C) для снижения усталости пайки. Дороже. |

## 3. Библиотека шаблонов stackup: стандартизированная точка старта

На основе тысяч серийных проектов выделены классические шаблоны stackup, оптимизированные по симметрии, непрерывности импеданса и тепловому балансу.

| Кол-во слоев | Тип структуры | Рекомендуемый stackup (сверху вниз) | Преимущества и применения |
| :--- | :--- | :--- | :--- |
| **4 слоя** | SIG/GND/PWR/SIG | L1: Signal, L2: Ground, L3: Power, L4: Signal | Максимальная экономичность для низкоскоростных цифро‑/аналоговых схем. |
| **6 слоев** | SIG/GND/SIG/SIG/PWR/GND | L1: Signal, L2: Ground, L3: Signal, L4: Signal, L5: Power, L6: Ground | Улучшает SI/EMC, дает два внутренних сигнальных слоя и два reference plane. |
| **8 слоев** | SIG/GND/SIG/PWR/GND/SIG/GND/SIG | L1: Signal, L2: Ground, L3: Signal, L4: Power, L5: Ground, L6: Signal, L7: Ground, L8: Signal | «Золотой стандарт» high‑speed. Пара PWR/GND в центре улучшает PDN, множественные GND сокращают return path. |
| **10 слоев** | Разделение high‑speed и power | L1/L2: high‑speed pairs, L3: GND, L4/L5: low‑speed, L6: PWR, L7: GND, L8/L9: high‑speed pairs, L10: low‑speed | Физически разделяет high‑speed (PCIe/USB) и низкоскоростные сигналы, снижает наводки. |
| **HDI (1+N+1)** | Microvia interconnect | L1(Microvia)-L2, L2-L(N-1) (Core Vias), L(N-1)-LN(Microvia) | Лазерные microvia для BGA ≤0.5mm, высокая плотность трассировки. |
| **Rigid‑Flex** | PI + FR‑4 | Жесткая зона (FR‑4) + гибкая зона (PI) | Надежные 3D‑соединения для камер, foldable‑устройств, динамического изгиба. |
| **MCPCB** | Metal core | L1: Copper Trace, Dielectric Layer, Aluminum/Copper Base | Для тепла: металлическая основа как эффективный радиатор, типично для LED/силовых модулей. |

## 4. Методы моделирования: импеданс / тепло / механика

Точное моделирование — ключ к прогнозированию и обеспечению `thermal reliability stackup`.

### 4.1 Моделирование импеданса (Impedance Modeling)

Контроль импеданса — фундамент high‑speed.

*   **Приближенная формула microstrip:**
    $$ Z_0 \approx \frac{87}{\sqrt{\epsilon_r + 1.41}} \ln\left(\frac{5.98H}{0.8W + T}\right) $$
*   **Приближенная формула симметричного stripline:**
    $$ Z_0 \approx \frac{60}{\sqrt{\epsilon_r}} \ln\left(\frac{1.9B}{0.8W + T}\right) $$

Где:
*   `Z₀`: характеристический импеданс (Ω)
*   `εr (Dk)`: диэлектрическая проницаемость
*   `H`: толщина диэлектрика между signal и reference plane
*   `W`: ширина дорожки
*   `T`: толщина меди
*   `B`: суммарная толщина диэлектрика между двумя reference plane (`B = 2H + T_inner`)

**Практика HILPCB:** мы используем решатели уровня Polar Si9000 и библиотеку материалов с реальными Dk/Df, подтвержденными TDR, удерживая импеданс в пределах **±7%** (лучше типичных ±10%).

### 4.2 Тепловое моделирование (Thermal Modeling)

Суть тепловой надежности — управление теплопередачей и тепловыми напряжениями.

*   **Тепловое сопротивление (Rth):**
    $$ R_{th} = \frac{L}{k \cdot A} $$
    где `L` — длина теплопути, `k` — теплопроводность, `A` — площадь сечения. Массив thermal vias снижает `Rth`.

*   **Эквивалентная теплопроводность массива thermal vias (k_eff):**
    $$ k_{eff} = k_{via} \cdot \frac{A_{via}}{A_{total}} + k_{diel} \cdot \frac{A_{diel}}{A_{total}} $$
    Т.е. плотная via‑решетка под компонентом обеспечивает эффективную передачу тепла как взвешенное среднее меди и диэлектрика.

### 4.3 Моделирование механических напряжений (Mechanical Stress Modeling)

Отказы via и усталость пайки часто вызваны CTE‑несоответствием.

*   **Тепловое напряжение (σ):**
    $$ \sigma = E \cdot (\alpha_1 - \alpha_2) \cdot \Delta T $$
    где `E` — модуль Юнга, `α₁/α₂` — CTE материалов, `ΔT` — температурный размах. При большой разнице CTE по оси Z (медь ~17 ppm/°C, FR‑4 после Tg ~50–70 ppm/°C) медная стенка via испытывает высокое растяжение, особенно в толстых платах и при большом aspect ratio.

## 5. Гибридные stackup и специальные структуры (Hybrid Stack & Special Structures)

Для баланса стоимости и производительности часто применяют hybrid stack.

*   **Rogers + FR‑4 hybrid:**
    *   **Сценарий:** RF/high‑speed слои на Rogers (например RO4350B, Dk=3.48), а core и PWR/GND — на high Tg FR‑4.
    *   **Вызовы:** разные системы смол и профили прессования; критична надежность стенок отверстий, часто требуется плазменная обработка.
    *   **Поддержка HILPCB:** база параметров прессования для Rogers/FR‑4, PI/FR‑4 и др.

*   **Back‑drilling (Controlled Depth Drilling):**
    *   **Сценарий:** для >25 Gbps stub via вызывает резонансы и отражения.
    *   **Процесс:** после изготовления с обратной стороны удаляют избыточную часть металлизированного столбика.
    *   **Поддержка HILPCB:** контроль глубины back‑drill ±50µm для минимизации stub.

*   **MCPCB + FR‑4 модульность:**
    *   **Сценарий:** в изделии одновременно есть силовая часть (MOSFET) и сложная цифровая логика.
    *   **Решение:** силовую часть реализуют на MCPCB, а управление — на многослойном FR‑4, соединяя модули коннектором или пайкой.

## 6. Процесс верификации: от материалов к надежности

Теоретически корректный stackup должен подтверждаться производством и испытаниями.

<div class="div-type-6">
    <h4>Пятишаговая методика HILPCB по верификации надежности stackup</h4>
    <ol>
        <li><strong>Входной контроль материалов (IQC):</strong> выборочные проверки Dk/Df, Tg (DSC), Td (TGA) и CTE (TMA) для каждой партии core и PP.</li>
        <li><strong>Контроль ламинации:</strong> тест‑платы с TDR‑графикой измеряются сразу после прессования для быстрой обратной связи и компенсации изменений толщин из‑за течения смолы.</li>
        <li><strong>Coupon‑дизайн и анализ:</strong> в каждом panel используется coupon, который проходит:
            <ul>
                <li><strong>TDR тест импеданса:</strong> проверка соответствия спецификации.</li>
                <li><strong>Cross‑section (металлография):</strong> совмещение слоев, толщина меди в отверстиях, надежность внутренних соединений, void/деламинация.</li>
                <li><strong>Peel strength:</strong> прочность сцепления меди с диэлектриком.</li>
            </ul>
        </li>
        <li><strong>Измерение warpage:</strong> оптические 3D‑методы (например Shadow Moiré) для контроля коробления при температуре рефлоу &lt; 0.75%.</li>
        <li><strong>Ускоренные испытания надежности:</strong>
            <ul>
                <li><strong>Thermal Shock:</strong> например -40°C…125°C, 1000 циклов.</li>
                <li><strong>IST (Interconnect Stress Test):</strong> быстрые нагрев/охлаждение coupon с мониторингом сопротивления via по оси Z.</li>
            </ul>
        </li>
    </ol>
</div>

## 7. Чек‑лист DFM/DFR: производимость и надежность

Ниже — сокращенная версия проверок, включающих материалы и конструктивные правила.

| Категория | Правило | Рекомендации/заметки | Метод проверки |
| :--- | :--- | :--- | :--- |
| **Материалы** | Выбор Tg | Для бессвинцовых профилей (Peak >245°C) рекомендуется Tg ≥ 170°C. | Datasheet, DSC |
|  | CTE (ось Z) | Предпочтительно Td > 340°C и Z‑CTE (после Tg) < 250 ppm/°C. | Datasheet, TMA |
|  | Стабильность Dk/Df | Разброс Dk по плате должен быть < 2%. | VNA, datasheet |
|  | Устойчивость к CAF | Для высоких напряжений/влажности выбирайте материалы с высокой CAF‑устойчивостью. | Datasheet, CAF report |
| **Stackup** | Симметрия | Core/PP и распределение меди должны быть симметричными. | Review, CAM |
|  | Толщина диэлектрика | Допуск толщины для слоев контроля импеданса ≤ ±10%. | Cross‑section, sim |
|  | Комбинация core/PP | Избегайте одиночного толстого PP (например 7628×2); лучше несколько тонких PP. | CAM, lamination spec |
|  | Баланс меди | Старайтесь выровнять coverage по слоям, избегайте резких «медь/пусто». | CAM analysis |
| **Импеданс** | Непрерывность reference plane | Нельзя пересекать разрывы reference plane. | DRC, SI sim |
|  | W/S и компенсация травления | Учитывайте etch compensation; типичный допуск ±1 mil. | CAM, AOI, section |
|  | Связь диффпары | Ошибка длины внутри пары < 5 mil и постоянный зазор. | EDA rules, TDR |
| **Via‑дизайн** | Aspect Ratio | Для мехсверления рекомендуется < 10:1 для качественной металлизации. | DFM, section |
|  | Annular Ring | Outer ≥ 2 mil, inner ≥ 1.5 mil. | DFM, X‑Ray, section |
|  | Thermal vias | Ø 0.3–0.5mm, плотный массив, заполнение и планаризация. | DFM |
|  | Via‑in‑Pad | Требуется resin fill + planar plating (POFV). | Spec, X‑Ray |
|  | Stub после back‑drill | Для >20 Gbps stub < 10 mil. | Back‑drill control, TDR |
| **Медь/трассировка** | Acid traps | Угол ≥ 90°; избегайте острых углов. | DFM |
|  | «Островки» меди | Удалять или подключать к GND. | DFM |
|  | BGA fan‑out | Предпочтительно dog‑bone или via‑in‑pad (если нужно). | DFM |
|  | PWR/GND planes | Плоскости по возможности цельные; разрывы не должны мешать return path. | PI sim, review |
| **Тепло** | Расстановка компонентов | Разносить мощные компоненты, избегать концентрации hot‑spots. | Thermal sim |
|  | GND как радиатор | Большие GND плоскости — хороший heat spreader. | Thermal sim |
|  | Тепловая медь | Большие copper pours + thermal vias к внутренним плоскостям. | DFM |
| **Механика/сборка** | Отступ от края | Компоненты/трассы ≥ 0.125" от края; для V‑cut ≥ 0.02". | DFM |
|  | Tooling holes | 3–4 NPTH для позиционирования SMT/тестов. | DFM |
|  | Panelization | Согласовать V‑cut/таб‑роут и технологическую рамку. | Review with HILPCB |
|  | Anti‑warpage | Симметрия stackup + макро‑баланс меди и компоновки. | Warpage measurement |
| **Надежность** | Защита via | Не ставить тяжелые компоненты прямо на via‑зоны. | Layout review |
|  | Pad design | Следовать IPC‑7351, избегать tombstoning/недопая. | DFM |
|  | Solder mask | Solder mask bridge ≥ 4 mil. | DFM |
|  | Vias под BGA | Избегать открытых via между шарами, чтобы не получить КЗ. | DFM |
|  | Финиш | ENIG (high‑freq/BGA), OSP (cost), ENEPIG (hi‑rel). | Review |

## 8. Сервисный замкнутый цикл HILPCB: партнер по надежности от дизайна до серии

В HILPCB мы строим работу вокруг `thermal reliability stackup` как вокруг замкнутого контура:

1.  **Этап дизайна:** инженерная поддержка по stackup и моделированию. Мы поддерживаем материалы от FR‑4 до Rogers/Megtron и помогаем оценить варианты замены.
2.  **Этап прототипа:** строгая coupon‑верификация (cross‑section, TDR) и испытания thermal shock с инженерным отчетом.
3.  **Этап серии:** закрепление параметров процесса и SPC‑контроль; автоматизированные DFM/DFR‑проверки перед запуском.
4.  **Обратная связь:** производственные и лабораторные данные (yield, распределение импеданса, результаты reliability) возвращаются в правила и библиотеку материалов.
