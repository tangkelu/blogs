# blogs_prompt_v5/_base_templates_v2 (APTPCB)

This folder contains **v2** base templates designed to better match the structure patterns found in top-ranking engineering blogs (answer-first opening, skimmable takeaways, stronger tables/checklists, clearer intent matching).

Use with:
- `python batch_blog_generation_aptpcb.py --base-templates-dir prompts_aptpcb/blogs_prompt_v5/_base_templates_v2`

Templates:
- `aptpcb-query.md`: Query / problem-solving landing page (what/how-to/specs/troubleshooting/cost/lead time).
- `aptpcb-pillar.md`: Pillar / hub guide (definition → metrics → selection → implementation → mistakes → FAQ → glossary).
- `aptpcb-playbook.md`: Buyer playbook (specs → risks → validation → supplier checklist → decision rules).
