# NQ-4 Exact-Product Completion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close `NQ-4` by identifying the smallest worthwhile exact-product supplementation batch for the safe-wave layer-count program without reopening held branches or pretending broader numeric readiness.

**Architecture:** This plan treats `NQ-4` as a narrow source-governance and evidence-pack hygiene pass, not a generic material expansion project. Work is split into three independent discovery lanes: `FR-4 family-shortcut cleanup`, `flex exact-product exception reassessment`, and `remaining safe-wave named-product gap scan`, then converged into one execution/control note plus any approved new fact cards.

**Tech Stack:** Markdown control notes, `llm_wiki` fact cards, existing evidence-pack logs, parallel sub-agents for bounded repository analysis.

## Guardrails

- `NQ-4` is not permission to reopen `H1` broadly.
- Do not treat `Taconic`, generic `FR-4`, generic `RCC/FRCC/build-up`, or generic rigid-flex branches as reopened.
- Do not convert narrow exact-product exceptions into branch-level readiness.
- Keep `B`, `C`, `D`, and `E` blocked numerics blocked.
- Prefer `do not add a card` over low-signal card creation.

## Multi-Agent Ownership

- Agent lane A: `FR-4 family-shortcut cleanup`
  - Scope: safe-wave evidence packs and readiness notes that still rely on generic `FR-4` or `high-Tg` shorthand where exact-product anchors may already exist or still be missing.
- Agent lane B: `flex exact-product exception reassessment`
  - Scope: whether `Panasonic-only narrow coverage` is still true after `Arlon 55NT / 85N / 85NT` and related branch normalization.
- Agent lane C: `remaining safe-wave named-product gap scan`
  - Scope: all six safe-wave blogs and evidence packs for named materials/products still lacking exact-product support.
- Main agent: plan synthesis, queue narrowing, write-set ownership, final control-note updates, and any approved card authorship.
