# APTPCB Internal Link Strategy

APTPCB 的内链策略和 HILPCB 不同，因为站点不是简单的 `products / services / tools` 架构，而是更偏内容 hub。

## 原则

- `pcb/` 与 `pcba/` 是强商业承接页
- `capabilities/` 与 `materials/` 是强知识承接页
- `industries/` 用于应用场景聚合
- `resources/` 用于工具、DFM、术语、下载

## 推荐比例

- 35% `pcb/` 或 `pcba/`
- 25% `capabilities/` 或 `materials/`
- 20% `resources/` 或 `tools/`
- 10% `industries/`
- 10% `quote/` 或 `blog/`

## 推荐位置

- 开头：1 个主承接页
- 中段：2-3 个
- FAQ / 结尾：1-2 个

## 分布要求

- 不要只在 FAQ 或结尾集中放链接
- 开头优先 `pcb/` 或 `pcba/` 主承接页
- 中段优先 `materials/`、`capabilities/`、`resources/` 或 `industries/`
- 同一路径类型不要连续堆满全文

## 不推荐

- 只互链博客
- 一篇文章全部导向 `quote/`
- 材料主题却完全不链 `materials/`
